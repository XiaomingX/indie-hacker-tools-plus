# 0. 商业模式与基本盘对决：OpenAI vs Google (Gemini) vs Meta (Llama) vs Perplexity vs Manus

## 核心赚钱逻辑：模型税 vs 广告税 vs 开源生态 vs 搜索税 vs 代理税

AI 行业的竞争已从“参数竞赛”转入“变现效率对决”。

1. **OpenAI**：**极致的“大模型订阅与 API 税”**。
   - **底层逻辑**：OpenAI 是目前唯一成功建立起 C 端大规模订阅习惯（ChatGPT Plus）和 B 端广泛 API 调用的公司。它赚的是**“智能溢价”**。其基本盘是先发优势带来的**极高用户粘性**和**开发者生态**。
2. **Google (Gemini)**：**防御性的“搜索广告护城河”**。
   - **底层逻辑**：Google 的核心依然是搜索广告。它推出 Gemini 是为了防止搜索入口被 AI 侵蚀。它赚的是**“广告剩余价值”**。其基本盘是**遍布全球的安卓与云服务分发渠道**。
3. **Meta (Llama)**：**“开源作为一种战略毁灭”**。
   - **底层逻辑**：Meta 不卖模型，它通过开源 Llama 摧毁其他人的模型护城河，从而降低自己的研发成本并稳固其社交广告的基本盘。它赚的是**“降低后的运营成本与广告效率”**。
4. **Perplexity**：**“对信息索引的重新定价”**。
   - **底层逻辑**：Perplexity 颠覆了搜索。它不给链接，直接给答案。它赚的是**“节省时间产生的效率税”**。其基本盘是目前全行业最好的**RAG (检索增强生成) 体验**。
5. **Manus**：**“Agentic AI 的中介费”**。
   - **底层逻辑**：Manus 代表了 AI 的下一个阶段——不仅是说，而是做。通过 Agent 完成具体任务，它赚的是**“劳动力替代税”**。

## 营收与成本结构对比 (2025 预估)

| 指标 | OpenAI | Google | Meta | Perplexity | Manus |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **主要收入** | 订阅 (60%) + API (40%) | 广告 (80%) + 云 (20%) | 广告 (98%) | 订阅 + 潜在广告 | 任务佣金/订阅 |
| **ARPU** | 高 (20美元/月) | 低 (广告驱动) | 低 | 中 | **潜在最高** |
| **算力成本占比** | 极高 (由于自研闭源) | 极高 (TPU 自研可对冲) | 极高 (但由于开源可摊薄) | 中 (主要在搜索调用) | 高 (长路径推理) |

## 结论：基本盘的稳固度

- **OpenAI** 的基本盘取决于其能否维持“SOTA (最高技术水平)”；
- **Google** 的基本盘正在被搜索方式的改变所解构；
- **Meta** 的基本盘最稳，因为它是“送水工”；
- **Perplexity** 正在虎口夺食；
- **Manus** 正在开辟无人区。

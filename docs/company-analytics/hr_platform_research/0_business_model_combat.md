# 0. 商业模式与基本盘对决：BOSS 直聘 vs 同道猎聘 vs 前程无忧 (51job)

## 核心赚钱逻辑：移动互联网直连 vs 猎头中介协同 vs 综合人力资源外包

招聘行业的对决本质上是**“对人才简历的索引权与沟通效率之争”**。

1. **BOSS 直聘 (Zhipin)**：**极致的“直连与算法匹配”**。
   - **底层逻辑**：BOSS 直聘彻底颠覆了投递简历的旧模式，改为**“直聊”**。它赚的是**“沟通权与曝光位”**。其基本盘是其**海量的 B 端小微企业主和 C 端极高活跃度的移动端用户**。它将招聘变成了一种高频的社交行为。
2. **同道猎聘 (Liepin)**：**极致的“猎头协同与中高端匹配”**。
   - **底层逻辑**：猎聘通过引入猎头（BHC 模式），解决中高端人才的非标匹配问题。它赚的是**“专业服务溢价与 B 端会员费”**。其基本盘是**积累多年的中高端人才简历库和猎头生态**。
3. **前程无忧 (51job)**：**“全产业链的人力资源外包”**。
   - **底层逻辑**：招聘只是它的入口，它更像是一个综合的人力资源服务商（含外包、培训、测评）。它赚的是**“B 端的综合管理费”**。其基本盘是**深厚的大型国企、外企客户关系和全方位的线下服务网络**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | BOSS 直聘 | 同道猎聘 | 前程无忧 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 增值服务 (C+B端) | B 端企业服务 + 猎头协同 | 招聘 (50%) + 外包与培训 (50%) |
| **营销/营收比** | **最高** (为了维持 C 端高日活) | 中 | 低 |
| **毛利率** | **最高** (纯线上模式) | 中高 | 中 (受外包业务拉低) |
| **变现瓶颈** | 虚假招聘信息治理成本 | 中高端市场容量见顶 | 品牌老化，移动端转型缓慢 |

## 结论：基本盘的稳固度

- **BOSS 直聘** 的基本盘最“活”，它是目前中国招聘市场的流量黑洞；
- **同道猎聘** 的基本盘最“深”，中高端市场的信任成本是其护城河；
- **前程无忧** 的基本盘最“稳”，作为老牌巨头，其 B 端存量客户的粘性极强。

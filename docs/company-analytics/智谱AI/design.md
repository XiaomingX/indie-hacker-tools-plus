# 智谱AI - 投研分析设计方案

## 1. 公司画像速览

**智谱AI**（北京智谱华章科技有限公司）
- **成立时间**：2019年
- **技术源头**：清华大学计算机系知识工程实验室（KEG）成果转化
- **核心产品**：ChatGLM系列大模型、智谱清言（ChatGLM应用）、API开放平台
- **定位**：中国"AI六小龙"之一，通用大模型+行业解决方案双轮驱动
- **融资历程**：多轮融资，估值已进入百亿元俱乐部

## 2. 分析设计逻辑

### 2.1 核心分析主线

```
宏观政策 → 行业格局 → 商业模式 → 财务健康 → 治理结构 → 估值判断 → 催化剂 → 风险排雷 → 终极决策
```

### 2.2 关键假设与检验点

| 假设编号 | 假设内容 | 检验方法 |
|---------|---------|---------|
| H1 | 智谱AI的GLM系列模型在国内处于第一梯队 | 对比公开Benchmark、客户反馈、开发者生态 |
| H2 | B端商业化路径已被验证 | 拆解营收结构、客户集中度、续约率 |
| H3 | 现金跑道充裕，无短期生存风险 | 分析融资节奏、烧钱速率、账面现金 |
| H4 | 清华背景构成持续技术壁垒 | 评估核心团队稳定性、研发投入占比 |
| H5 | 政策环境对国产大模型有利 | 梳理AI监管政策、算力自主可控政策 |

### 2.3 信息源优先级

1. **一级信源**：公司官方公告、产品发布会、招股书
2. **二级信源**：权威财经媒体（36氪、晚点、财新）、投资人公开访谈
3. **三级信源**：券商研报、行业报告（强制降权30%）
4. **四级信源**：社交媒体、自媒体（仅作情绪参考，不作为事实依据）

## 3. 分析模块设计

### Module 1: 宏观与行业（industry_analysis.md）

**分析维度**：
- 中国AI产业政策演进（2023-2026）
- 大模型行业所处周期阶段（Gartner曲线位置）
- 算力供给与芯片制裁影响
- TAM/SAM/SOM拆解与TAM谬误预警
- 行业融资热度与估值中枢变化

**输出结构**：政策Beta → 周期定位 → 市场空间 → 算力约束 → 行业总结

### Module 2: 商业模式（business_model.md）

**分析维度**：
- 营收结构定量拆解（API收入、B端定制、C端订阅、政企项目）
- 单位经济模型（UE）：CAC、LTV、毛利率
- 护城河五维评估：定价权、无形资产、转换成本、网络效应、成本优势
- 规模经济与范围经济分析

**输出结构**：营收拆解 → UE模型 → 护城河评估 → 商业模式可持续性

### Module 3: 竞争格局（competitor_analysis.md）

**分析维度**：
- "AI六小龙"格局：智谱、月之暗面、MiniMax、百川、零一万物、阶跃星辰
- 巨头威胁：百度文心、阿里通义、字节豆包、腾讯混元
- 开源社区冲击：DeepSeek、Qwen开源系列
- CR3/CR5集中度变化趋势
- 竞争是否边际改善（价格战是否见底）

**输出结构**：竞争全景 → 对标分析 → 格局演变 → 竞争总结

### Module 4: 财务质量（financial_quality.md）

**分析维度**：
- 融资历程与估值演变
- 现金跑道估算（假设无新融资）
- 烧钱速率与收入增长的匹配度
- 单位经济模型的边际改善趋势
- 研发投入占比与资本化策略

**输出结构**：融资全景 → 现金跑道 → 烧钱效率 → UE趋势 → 财务总结

### Module 5: 治理结构（governance.md）

**分析维度**：
- 创始团队背景与稳定性
- 股权结构与控制权安排
- 投资人阵容与条款博弈
- 关键人才流失风险
- 清华系资源整合能力

**输出结构**：团队画像 → 股权结构 → 投资人分析 → 人才风险 → 治理总结

### Module 6: 估值与安全边际（valuation_safety.md）

**分析维度**：
- 最近一轮估值与倍数分析
- 对标公司估值比较（海外：OpenAI/Anthropic；国内：月之暗面/MiniMax）
- PS/EV/Revenue倍数合理性
- 下一轮估值的合理区间
- 安全边际计算（悲观/中性/乐观场景）

**输出结构**：当前估值 → 对标分析 → 倍数合理性 → 场景估值 → 安全边际

### Module 7: 催化剂（catalysts.md）

**分析维度**：
- 未来1-2年的业绩超预期可能性
- IPO窗口与市场环境
- 大客户签约/战略合作
- 政策利好（算力补贴、国产替代）
- 技术突破（新一代模型发布）

**输出结构**：催化剂清单 → 概率评估 → 时间窗口 → 催化剂总结

### Module 8: 逆向分析（reverse_analysis.md）

**分析维度**：
- "事前验尸"：假设投资亏损80%，倒推原因
- 极端场景构建：
  - 大厂免费策略彻底击穿商业化
  - 开源模型技术追平闭源
  - 核心团队集体出走
  - 融资环境急剧恶化
  - 政策监管收紧
- 每个场景的概率与杀伤力评估

**输出结构**：Pre-mortem场景 → 概率评估 → 杀伤力矩阵 → 风险总结

### Module 9: 资金面与情绪（sentiment_funds.md）

**分析维度**：
- 一级市场对AI赛道的热度变化
- 头部VC的跟投意愿
- 媒体情绪指数（正面/负面报道比例）
- 招聘热度作为先行指标
- 开发者社区活跃度（GitHub Star、HuggingFace下载）

**输出结构**：一级市场热度 → VC情绪 → 媒体情绪 → 先行指标 → 情绪总结

### Module 10: 芒格检查清单（munger_checklist.md）

**分析维度**：
- 多学科栅格模型检验（物理、生物、心理、经济、社会）
- "Too Hard"堆判定
- 能力圈检验：投资人能否理解这个商业模式？
- 管理层诚信度检验
- 长期持有可行性

**输出结构**：栅格检验 → 能力圈 → 诚信度 → 长期可行性 → 综合判定

### Module 11: 最终决策（final_report.md）

**分析维度**：
- 综合前述10个模块的核心发现
- 执行"Why Now?"检验
- 给出明确评级（强烈推荐/推荐/中性/规避/强烈规避）
- 投资操作建议（如可投资）
- 持续跟踪指标

**输出结构**：核心摘要 → Why Now检验 → 评级 → 操作建议 → 跟踪指标

## 4. 质量控制

- **交叉验证**：关键结论至少需要2个独立信源支撑
- **证伪机制**：每个正面结论必须附带反面论点
- **数据新鲜度**：优先使用2025H2-2026Q1数据
- **可追溯性**：所有数据点标注来源和时间

# 4. 财务真实质量与股东盈余 - 智谱AI (2513.HK)

## 1. 股东盈余 (Owner Earnings) 穿透分析

智谱AI 作为处于极速扩张期的硬科技企业，目前财务表现为典型的“高研发、高投入、短期巨亏”。

| 项目 (2025 财年数据) | 金额 (亿人民币) | 说明 |
|--------------------|---------------|------|
| **归母净利润 (GAAP)** | -47.18 | 包含股份支付、估值波动等非现金项 |
| **经调整净利润** | **-31.82** | **反映核心业务的真实现金消耗** |
| **加：折旧与摊销** | ~8.0 | 算力集群及无形资产摊销 |
| **减：维持性资本开支** | ~15.0 | AI 算力的高频更新与研发投入 |
| **股东盈余 (Owner Earnings)** | **~ -38.82** | **仍处于高强度的“失血”阶段** |

**Owner Earnings 评价**：智谱AI 目前完全不具备内生造血能力。
- **冷酷事实**：2025 年每赚 1 元钱营收，需要消耗约 4.4 元的经调整亏损。这种商业模式在 2026 年必须证明其营收增速（131.9%）能持续超越亏损增长率，才能维持资本市场的信心。

---

## 2. 核心财务比率与经营效率

### 2.1 毛利率分析
- **综合毛利率**：41%。相对于 SaaS 行业（70%-80%）偏低，反映出算力成本仍占据营收的大头。
- **MaaS 毛利拐点**：MaaS 业务毛利率提升至 18.9%，显示出随着标准化 API 调用量的增加，盈利能力正在从底部的负值区间快速爬升。

### 2.2 研发投入质量
- **研发占比**：极高。绝大部分资金用于购置算力资源和雇佣顶级算法人才。这种投入是企业长期的“护城河”，但在短期内是财务报表的“杀手”。

---

## 3. 资产负债表与现金流管理 (Post-IPO)

- **现金储备**：通过港股 IPO 募集了巨额资金，预计当前现金余额足以支撑未来 3-4 年的高强度烧钱。
- **存货与预付**：预付算力租赁费用显著增加，反映出算力资源的紧张和公司积极的资源锁定策略。

---

## 4. 结论与总结

**财务评定：高杠杆、高成长、重度依赖融资驱动的硬科技模式**。

智谱AI 的财务质量不取决于当下的盈亏，而取决于其 **“营收/算力成本”的比率变动**。

**关键监控指标**：
1. **经调整亏损缩窄速度**：关注 2026 年半年报是否能实现亏损额的边际递减。
2. **应收账款占比**：由于 B 端业务较多，需警惕政企客户长周期回款导致的现金流压力。
3. **MaaS 业务的营收占比**：这是从“重人力交付”转向“轻资产软件”的关键指标。

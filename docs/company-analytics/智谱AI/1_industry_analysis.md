## 1. 政策Beta：政策是终极Beta

### 1.1 政策演进时间线

| 时间 | 政策/事件 | 影响 |
|------|---------|------|
| 2023.08 | 《生成式AI服务管理暂行办法》生效 | 全球首批生成式AI法规，建立算法备案+安全评估制度 |
| 2024 | 各地密集出台AI产业政策 | 北京/上海/深圳设立专项资金和算力补贴 |
| 2024-2025 | 《人工智能法（草案）》推进立法 | 从单点监管走向系统性立法 |
| 2025.01 | **智谱AI被列入美国实体清单** | 限制获取美国技术，倒逼国产化 |
| 2025.07 | 智谱AI以MIT许可证开源模型 | 响应国家开源政策，构建开发者生态 |
| 2026.01 | 智谱AI港股上市（2513.HK） | 政策支持AI企业资本市场融资 |

### 1.2 政策定性判断

**结论：政策强力支持，但监管框架逐步收紧**

- **利好面**：
  - "AI+"战略纳入新质生产力框架，政策确定性高
  - 各地算力补贴降低企业成本
  - 国产替代政策为智谱AI等本土玩家创造护城河
  - 港股上市通道畅通，政策不阻拦

- **约束面**：
  - 算法备案+安全评估增加合规成本
  - 数据合规要求趋严（训练数据来源审查）
  - 内容安全监管持续，限制部分应用场景
  - 跨境数据流动受限，影响海外业务拓展

**政策Beta评分：8/10（强力支持）**

## 2. 周期定位：大模型行业在Gartner曲线哪里？

### 2.1 Gartner曲线演进

```
                泡沫顶峰

   触发期 ──→ 膨胀期 ──→ [当前位置？] ──→ 复苏期 ──→ 成熟期
                                      ↓
                                 泡沫破裂低谷
```

### 2.2 周期判断

**当前位置：泡沫破裂低谷期 → 复苏早期过渡阶段**

判断依据：
1. **价格战已见底**：API价格从2024年的地板价（如DeepSeek引发的降价潮）趋于稳定
2. **融资环境分化**：头部公司（智谱、月之暗面）仍能融资，但中小玩家融资困难
3. **商业化落地加速**：从"百模大战"转向应用落地，B端收入占比提升
4. **淘汰赛开始**：零一万物等公司已显疲态，行业集中度提升
5. **开源冲击波**：DeepSeek等开源模型改变了竞争格局，但闭源仍有差异化空间

### 2.3 周期对智谱AI的影响

| 泡沫顶峰（2023-2024H1） | ✅ 已完成高价融资，估值锁定在30亿美元 |

## 4. 市场空间：TAM/SAM/SOM拆解

### 4.1 TAM谬误预警

**常见谬误**："中国AI市场将达万亿规模"

**纠偏**：
- 万亿级TAM包含硬件、应用、服务全产业链，纯大模型API/订阅只是其中一小部分
- 大模型公司的实际可触达份额受制于：客户付费意愿、竞争格局、开源替代

### 4.2 TAM/SAM/SOM拆解

| 层级 | 定义 | 规模估算 | 说明 |
|------|------|---------|------|
| **TAM** | 中国AI市场总规模 | ~5000亿元（2026E） | 包含硬件、软件、服务 |
| **SAM** | 大模型相关市场 | ~800-1000亿元（2026E） | API、定制、订阅、解决方案 |
| **SOM** | 智谱AI可触达份额 | ~50-80亿元（2026E） | 考虑竞争、渠道、品牌 |

### 4.3 市场空间评分

**市场空间评分：7/10（空间大，但实际可触达份额有限）**

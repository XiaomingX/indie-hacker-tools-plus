# 0. 简单性与能力圈初筛 - 阶跃星辰 (StepFun) (2026)

## 1. 业务可理解性

**赚钱逻辑**：阶跃星辰是 AI 时代的“智力批发商”。它开发万亿参数的大模型 (Step 系列)，并将其嵌入手机、汽车等终端，赚取授权费、调用费及 C 端订阅费。

- **核心竞争力**：国内顶尖的 MoE (混合专家模型) 架构研发能力。
- **业务锚点**：AI + 终端 (手机/车)。

## 2. 护城河分析

1. **人才密度**：由姜大昕博士领衔，汇聚了微软、字节等大厂的核心算法团队。
2. **终端粘性**：模型深度集成于 OPPO、吉利等硬件底层，具有极高的生态排他性。

## 3. 结论

**结论**：**属于“AGI 底层基础设施”的标的**。

**理由**：阶跃星辰不仅有“大脑”（模型），还有“身体”（产业链资本与终端份额），是国产大模型中最具 IPO 潜力的选手之一。

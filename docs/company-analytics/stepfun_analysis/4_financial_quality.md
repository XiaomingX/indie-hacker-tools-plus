# 4. 财务真实质量与单位经济模型 - 阶跃星辰

## 1. 财务画像 (2025-2026)

作为一家高研发投入的准 IPO 企业，阶跃星辰的财务核心不在于“利润”，而在于 **“单位算力的智能化产出比”**。

- **营收规模**：预计 2025 年营收突破 10 亿人民币（含授权收入与 API 消耗）。
- **毛利水平**：API 调用毛利约为 30-40%，授权费毛利 > 90%。
- **现金流状况**：B+ 轮 50 亿融资后，现金储备极其充裕，可支撑其在 2026 年底前的所有算力开销。

---

## 2. 核心指标深潜

### 2.1 推理成本 (Inference Cost)
- **表现**：得益于 MoE 架构，Step-2 系列模型的推理成本相比同等参数量的稠密模型下降了 70%。这使其在价格战中具备极强的财务韧性。

### 2.2 资本开支 (CapEx)
- **投向**：大部分现金流向了算力节点租赁及核心人才招募。阶跃星辰目前的算力效率处于行业前 1%。

---

## 3. 风险与真实性挑战

- **收入确认**：需关注产业链股东（如手机厂商）的“置换式”协议。
- **坏账计提**：在 AI 初创公司普遍缺钱的背景下，下游开发者的欠费风险。

---

## 4. 结论与总结

**财务评定：阶跃星辰是国产模型中“钱花得最准”的公司之一**。

其财务健康度高度依赖于 **港股 IPO 的顺利与否**。

**关键监控指标**：
- **季度 API 调用量的复合增长率**。
- **人均研发产出比 (Revenue per Employee)**。

# 3. 竞争格局与市场份额 - 阶跃星辰 (2026)

## 1. 行业格局：大模型“决赛圈”

2026 年，中国大模型市场已形成“三三制”格局：

- **第一梯队 (国产基座)：阶跃星辰、月之暗面 (Moonshot)、MiniMax**。
- **第二梯队 (大厂自研)：阿里通义、百度文心、字节豆包**。

### 阶跃星辰的差异化占位：
- **月之暗面**：专注超长文本 (Long Context)。
- **MiniMax**：专注情感社交与游戏 AI。
- **阶跃星辰**：专注 **“逻辑推理 + 端侧部署”**。

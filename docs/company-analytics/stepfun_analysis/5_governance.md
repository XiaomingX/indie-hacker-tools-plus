# 5. 治理结构与管理层画像 - 阶跃星辰 (2026)

## 1. 治理特色：技术极客与产业老兵的共治

### 1.1 姜大昕 (CEO) 的“微软基因”
- **画像**：原微软全球副总裁，具备管理万级研发团队的经验。
- **治理逻辑**：极其强调“工程纪律”。阶跃星辰的模型迭代频率极高，源于其高度自动化的评估与对齐流水线。

### 1.2 印奇 (董事长) 的“产业整合”
- **画像**：旷视科技创始人。
- **角色**：2026 年印奇的加入，补齐了阶跃星辰在资本运作、政府关系及硬件落地的短板。这标志着阶跃星辰从“纯模型公司”向“产业平台”转型。

# 2. 商业模式与护城河深潜 - 阶跃星辰

## 1. 核心商业模式穿透

阶跃星辰采取的是 **“全栈模型 + 垂直场景 + 硬件集成”** 的三位一体模式：

- **智力批发 (B端 API)**：向中小型企业提供 Step-2/3 系列的高精度 API 调用。
- **端侧赋能 (OS 集成)**：这是其核心护城河。将 AI 引擎深度嵌入 OPPO、吉利等终端操作系统，收取“智力授权费”或按装机量计费。
- **C端流量 (跃问/冒泡鸭)**：通过自研应用获取第一手用户数据，反哺模型训练。

---

## 2. 护城河深潜 (The Moat)

### 2.1 MoE 架构的先发优势
- **护城河**：万亿参数模型不是简单的算力堆砌，而是极其复杂的分布式计算与混合专家调度。阶跃星辰在 MoE 上的工程实现领先国内同行 6-12 个月，这意味着同样的算力下，其模型更“聪明”且推理更便宜。

### 2.2 产业链“肉搏式”绑定
- **护城河**：其股东包括华勤、龙旗等 ODM 巨头。这意味着阶跃星辰的模型在“出厂前”就已经写入了千万台电子设备的芯片中，这种物理级别的渗透率构成了极高的竞争门槛。

---

## 3. 定价权测试 (Pricing Power)

在 2026 年大模型降价潮中，阶跃星辰通过 **“Step-2 Literature Master (文学大师版)”** 等高度定制化模型，在网文、剧本杀、内容营销等特定领域保持了高客单价。

## 4. 结论

**商业模式评定：阶跃星辰是“长在硬件里的智力神经”**。

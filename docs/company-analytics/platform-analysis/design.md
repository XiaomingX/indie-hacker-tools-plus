# 平台对比研究 设计文档

## 分析方法论

### 巴菲特-芒格投资框架
```
价值投资核心 = 内在价值 × 安全边际 × 确定性
```

**巴菲特视角**：
- 业务理解：这家企业销售什么？客户为何选择它而非竞争对手？
- 护城河：什么是阻止竞争对手复制其商业模式的壁垒？
- 管理质量：管理层是否理性、坦诚、以股东利益为导向？
- 定价权：企业是否有提价能力而不失去客户？

**芒格视角（栅格模型）**：
- 心理学：人类认知偏误如何影响消费决策？
- 经济学：激励机制如何塑造竞争格局？
- 生物学：网络效应如何产生赢家通吃？
- 工程学：规模经济如何压低成本？
- 数学：复利效应如何放大优势？

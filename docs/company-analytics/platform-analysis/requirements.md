# 投研 Agent 需求规格说明书

## 项目概述
基于 touyan-agent-skill.md 的价值投资研究框架，针对电商平台和视频平台进行深度对比分析，模拟巴菲特和芒格的投资视角，解读为何特定平台能够脱颖而出。

## 分析任务

### 任务一：电商平台对比研究
**研究对象**：拼多多、微店、云集

**核心问题**：为何拼多多发展最迅速、市值最高？

**分析维度**：
1. 发展时间线与历史选择对比
2. 商业模式与护城河分析
3. 竞争格局演变
4. 资本配置与财务质量
5. 治理结构与管理团队
6. 估值与安全边际
7. 催化剂与拐点（Why Now?）
8. Pre-mortem 逆向风险分析
9. 资金面与情绪面
10. 芒格栅格模型复核

### 任务二：视频平台对比研究
**研究对象**：快手、抖音（字节跳动）、淘宝直播、斗鱼直播、B站（bilibili）

**核心问题**：为何字节跳动发展最迅速、市值最高？

**分析维度**：
同上

## 输出要求
- 生成 requirements.md、design.md、tasks.md
- 逐项执行tasks.md中的分析任务
- 每项分析后产出对应报告
- 最终输出完整的投资分析报告

## 遵循框架
- 巴菲特价值投资理论
- 芒格栅格模型（Multiple Mental Models）
- A股投研标准流程（Policy Beta、TAM Fallacy Warning、库存/信用周期等）

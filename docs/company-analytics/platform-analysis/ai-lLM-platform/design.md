# 中国大模型AI平台投研分析 设计文档

## 分析方法论

### 巴菲特-芒格投资框架
```
价值投资核心 = 内在价值 × 安全边际 × 确定性
```

**巴菲特视角**：
- 业务理解：这个AI平台能做什么？客户为何选择它？
- 护城河：什么是阻止竞争对手复制其成功的壁垒？
- 管理质量：管理层是否理性、坦诚、以股东利益为导向？
- 定价权：平台是否有提价能力？

**芒格视角（栅格模型）**：
- 心理学：用户行为、付费心理、习惯形成
- 经济学：规模效应、网络效应、平台经济
- 生物学：生态系统、赢家通吃
- 工程学：技术架构、算力效率
- 数学：货币化率、获客成本、LTV

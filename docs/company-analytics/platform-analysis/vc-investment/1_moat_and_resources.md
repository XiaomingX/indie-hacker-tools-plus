# 1. 核心护城河与资源禀赋对比

## 护城河鉴定：谁是“认知的灯塔”？谁是“服务的堡垒”？

### 1. Sequoia：不可逾越的“历史业绩”与“全球网络”
- **核心禀赋**：**过去 50 年几乎投中了所有伟大的公司 (Apple, Google, WhatsApp...)**。红杉的护城河在于“如果你不拿红杉的钱，你的对手拿了，你就会害怕”。
- **转换成本**：**极高 (基于品牌信用溢价)**。
- **真伪判定**：**品牌与信息流壁垒**。

### 2. a16z：极致的“投后服务”与“媒体矩阵”
- **核心禀赋**：**数以百计的投后支持人员 + 全球最有影响力的科技博客/播客**。a16z 的护城河在于它能帮创始人解决“找人、找客户、找政府”的所有麻烦。
- **转换成本**：**中高 (基于服务依赖)**。
- **真伪判定**：**运营效率壁垒**。

### 3. Benchmark：极致的“合伙人直投”与“精英声誉”
- **核心禀赋**：**每一个项目都由顶级合伙人亲力亲为，不设分析师**。
- **转换成本**：**低 (但入场券稀缺)**。
- **真伪判定**：**稀缺性壁垒**。

## 资源禀赋横向对比

| 资源维度 | Sequoia | a16z | Benchmark |
| :--- | :--- | :--- | :--- |
| **Deal Flow (项目源)** | **最强** (全球化) | 强 (硅谷中心) | 强 (精英推荐) |
| **投后赋能力** | 强 | **最强** (工业化服务) | 弱 (仅合伙人建议) |
| **资本动员力** | **最强** | 强 | 中 |
| **退出成功率** | **最高** | 高 | 极高 (针对早期) |

## 结论：护城河的稳固性排序

**Sequoia > Benchmark >> a16z**

- **Sequoia** 赢在“系统”，它是科技资本的终极收割机。
- **Benchmark** 赢在“技艺”，它是 VC 原始精神的守护者。
- **a16z** 赢在“营销”，它是资本界的超级 MCN。

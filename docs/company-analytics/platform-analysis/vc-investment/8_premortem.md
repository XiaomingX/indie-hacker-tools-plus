# Pre-mortem逆向风险分析

## 核心结论
**创投机构最可能的价值毁灭场景是"估值泡沫破裂+流动性收紧+投资失败"三重叠加。软银愿景基金是典型的Pre-mortem案例，伯克希尔凭借零成本资本和理性决策，风险最低。**

---

## 一、Pre-mortem思维框架

### 1.1 巴菲特的忠告

> "我们应该首先思考什么会让我们投资失败，然后避免这些事情。"

### 1.2 芒格的反向思维

> "告诉我我会死在哪里，我就永远不去那里。"

---

## 二、软银愿景基金Pre-mortem分析

### 2.1 场景一：WeWork估值崩塌（已发生）

**情景描述**：
- WeWork估值从470亿美元跌至约20亿美元
- 孙正义投资185亿美元
- 愿景基金一期损失约100亿美元

**传导路径**：
```
WeWork失败 → 愿景基金亏损 →
→ LP信任下降 →
→ 募资困难 →
→ 投资收缩
```

### 2.2 场景二：愿景基金二期困境

**情景描述**：
- 愿景基金二期规模缩水
- LP投资意愿下降
- 软银被迫自投

**传导路径**：
```
募资困难 → 规模缩水 →
→ 投资能力下降 →
→ 回报下降 →
→ 软银股价下跌
```

## 三、伯克希尔Pre-mortem分析

### 3.1 场景一：巴菲特离世（概率100%）

**情景描述**：
- 巴菲特94岁，早晚发生
- 潜在接班人能力待验证
- 文化传承风险

**传导路径**：
```
巴菲特离世 → 市场恐慌 →
→ 股价下跌 →
→ 新管理层能力验证
```

### 3.2 场景二：保险业务危机（概率极低）

**情景描述**：
- 超大型自然灾害
- 保险业务大规模赔付
- 浮存金受损

## 四、黑石资本Pre-mortem分析

### 4.1 场景一：市场大跌（概率中等）

**情景描述**：
- 二级市场估值大幅下跌
- PE组合估值下跌
- 业绩提成下降

**传导路径**：
```
市场大跌 → PE估值下跌 →
→ 业绩下滑 →
→ AUM下降 →
→ 管理费收入下降
```

### 4.2 场景二：杠杆风险（概率低）

**情景描述**：
- 高杠杆收购
- 利率上升
- 偿债压力

## 五、多重风险叠加情景

### 5.1 黑天鹅情景（概率5%）

**情景描述**：
全球金融危机 + 创投泡沫破裂 + 人才流失

```

→ 大多数机构倒闭
```

## 七、核心结论

| 机构 | 核心风险 | 概率 | 影响 |
|------|----------|------|------|
| 伯克希尔 | 巴菲特离世 | 100%（早晚） | 中 |
| 软银 | 商业模式失败 | 30% | 极大 |
| 黑石 | 市场大跌 | 20% | 中 |

**巴菲特总结**：
> "Pre-mortem不是让我们恐惧，而是让我们'知己知彼'。了解最坏的情况，才能在风险来临时从容应对。"

**芒格补充**：
> "风险管理不是避免所有风险，而是承担'正确的风险'。伯克希尔承担的风险是'系统性风险'，这是无法避免的。但我们可以通过'理性'来控制风险。"

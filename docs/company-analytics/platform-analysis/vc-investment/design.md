# 创投机构投研分析 设计文档

## 分析方法论

### 巴菲特-芒格投资框架
```
价值投资核心 = 内在价值 × 安全边际 × 确定性
```

**巴菲特视角**：
- 业务理解：这机构投资什么？如何在不确定性中寻找确定性？
- 护城河：什么是阻止竞争对手复制其成功的壁垒？
- 管理质量：管理层是否理性、坦诚、以股东利益为导向？
- 定价权：机构是否有持续创造回报的能力？

**芒格视角（栅格模型）**：
- 心理学：投资者行为、风险偏好、损失厌恶
- 经济学：规模效应、网络效应、复利
- 生物学：进化论、适者生存
- 工程学：风险控制、组合管理
- 数学：概率论、期望值、凯利公式

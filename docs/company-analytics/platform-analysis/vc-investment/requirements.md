# 创投机构投研分析 需求规格说明书

## 项目概述
基于 touyan-agent-skill.md 的价值投资研究框架，针对全球头部创投机构进行深度对比分析，模拟巴菲特和芒格的投资视角，解读创投行业的竞争格局与估值逻辑。

## 分析背景
创投机构是全球科技创新背后的重要推手。在一级市场估值泡沫、二级市场流动性收紧的背景下，各创投机构呈现出明显的发展分化。伯克希尔哈撒韦是长期价值投资的典范，柏基投资是全球最大的长期投资机构之一。

### 核心问题
为何柏基投资发展最迅速，是全球最大的长期投资机构之一？

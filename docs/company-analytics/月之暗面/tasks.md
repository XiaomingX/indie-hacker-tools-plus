## 任务详细说明

### T1: 宏观周期、政策导向与AI行业分析

**目标**：理解月之暗面所处的宏观环境和行业周期位置

**执行内容**：
1. 梳理2023-2026年中国AI产业政策演进（生成式AI管理办法、算力基础设施规划、国产替代政策）
2. 判断大模型行业在技术成熟度曲线的位置（是否已过炒作期进入爬升期？）
3. 分析算力供给约束（英伟达禁令影响、国产算力替代进度）
4. 拆解AI应用层TAM/SAM/SOM，执行TAM谬误预警
5. 分析行业融资热度与估值中枢变化趋势

**关键问题**：
- 政策是助力还是约束？监管是否在收紧？
- 行业是否已进入淘汰赛阶段？
- AI应用层的商业化拐点是否已到来？

**输出要求**：`reports/月之暗面/1_industry_analysis.md`

---

### T2: 商业模式与护城河深潜

**目标**：拆解月之暗面的营收结构，评估商业模式可持续性

**执行内容**：
1. 定量拆解营收结构：
   - C端订阅收入（Kimi+付费版用户数、ARPU）
   - B端API调用收入（按调用量/Token计费）
   - 企业定制解决方案
2. 构建单位经济模型（UE）：
   - CAC（获客成本）估算：信息流投放、渠道分成
   - LTV（客户生命周期价值）估算
   - 毛利率水平与趋势（推理成本是核心变量）
3. 护城河五维评估：
   - 定价权：在价格战环境下是否有定价权？
   - 无形资产：长文本技术专利、品牌认知
   - 转换成本：用户迁移成本（对话历史、使用习惯）
   - 网络效应：开发者生态、数据飞轮
   - 成本优势：推理成本优化能力、规模效应
4. C端流量逻辑深潜：
   - MAU/DAU趋势与获客效率
   - 用户留存曲线（次日/7日/30日留存）
   - 与豆包等巨头产品的流量争夺态势

**关键问题**：
- C端流量能否转化为可持续收入？
- UE模型是否已跑通？边际是否在改善？
- 长文本先发优势是否仍有意义？

**输出要求**：`reports/月之暗面/2_business_model.md`

---

### T3: 竞争格局与对手对比

**目标**：理解月之暗面在竞争格局中的位置，判断竞争是否边际改善

**执行内容**：
1. "AI六小龙"横向对比：
   - 月之暗面 vs 智谱AI vs MiniMax vs 百川智能 vs 零一万物 vs 阶跃星辰
   - 各家差异化定位、融资规模、估值、核心产品
2. 巨头威胁评估：
   - 字节豆包（流量优势+价格战）
   - 百度文心一言（搜索流量+企业客户）
   - 阿里通义千问（开源+云协同）
   - 巨头的免费/低价策略对创业公司的冲击
3. 开源社区冲击：
   - DeepSeek（技术黑马）
   - Qwen开源系列、LLaMA生态
   - 开源模型是否正在追平闭源？
4. 月之暗面差异化分析：
   - 长文本技术壁垒是否仍存在？
   - C端产品体验优势
   - 品牌认知度
5. CR3/CR5集中度变化
6. 竞争是否边际改善（价格战是否见底、产能是否出清）

**关键问题**：
- 月之暗面的差异化定位是否可持续？
- 大厂的免费策略是否已击穿创业公司的生存空间？
- 六小龙中谁最可能出局？

**输出要求**：`reports/月之暗面/3_competitor_analysis.md`

### T4: 财务质量与资本配置

**目标**：评估月之暗面的财务健康度和资本使用效率

**执行内容**：
1. 融资历程与估值演变：
   - 各轮融资时间、金额、估值、投资方
   - 估值增长曲线是否合理
2. 现金跑道估算：
   - 假设无新融资，当前现金可支撑多久？
   - 月均烧钱速率估算（算力成本+人员成本+获客成本）
3. 烧钱效率分析：
   - 每烧1元钱带来多少收入增长？
   - 与同行对比效率如何？
4. 单位经济模型边际趋势：
   - 毛利率是否在改善？（推理成本下降）
   - CAC是否在上升？（流量竞争加剧）
5. 融资依赖度分析：
   - 是否存在"融资依赖症"？
   - 自身造血能力何时能覆盖运营成本？

**关键问题**：
- 现金跑道是否安全（>18个月）？
- 烧钱效率是否在改善？
- 下一轮融资的紧迫性如何？

**输出要求**：`reports/月之暗面/4_financial_quality.md`

### T5: 治理结构与管理层画像

**目标**：评估创始团队能力、股权结构合理性、治理风险

**执行内容**：
1. 创始人画像：
   - 杨植麟背景（清华、CMU、循环智能）
   - 技术能力 vs 商业化能力
   - 公开言论与实际执行的一致性
2. 核心团队分析：
   - 技术负责人、产品负责人、商业化负责人
   - 团队稳定性（是否有关键人员流失）
3. 股权结构分析：
   - 创始团队持股比例
   - 是否存在AB股/同股不同权安排
   - 控制权是否稳固
4. 投资人阵容分析：
   - 头部VC背景（红杉、阿里、美团等）
   - 是否存在对赌/清算优先权等特殊条款
   - 战略投资者的潜在利益冲突
5. 关键人才风险：
   - 核心技术人员流失概率
   - 人才市场竞争态势

**关键问题**：
- 杨植麟是否具备将技术能力转化为商业成功的素质？
- 股权结构是否会导致决策效率低下？
- 战略投资者（如阿里）是否会限制公司独立性？

**输出要求**：`reports/月之暗面/5_governance.md`

### T6: 估值深潜与安全边际

**目标**：评估当前估值合理性，计算安全边际

**执行内容**：
1. 最近一轮估值分析：
   - 估值金额、估值倍数（PS、EV/Revenue、EV/MAU）
   - 估值增长是否匹配业务增长
2. 对标公司估值比较：
   - 海外：OpenAI（$300B+）、Anthropic（$60B+）
   - 国内：智谱AI（$3B+）、MiniMax（$2.5B+）
   - 倍数差异的合理性分析
3. 场景估值：
   - 悲观场景：商业化不及预期，估值下调50%
   - 中性场景：维持当前增速，估值稳定
   - 乐观场景：MAU破亿+IPO，估值上调100%
4. 安全边际计算：
   - 当前估值对应的隐含增长率
   - 与实际增长率的偏离度

**关键问题**：
- 当前估值是否已透支未来2-3年的增长？
- 与海外对标公司相比，估值折价是否合理？
- 下一轮融资的合理估值区间是多少？

**输出要求**：`reports/月之暗面/6_valuation_safety.md`

### T7: 催化剂与拐点分析

**目标**：识别未来1-2年的催化剂和估值修复触发点

**执行内容**：
1. 业绩催化剂：
   - MAU/DAU突破关键节点
   - C端付费转化率提升
   - B端大客户签约
   - 单位经济模型转正
2. 资本市场催化剂：
   - IPO窗口（A股/港股/美股）
   - 战略投资者引入
   - 估值重定价事件
3. 政策催化剂：
   - 算力补贴政策
   - 国产替代加速
   - AI应用层鼓励政策
4. 技术催化剂：
   - 新一代模型发布
   - 多模态能力突破
   - 推理成本大幅下降

**关键问题**：
- 最可能的催化剂是什么？概率多大？
- 催化剂的时间窗口是什么时候？
- 催化剂是否已被市场定价？

**输出要求**：`reports/月之暗面/7_catalysts.md`

### T8: 逆向思考与尾部风险

**目标**：执行"事前验尸"，识别可能导致投资失败的极端场景

**执行内容**：
1. Pre-mortem场景构建：
   - 场景1：字节豆包免费策略彻底击穿C端付费（概率：？%，杀伤力：？/10）
   - 场景2：开源模型技术追平闭源（概率：？%，杀伤力：？/10）
   - 场景3：核心团队集体出走（概率：？%，杀伤力：？/10）
   - 场景4：融资环境急剧恶化，无法完成下一轮（概率：？%，杀伤力：？/10）
   - 场景5：政策监管收紧，限制AI应用（概率：？%，杀伤力：？/10）
   - 场景6：技术路线被颠覆（如新架构替代Transformer）（概率：？%，杀伤力：？/10）
   - 场景7：阿里战略投资导致公司丧失独立性（概率：？%，杀伤力：？/10）
2. 风险矩阵：
   - 概率×杀伤力的风险评分
   - 最致命的风险组合

**关键问题**：
- 哪个风险最被低估？
- 是否存在"黑天鹅"级别的风险？
- 这些风险是否已被定价？

**输出要求**：`reports/月之暗面/8_reverse_analysis.md`

### T9: 资金面与情绪面

**目标**：评估一级市场对月之暗面和AI赛道的情绪热度

**执行内容**：
1. 一级市场热度：
   - AI赛道整体融资趋势（2023-2026）
   - 头部VC的投资节奏变化
   - 大模型赛道是否已从"抢投"转为"观望"？
2. 媒体情绪分析：
   - 正面/负面报道比例变化
   - 关键叙事变化（从"技术突破"到"商业化落地"？从"六小龙"到"淘汰赛"？）
3. 先行指标：
   - 招聘热度（Boss直聘岗位数量变化）
   - 产品下载量趋势（七麦数据）
   - 社交媒体声量（微博、小红书、知乎讨论热度）
4. 竞争对手融资动态：
   - 其他五小龙的融资进展
   - 是否有公司已掉队？

**关键问题**：
- 一级市场对大模型赛道是否在降温？
- 媒体叙事是否在转向负面？
- 先行指标是否显示业务放缓？

**输出要求**：`reports/月之暗面/9_sentiment_funds.md`

### T10: 芒格多维检查清单

**目标**：运用芒格栅格模型进行多维复核

**执行内容**：
1. 多学科栅格检验：
   - 物理学：是否违背基本规律？（如永动机式增长预期）
   - 生物学：是否符合进化逻辑？（如适者生存的竞争环境）
   - 心理学：是否存在认知偏差？（如确认偏误、锚定效应、叙事幻觉）
   - 经济学：是否符合经济规律？（如规模不经济问题）
   - 社会学：是否符合社会趋势？（如AI替代焦虑、隐私担忧）
2. "Too Hard"堆判定：
   - 商业模式是否过于复杂？
   - 估值是否过于依赖假设？
3. 能力圈检验：
   - 投资人能否用3句话说清楚这个商业模式？
   - 是否理解其盈利逻辑？
4. 管理层诚信度：
   - 管理层是否言行一致？
   - 是否存在"画大饼"历史？

**关键问题**：
- 这家公司是否落入"Too Hard"堆？
- 是否在投资人的能力圈内？
- 管理层是否值得信任？

**输出要求**：`reports/月之暗面/10_munger_checklist.md`

### T11: 最终投资决策报告

**目标**：综合前述分析，给出明确投资建议

**执行内容**：

   - 买入时机
   - 退出策略
5. 持续跟踪指标：
   - 需要监控的关键指标
   - 触发重新评估的条件

**输出要求**：`reports/月之暗面/11_final_report.md`

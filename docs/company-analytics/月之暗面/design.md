# 月之暗面（Moonshot AI / Kimi）- 投研分析设计文档

## 1. 研究框架

### 1.1 分析方法论

采用 **Top-Down + Bottom-Up** 结合的分析方法：

- **Top-Down**：宏观环境 → AI行业趋势 → 竞争格局 → 公司定位
- **Bottom-Up**：产品分析 → 商业模式 → 财务质量 → 估值安全边际

### 1.2 一级市场特殊分析维度

| 维度 | 分析重点 | 权重 |
|------|---------|------|
| 商业模式 | C端流量变现、B端API、UE模型 | 25% |
| 竞争格局 | 六小龙+巨头+开源三方博弈 | 20% |
| 财务健康 | 现金跑道、烧钱效率、融资节奏 | 20% |
| 估值合理性 | 估值倍数、对标分析、隐含增长率 | 15% |
| 治理团队 | 创始人能力、团队稳定性、投资人条款 | 10% |
| 催化剂 | IPO窗口、MAU里程碑、商业化拐点 | 10% |

### 1.3 证伪机制

每个分析维度必须包含：
- **正面论点**：支持投资的核心逻辑
- **反面风险**：等权重的反驳/风险点
- **验证方法**：如何判断正反哪方成立

## 2. 月之暗面研究特殊性

### 2.1 公司画像

- **成立时间**：2023年3月
- **创始人**：杨植麟（清华系、前循环智能联合创始人）
- **核心产品**：Kimi智能助手（2023年10月发布）
- **定位**：国内首个主打超长文本的AI助手
- **融资轮次**：已完成多轮融资，估值约$3B+

### 2.2 关键研究问题

1. **流量壁垒问题**：Kimi的先发优势在巨头全面下场后是否仍有意义？
2. **商业化问题**：C端订阅+B端API的双轮驱动能否跑通？
3. **烧钱可持续性**：当前现金跑道是否安全？
4. **技术护城河**：长文本技术是否已被追平？
5. **退出路径**：IPO时间窗口与可行性

### 2.3 对标公司选择

| 对标维度 | 公司 | 对标意义 |
|----------|------|---------|
| 海外C端 | OpenAI (ChatGPT) | C端AI产品商业化天花板 |
| 海外B端 | Anthropic | B端API+安全路线 |
| 国内六小龙 | 智谱AI、MiniMax | 同代竞争对手 |
| 国内巨头 | 字节豆包、百度文心 | 巨头下场的冲击力度 |

## 4. 执行计划

### 4.1 时间安排

- 阶段一（T1-T3）：行业与竞争格局，约6小时
- 阶段二（T4-T6）：商业模式与财务估值，约6小时
- 阶段三（T7-T9）：催化剂与风险分析，约5小时
- 阶段四（T10-T11）：综合决策报告，约4小时
- **总计**：约21小时

### 4.2 质量控制

- 每个报告完成后暂停，等待用户确认
- 关键数据必须交叉验证（至少2个独立来源）
- 所有预测性结论必须标注概率和假设条件

## 5. 预期输出

- 11份结构化分析报告

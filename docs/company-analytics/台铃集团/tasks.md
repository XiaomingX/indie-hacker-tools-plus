# 台铃集团 - 执行任务清单

## 任务概览
共11个分析模块，按顺序执行，每个模块完成后暂停等待用户确认。

---

## Task 1: 宏观政策与行业分析
- **输出文件**: `1_industry_analysis.md`
- **执行内容**:
  - [ ] 电动两轮车行业政策梳理（新国标GB17761-2018、以旧换新政策、出口政策）
  - [ ] 行业市场规模测算（2020-2025年产量/销量数据）
  - [ ] 行业天花板估算（保有量、替换周期、新增需求）
  - [ ] 行业分类判定：竞争不充分的成熟市场
  - [ ] 库存/信用周期定位
  - [ ] 上游原材料（铅酸/锂电池、钢材、铝材）价格趋势
- **数据来源**: 中国自行车协会、上市公司年报、行业研报
- **预计耗时**: 1轮搜索+分析

## Task 2: 商业模式与产品分析
- **输出文件**: `2_business_model.md`
- **执行内容**:
  - [ ] 台铃产品线梳理（电动自行车/电动摩托车/电轻摩）
  - [ ] 收入结构推算（产品线、区域、渠道）
  - [ ] 品牌定位分析（"省电科技"核心卖点）
  - [ ] 护城河评估：品牌力、渠道深度、技术壁垒、成本优势
  - [ ] 与雅迪/爱玛的产品策略对比
- **数据来源**: 台铃官网、行业研报、媒体报道、产品评测
- **预计耗时**: 1轮搜索+分析

## Task 3: 竞争格局与对手对比
- **输出文件**: `3_competitor_analysis.md`
- **执行内容**:
  - [ ] 行业CR3/CR5集中度分析
  - [ ] 雅迪控股（01585.HK）详细对标：营收、利润率、渠道、战略
  - [ ] 爱玛科技（603529）详细对标
  - [ ] 新日股份（603787）、小牛电动（NIU）简要对比
  - [ ] 台铃市场排名与份额估算
  - [ ] 竞争格局演变趋势（价格战是否结束、产能出清进度）
- **数据来源**: 上市公司财报、行业研报、市场份额数据
- **预计耗时**: 2轮搜索+分析

## Task 4: 财务质量分析
- **输出文件**: `4_financial_quality.md`
- **执行内容**:
  - [ ] 台铃营收规模推算（基于行业排名和可比公司）
  - [ ] 毛利率/净利率区间推算
  - [ ] 现金流质量间接评估
  - [ ] 资产负债结构推算
  - [ ] 与雅迪/爱玛的财务指标对比
  - [ ] 关键财务风险点排查
- **数据来源**: 工商信息、可比公司财报、行业报告
- **注意**: 所有推算需标注假设前提和置信区间
- **预计耗时**: 1轮搜索+分析

## Task 5: 治理结构与管理层
- **输出文件**: `5_governance.md`
- **执行内容**:
  - [ ] 实际控制人与股权结构（企查查/天眼查）
  - [ ] 核心管理团队背景
  - [ ] 关联公司与关联交易排查
  - [ ] 司法风险（诉讼、失信、行政处罚）
  - [ ] 知识产权布局（专利数量与质量）
- **数据来源**: 国家企业信用信息公示系统、裁判文书网、专利数据库
- **预计耗时**: 1轮搜索+分析

## Task 6: 估值与安全边际
- **输出文件**: `6_valuation.md`
- **执行内容**:
  - [ ] 可比公司估值汇总（PE/PB/PS/EV/EBITDA）
  - [ ] 台铃估值区间测算（PE法、PS法）
  - [ ] DCF粗估（基于行业增速假设）
  - [ ] 敏感性分析（WACC、增速假设变动）
  - [ ] 安全边际计算
  - [ ] IPO估值预期分析（如有）
- **数据来源**: 上市公司实时估值、行业倍数
- **预计耗时**: 1轮搜索+分析

## Task 7: 催化剂与拐点分析
- **输出文件**: `7_catalysts.md`
- **执行内容**:
  - [ ] IPO进程与时间窗口
  - [ ] 新国标换车潮催化（存量替换需求）
  - [ ] 以旧换新补贴政策落地
  - [ ] 海外市场拓展（东南亚、印度等）
  - [ ] 锂电化/智能化技术升级
- **数据来源**: 政策文件、行业新闻、公司动态
- **预计耗时**: 1轮搜索+分析

## Task 8: 逆向分析（Pre-mortem）
- **输出文件**: `8_reverse_analysis.md`
- **执行内容**:
  - [ ] 假设投资亏损80%的场景构建
  - [ ] 价格战加剧导致利润率崩塌风险
  - [ ] 新国标执行不及预期风险
  - [ ] 锂电池安全事故品牌冲击风险
  - [ ] 管理层/治理风险
  - [ ] 行业需求见顶风险
  - [ ] 海外拓展失败风险
- **数据来源**: 历史行业事故案例、竞争格局分析
- **预计耗时**: 1轮分析

- **执行内容**:

  - [ ] 台铃近期负面新闻梳理

  - [ ] 消费者口碑分析（电商平台评价、社交媒体）
  - [ ] 品牌舆情风险评估
- **数据来源**: 新闻媒体、社交平台、电商评价
- **预计耗时**: 1轮搜索+分析

- **预计耗时**: 1轮综合分析

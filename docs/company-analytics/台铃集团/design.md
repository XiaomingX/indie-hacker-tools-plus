# 台铃集团 - 分析方案设计文档

## 分析框架
基于 touyan-agent 价值投资框架（巴菲特/芒格），结合台铃集团非上市公司的特点进行适配。

## 方法论

### 数据获取策略
| 数据类型 | 来源 | 可靠性评级 |
|---------|------|-----------|
| 行业规模与趋势 | 中国自行车协会、艾瑞/沙利文行业报告、上市公司年报 | 高 |
| 竞争格局 | 雅迪/爱玛/新日财报、行业研报 | 高 |
| 台铃财务概要 | 工商信息（企查查/天眼查）、媒体报道、行业排名 | 中 |
| 治理与合规 | 国家企业信用信息公示系统、裁判文书网、行政处罚公示 | 高 |
| 估值对标 | 港股雅迪、A股爱玛/新日、纳斯达克小牛的公开估值 | 高 |
| 产品与技术 | 官网、产品评测、专利数据库 | 中 |
| 舆情与新闻 | 财经媒体、社交媒体、消费者论坛 | 中低 |

### 分析方法

#### 1. 行业分析方法
- 自上而下：宏观政策 → 行业规模 → 细分市场
- 电动两轮车行业分类为"竞争不充分的成熟市场"
- 重点分析新国标带来的存量替换周期

#### 2. 竞争分析方法
- 以雅迪（行业第一）和爱玛（行业第二）为核心对标
- 采用波特五力模型分析竞争强度
- 重点对比：渠道覆盖、产品定位、价格带、技术路线

#### 3. 财务分析方法
- 由于非上市，采用间接推算法：
  - 通过行业排名推算营收规模区间
  - 通过可比公司毛利率推算盈利水平
  - 通过产能/门店规模推算运营效率
- 标注所有推算的假设前提和置信区间

#### 4. 估值方法
- **可比公司法（核心）**：
  - PE法：对标雅迪/爱玛的PE区间
  - PS法：对标行业PS中位数
  - EV/EBITDA：对标行业EV/EBITDA
- **DCF辅助**：
  - 基于行业增速假设未来现金流
  - WACC取值参考可比公司
  - 敏感性分析（乐观/中性/悲观）

#### 5. 逆向分析方法
- 事前验尸（Pre-mortem）：假设投资亏损80%，倒推原因
- 关注行业特有风险：安全事故、政策变化、价格战

## 报告结构设计

每份报告遵循统一结构：
1. 核心结论（Executive Summary）
2. 详细分析
3. 关键发现与风险提示
4. 数据来源与假设说明

## 质量控制
- 所有推算数据必须标注假设前提
- 财务数据与可比公司交叉验证
- 信息降权：管理层表态和媒体宣传强制打折
- 证伪优先：首要任务是排雷而非找亮点

## 执行节奏
- 每完成一个模块暂停，等待用户确认
- 用户可补充最新信息或修正假设
- 最终报告在所有模块确认后综合产出

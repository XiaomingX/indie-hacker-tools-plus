# 0. 行业定义、TAM与产业链全景拆解

## 业务属性界定
大模型基础设施厂商（如智谱、月之暗面、MiniMax等）的商业模式基础为：购入通用计算硬件资源（GPU），通过模型预训练与微调，转化为自然语言处理及推理能力（即 Token），随后按次调用计费或通过项目制交付变现。在财务本质上，这属于高度依赖资本开支（Capex）的重资产数据加工模型。

## 产业链产值分布与利润转移机制
1. **上游算力供给方（获取产业主要资金流）**
   - **角色**：核心加速卡供应商（如英伟达）、公有云服务商。
   - **状态与利润率**：处于相对卖方市场。硬件毛利率在 70% 以上，云算力租赁毛利率通常维持在 20%-30% 之间。是目前该产业链条中具备明确正向现金流的环节。
2. **中游大模型基座厂商（核心资本消耗节点）**
   - **角色**：独立通用大模型企业及大厂自研团队。
   - **状态与利润率**：面临极高的固定资产摊销。由于商业化进程尚未覆盖基础设施的资本支出折旧，且受制于激烈的定价竞争，目前该环节的主流厂商处于整体净利亏损状态，纯推理端 API 毛利多处在盈亏平衡点以下甚至负值区间。
3. **下游 AI 应用层**
   - **角色**：基于基座模型开发的各类终端工具应用企业。
   - **状态与利润率**：作为模型能力的调用方，利润表现直接受制于上游 API 的定价变动。其商业模式多为轻资产的 SaaS 订阅或广告变现，但同质化竞争严重，获客成本（CAC）偏高。

## 产业链结构评述
该产业现阶段的核心资金运转机制为：外部风险投资与战略投资投入中游基座厂商，这部分资金多数转化为资本支出，进而流向上游算力提供方。中游环节在实现具备自我造血能力的规模化商业收入前，承担着较高的流动性与技术迭代风险。

# 4. 行业产能周期与资本开支节奏

## 资本支出的前置性与报废的高频性
大模型行业的基础设施投入具备两个显著的财务属性：

- **资本支出（Capex）的阶梯式前置**：训练下一代大模型（如万亿级或多模态）必须在开始前完成算力集群（通常为数千至上万张 GPU）的采购或租赁部署。这种开支具有刚性，且随着模型参数量的扩大呈现指数级增长。
- **核心资产的快速损耗**：底层算法架构与模型的迭代速度极快。基于上一代算法训练出的模型价值，在开源社区发布更新的底层模型后，往往出现大幅缩水。对应的算力设备如果在 2-3 年内未能通过推理商业化产生正规现金回流，将面临实质性的资产减值风险。

## 融资环境与周期传导
- 行业发展高度依赖一级市场（PE/VC）注资。随着 2024-2025 年资本回归理性，部分独立大模型厂商已启动在公开二级市场（如港股）的 IPO 进程。
- 这一资本化动作反映出前期一级市场投资人对流动性退出的诉求，同时也表明维持模型迭代所需的资金规模已超出私募融资的常规承受区间。

## 周期风险提示
由于资本支出（用于训练）无法削减且必须前置，而商业收入（推理收费）回笼缓慢。行业在未达到供需平衡点前，必然长期处于净现金流失血状态。这一属性决定了该行业不具备传统周期成长股的投资安全垫，而是一种具有期权性质的高风险标的。

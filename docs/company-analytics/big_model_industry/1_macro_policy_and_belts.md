# 1. 宏观政策、供需硬缺口与（中国）产业带分析

## 产业地理聚集特征
行业资源呈现向极少数一线城市集中的趋势，核心驱动因素为顶尖算法人才的分布及政策侧的支持力度：

1. **北京（海淀等区域）**
   - **聚集基础**：依托清华大学、北京大学等高校资源及中科院所，具备密集的高阶算法人才池。
   - **政策配套**：地方政府在算力统筹调度与需求侧给予了具体支持，例如面向辖区企业的“算力券”补贴政策。此类政策在一定程度上缓解了模型厂商早期的算力成本压力。
2. **上海（徐汇/浦东等区域）**
   - **聚集基础**：产业数字化需求丰富，更贴近长三角制造业、游戏、金融及消费互联网应用端。
   - **配套特征**：地方在算力基础设施建设（如智算中心投用）及数据流通机制上提供产业配套，利于企业在商业化应用场景及 C 端产品的探索与落地。

## 政策环境与监管要求
- **准入机制**：国内实行生成式 AI 备案制度。模型上线对数据语料及算法具备双重审核要求，客观上提升了从事基座大模型研发的合规门槛。
- **算力补贴**：多地政府出台的算力采购补贴（算力券），其底层经济学含义为利用财政手段平抑模型研发极高的固定成本开支，反映出纯商业化的算力成本目前仍超出部分初创企业的自然承受能力。

## 供需数据量化状态
- **基座供给**：目前已通过备案的大模型数量在两百个量级，能够提供通用文本处理能力的底层模型面临同质化竞争及局部供给过剩。
- **商业需求**：具备高复购率与强付费意愿（能够覆盖调用成本）的 ToB 落地场景仍在验证阶段；多数通用 API 调用量来自于低毛利或亏损获客的需求测试。总体处于供给能力超前于真实变现需求的阶段。

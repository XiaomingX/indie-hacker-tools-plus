# 5. 财务真实质量与 ROIC 杜邦归因

## 核心财务指标剖析
在评估通用大模型基础设施企业时，由于绝大多数尚未实现盈利，传统的 ROIC（投入资本回报率）及 NOPAT（息税前净利润）分析体系需转换为对负向财务指标的监控：

1. **净利率 (Margin) 压力来源**
   - **成本刚性**：单次推理调用的云端算力、带宽和摊销的研发开支构成固定成本底线。
   - **收入端通缩**：受同业竞争及互联网巨头云业务引流策略的影响，API 服务定价处于下降通道，行业整体推理端毛利率在较长时间内承压，甚至表现为负数。

2. **资产周转率 (Turns) 的低效表现**
   - 大额算力资产（以预训练为主的集群购买或长期租赁租赁合同）在模型发布后，若未获得充足的 C 端或 B 端日活跃调用量，则算力资源处于部分空置状态。资产周转率偏低进一步放大固定资产折旧对当期损益的负面影响。

## 自由现金流 (FCF) 的系统性缺口
- **经营性现金流 (CFO)**：除了极少部分实现 C 端出海高客单价订阅或承接大型 B 端私有化开发项目的厂商外，多数基座模型企业的日常经营现金收入无法覆盖员工薪酬及租金支出。
- **自由现金流 (FCF) 评估**：经营现金流为负叠加持续刚性的高额资本开支（Capex），导致自由现金流表现为系统性且高额的流出。企业的存活周期完全取决于账面融资结余及后续的外部输血能力。

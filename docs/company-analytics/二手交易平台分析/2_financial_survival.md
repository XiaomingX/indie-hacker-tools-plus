# 2. 财务生存能力与单体模型对决

## 谁在流血？谁在收租？

在二手电商这个看似毛利丰厚的赛道里，隐藏着极其残酷的单体经济学 (Unit Economics)。

### 1. 闲鱼：极致的“轻量化”收租模式
- **单体模型**：闲鱼几乎不参与物流和质检，其单笔交易的边际成本接近于零。
- **财务特征**：
  - **毛利率**：预计在 80% 以上（如果仅计算服务费收入对冲运营成本）。
  - **造血能力**：由于背靠阿里，其服务器和支付成本被大幅平摊。闲鱼不需要为了生存而烧钱，它的存在本身就是为了提升淘宝的生态粘性。
- **生存现状**：极度安全。

### 2. 转转：高毛利幻象下的“履约地狱”
- **单体模型**：转转的单笔交易（以手机为例）毛利看似很高（质检溢价），但必须扣除：
  1. **逆向物流费**（双向包邮）。
  2. **质检人力的时间成本**。
  3. **库存减值损失**（自营回收部分，手机跌价极快）。
  4. **线下门店租金与人力摊销**。
- **财务特征**：
  - **毛利率**：名义毛利可能达 20%，但扣除履约成本（Fullfillment）后的**边际利润（Contribution Margin）极低**。
  - **烧钱率 (Burn Rate)**：为了维持数以千计的质检员和数百家线下门店，转转每月的刚性开支巨大。
- **生存现状**：**高危**。转转必须维持极高的交易规模才能覆盖固定成本。一旦市场份额被闲鱼的“验货宝”蚕食，转转会迅速陷入现金流枯竭。

### 3. 得物：高溢价带来的“品牌税”
- **单体模型**：得物的客单价极高（潮流属性）。虽然同样需要高昂的鉴定人力，但其单笔收取的服务费（通常 5%-10% 佣金 + 数十元鉴定费）足以覆盖履约成本。
- **财务特征**：
  - **现金流**：极好。由于“先收钱再鉴定”的模式，得物账面上常年沉淀着海量资金。
  - **盈利能力**：是垂直电商中极少数能实现稳定规模化盈利的企业。
- **生存现状**：稳健。其挑战在于如何维持这种高溢价的潮流光环，防止品牌稀释。

## 财务指标对比 (2025 预估)

| 指标 | 闲鱼 | 转转 | 得物 |
| :--- | :--- | :--- | :--- |
| **毛利率 (加回履约成本前)** | >85% | ~25% | ~45% |
| **履约成本占营收比** | <5% | >20% | ~15% |
| **获客成本 (CAC)** | 极低 (淘系输血) | 高 (需在社交媒体买量) | 中 (社区自然留存高) |
| **现金流健康度 (Runway)** | 无限 (阿里奶妈) | 依赖融资/高周转 | 强劲 (预收货款多) |

## 结论：谁的财务逻辑更像“生意”？

- **闲鱼** 是典型的**互联网地租生意**，稳赚不赔。
- **得物** 是典型的**品牌溢价生意**，靠信誉收税。
- **转转** 更像是一个**效率极低的传统二手批发商**，它在用高科技的外壳包裹着极其沉重且低效的物流/人力账本。在目前的资本环境下，转转的财务脆弱性最为突出。

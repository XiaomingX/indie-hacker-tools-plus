# 二手交易平台竞品分析设计文档

## 1. 分析框架设计

### 1.1 价值投资分析框架

| 维度 | 评估内容 |
|------|---------|
| 时间窗口 | 创立时机与时代红利 |
| 商业模式 | 营收结构与护城河 |

## 2. 各平台禀赋识别

| 平台 | 评级 | 核心禀赋 |
|------|------|---------|
| 闲鱼 | AAA | 阿里系+流量+二手心智 |
| 得物 | AA | 鉴定壁垒+潮流电商 |
| 识货 | AA | 虎扑+运动鉴定 |
| 转转 | AA | 回收+C2B模式 |
| 什么值得买 | A | 消费决策+导购 |
| 慢慢买 | A | 比价工具 |

---

## 3. 护城河评级标准

- **无形资产**: 品牌、鉴定能力 (0-25)
- **转换成本**: 用户习惯沉淀 (0-20)
- **网络效应**: 买家卖家双边网络 (0-25)
- **成本优势**: 规模经济 (0-15)
- **定价权**: 佣金率与变现 (0-15)

### 综合评级
- **AAA (80+)**: 极强的持久竞争优势
- **AA (60-79)**: 较强竞争优势
- **A (40-59)**: 中等竞争优势

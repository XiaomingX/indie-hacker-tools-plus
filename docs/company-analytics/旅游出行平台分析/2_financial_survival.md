# 2. 财务生存能力与单体模型对决

## 谁在数钱？谁在补贴？

OTA 行业的财务逻辑是“单次撮合收益”与“流量成本”的残酷算术。

### 1. 携程：高毛利的“盈利收割机”
- **单体模型**：高客单价、高佣金率。出境游和商旅业务的利润极其丰厚。
- **财务特征**：
  - **毛利率**：长期维持在 80% 左右。
  - **投资收益**：携程不仅自己赚钱，还通过投资全球各地的 OTA 平台（如 MakeMyTrip）获取丰厚分红。
- **生存现状**：极度安全。其账面现金流足以支撑其在全球范围内发起任何形式的价格战。

### 2. 同程旅行：极致的“资本效率”
- **单体模型**：轻资产、低毛利（受限于低单价）、低费用。由于不需要高额买量，其净利率表现异常亮眼。
- **财务特征**：
  - **ROE (净资产收益率)**：在三者中处于高位，因为其不需要保留大量资金进行流量博弈。
  - **资产周转率**：极高。
- **生存现状**：稳健。

### 3. 美团酒旅：本地生活的“现金奶牛”
- **单体模型**：低毛利、极高周转。酒旅业务是美团内部利润率最高、现金流最好的板块，长期为外卖和新业务提供“弹药”。
- **财务特征**：
  - **边际成本**：接近于零。在美团 APP 内，酒店订单的增加几乎不需要额外的营销支出。
  - **财务防御力**：由于其与高频外卖业务绑定，其抗周期能力行业第一。
- **生存现状**：极度安全。

## 财务指标对比 (2025 预估)

| 指标 | 携程 | 同程旅行 | 美团酒旅 |
| :--- | :--- | :--- | :--- |
| **净利率** | **~25%** | ~15% | ~20% |
| **单用户变现价值** | **最高** | 低 | 中 (高频次弥补低单价) |
| **营销费用占比** | 高 (需要维持品牌存在感) | **极低** | 极低 |
| **现金储备** | **卓越** | 良好 | 卓越 |

## 结论：谁的财务逻辑更具韧性？

- **美团酒旅** 是**“最难被杀死的对手”**。它的利润来自于生态的协同，对手很难通过单一维度的价格战打垮它。
- **携程** 是**“最有质量的赢家”**。它的利润来自于对高端需求的绝对垄断，具有极高的壁垒溢价。
- **同程旅行** 是**“最高效的拾荒者”**。它在利用大平台的剩余流量赚取丰厚回报，但其缺乏独立的财务抗压性。

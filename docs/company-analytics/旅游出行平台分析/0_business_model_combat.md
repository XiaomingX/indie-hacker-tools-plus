# 0. 商业模式与基本盘对决：携程 vs 同程旅行 vs 美团酒旅

## 核心赚钱逻辑：一站式操作系统 vs 流量寄生 vs 高频带低频

旅游出行行业的商业模式本质是**“对稀缺库存的数字化分配权”**。由于流量获取方式的不同，这三巨头在产业链中的地位有着天壤之别：

1. **携程 (Trip.com)**：**极致的“全球旅行管家”**。
   - **底层逻辑**：携程是中国乃至全球最成熟的 OTA（在线旅游代理）平台。它不生产门票或酒店，但它生产**“确定性”**。通过深耕供应链 20 年，携程建立了最厚的国际航权和高星酒店库存。其基本盘在于**高净值商旅用户**和**全球化履约能力**。它赚的是**“全流程服务的重中介费”**。
2. **同程旅行 (Tongcheng)**：**下沉市场的“流量杠杆”**。
   - **底层逻辑**：同程旅行是典型的**“流量寄生者”**。它深度绑定腾讯生态（微信九宫格），以极低的获客成本获取了大量非一线城市的“小白”用户。其核心是**“效率匹配”**。它赚的是**“极低获客成本下的佣金差额”**。其基本盘在于**微信生态的排他性入口**。
3. **美团酒旅 (Meituan)**：**本地生活下的“效率收割机”**。
   - **底层逻辑**：美团奉行的是**“高频带低频”**。通过送外卖这种高频动作，顺带手解决了用户的本地酒店和门票预订。它不追求高端出境游，它追求的是**“本地 5 公里内的闭环”**。它赚的是**“流量边际成本为零的利润”**。其基本盘在于其**庞大的地推团队**和**本地生活服务的绝对垄断**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | 携程 | 同程旅行 | 美团酒旅 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 高星酒店 (40%) + 交通 (35%) + 国际业务 | 交通票务 (核心流量入口) + 住宿 (主要利润) | 到店、酒店及旅游佣金 |
| **单用户价值 (ARPU)** | **最高** (高客单价出境游、商旅) | 中 (主要为性价比产品) | 中低 (本地生活消费为主) |
| **毛利属性** | **最高** (轻资产运营，高毛利国际业务占比升) | 高 (获客成本极低，运营杠杆高) | 高 (作为美团内部的“提款机”，毛利稳定) |
| **变现瓶颈** | 反垄断监管压力，国际政治风险 | 对微信流量的依赖，下沉市场竞争加剧 | 抖音、小红书在种草环节的截流 |

## 结论：基本盘的稳固度

- **携程** 的基本盘最“硬”，其全球供应链的深度和高端用户的忠诚度，短时间内无法被纯流量平台（抖音、美团）撬动。
- **美团酒旅** 的基本盘最“活”，只要外卖还在送，酒旅的流量就是白给的，其成本优势是所有对手的噩梦。
- **同程旅行** 的基本盘最“虚”，一旦腾讯流量策略调整或微信入口不再唯一，同程必须证明自己在脱离微信后还能独立行走。

# 4. 极端压力测试与死法推演 (Pre-mortem)

## 假如明天是末日：谁会第一个倒下？

机场作为无法移动的基础设施，最怕的是“流量的永久性枯竭”。

### 场景一：免税渠道彻底转向“线上+市内”（商业逻辑崩坏）
- **背景**：离岛免税、市内免税及跨境电商彻底取代机场免税，机场沦为单纯的安检通道。
- **上海机场**：**毁灭性打击**。虽然不会破产（有航空性收入垫底），但其目前 1000 亿+ 的估值是基于其“非航变现能力”的。在该场景下，上机将失去 80% 的估值溢价，沦为一个普通的低回报路桥公司。
- **白云机场/深圳机场**：**中伤**。它们受到的估值冲击会由于原本依赖度较低而相对可控。

### 场景二：深中通道与高速铁路网的“二重冲击”（客流分流）
- **背景**：深中通道通车导致大量客流流向深圳或广州；同时，长途高铁将 1000 公里以内的客流吞噬殆尽。
- **深圳机场**：**重伤**。在地理位置上，深圳机场最容易受周边陆路交通的影响。如果客流下滑 20%，其非航变现价值将呈指数级萎缩。
- **上海机场**：**稳健**。上海作为经济终点站，其客流的不可替代性极强。

### 场景三：全球航空燃油费暴涨导致的“平价飞行”终结
- **背景**：能源价格飙升使机票价格翻倍，航空从平民化回归精英化。
- **白云机场**：**重伤**。其庞大的吞吐规模依赖于价格敏感的国内旅客。一旦飞行变成奢侈，白云机场的规模红利将瞬间变成运营赤字。
- **上海机场**：**避风港**。其服务的商务旅客对价格相对不敏感。

## 死亡排序预测

**第一个倒下：上海机场 (估值神话终结)**
- **理由**：它是“过度溢价”的典型。它所有的光环都建立在“免税收租”这一个单一支柱上。一旦支柱被政策或渠道拆除，它就是那个裸泳的人。

**第二个倒下：深圳机场 (受区域竞争影响)**
- **理由**：它是大湾区最活跃但也最不稳固的一环。面对广州和香港的夹击，它的生存空间必须靠不断创新来“抢”。

**最后赢家：白云机场**
- **理由**：**“规模即生存”**。它是中国最扎实的综合交通枢纽，其航网结构的深度和国内基本盘的广度，为其提供了最厚的安全垫。

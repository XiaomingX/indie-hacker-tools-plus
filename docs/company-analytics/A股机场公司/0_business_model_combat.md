# 0. 商业模式与基本盘对决：上海机场 vs 白云机场 vs 深圳机场

## 核心赚钱逻辑：免税收租者 vs 规模增长者 vs 物流广告商

机场行业的商业模式本质是**“航空性业务引流 + 非航业务变现”**。然而，由于流量属性（国际 vs 国内）和变现渠道的不同，三巨头的基本盘已发生深刻分化：

1. **上海机场 (Shanghai Airport)**：**极致的“高净值流量税收站”**。
   - **底层逻辑**：上海机场（浦东+虹桥）是中国国际航线最密集的枢纽，其基本盘是**极高溢价的国际旅客流**。它曾是典型的“免税收租模式”，靠着国际旅客在日上免税店的消费扣点躺着赚钱。虽然 2024 年免税协议重签（从高扣点转向低保底），但其作为中国与全球连接点的地位依然是其最核心的收租权。
2. **白云机场 (Baiyun Airport)**：**稳健的“南大门枢纽”**。
   - **底层逻辑**：白云机场靠的是**极致的吞吐规模与综合成本控制**。其基本盘是庞大的国内客流量和持续恢复的东南亚航线。它的变现模式比上机更均衡，不单一依赖免税，而是通过航空性收费（起降费）和多元化的非航业务（商业租赁）来实现稳健利润。
3. **深圳机场 (Shenzhen Airport)**：**活力的“跨境电商中转站”**。
   - **底层逻辑**：深圳机场的基本盘在于**珠三角活跃的经济动能与爆发式的跨境物流**。它不追求免税神话，而是走了一条“广告+货运”的差异化道路。受益于希音（Shein）、特姆（Temu）等跨境电商的崛起，其货运业务已成为其独特的利润增长极。

## 营收与成本结构对比 (2025 预估)

| 指标 | 上海机场 | 白云机场 | 深圳机场 |
| :--- | :--- | :--- | :--- |
| **主要利润来源** | 免税扣点 (虽降但仍是大头) + 航空服务 | 航空性收费 + 商业租赁 | 广告收入 + 航空物流 + 航空服务 |
| **客流属性** | **高净值** (商务、国际旅游占比最高) | 高规模 (国内中转、商旅) | 高活力 (大湾区通勤、货运密集) |
| **免税依赖度** | **极高** (利润波动与免税政策高度正相关) | 中 (主要作为配套服务) | 低 (几乎不作为核心利润点) |
| **成本结构** | 极高的资产折旧 (浦东三期等) | 持续改扩建带来的折旧压力 | 相对较轻的资产包，高效率的运营 |

## 结论：基本盘的稳固度

- **上海机场** 的基本盘是“不可替代的国际门户”，只要中国还需要与世界做生意，它的高净值流量价值就永远存在。
- **白云机场** 的基本盘是“规模韧性”，在消费降级趋势下，其国内枢纽的地位比上海机场更具抗波动性。
- **深圳机场** 的基本盘是“产业溢价”，它深度绑定了大湾区的制造业出海，是三者中唯一具有“物流成长性”的标的。

# 2. 财务生存能力与单体模型对决

## 谁在收租？谁在还债？

电力行业是典型的资本密集型行业，财务报表的真相往往隐藏在“折旧”与“利息”之下。

### 1. 长江电力：进入“收获期”的提款机
- **单体模型**：水电站初始投资巨大，但一旦建成，其运维成本微乎其微。
- **财务特征**：
  - **ROIC (资本回报率)**：随着大坝折旧完毕，ROIC 将呈现爆发式上升。长电已进入现金流归还债务、财务费用骤降的良性循环。
  - **造血能力**：经营性现金流远超净利润。它是 A 股最典型的“高分红、低负债（指趋势下降）”标杆。
- **生存现状**：极度安全。

### 2. 中国核电：高负债支撑的稳健增长
- **单体模型**：核电建设需要十年以上的长周期。目前的盈利主要靠在运机组，但在建机组占据了大量现金流并产生了沉重的财务杠杆。
- **财务特征**：
  - **资产负债率**：长期维持在 70% 以上的高位。
  - **成长逻辑**：它的利润增长不靠效率，靠的是“新机组并网”。这是一种“不断滚动杠杆”的扩张模式。
- **生存现状**：安全，但缺乏爆发力。其利润释放受制于高额的财务费用支出。

### 3. 三峡能源：被电价波动的“重资产”绞杀
- **单体模型**：新能源单瓦投资额虽在下降，但其发电小时数（1000-2000小时）远低于水电（4000小时）和核电（7500小时）。这意味着其资本回收期被拉得极长。
- **财务特征**：
  - **盈利脆弱性**：随着新能源全面进入电力市场化交易，电价下行压力巨大。三峡能源在 2024-2025 年面临着“增量不增利”甚至“单瓦利润下降”的窘境。
  - **补贴依赖**：账面上依然挂着不少未到位的政府补贴，现金流质量劣于长电。
- **生存现状**：**中危**。其生存极度依赖低息融资环境。一旦利率上行，其高杠杆扩张模式将面临崩盘。

## 财务指标对比 (2025 预估)

| 指标 | 长江电力 | 中国核电 | 三峡能源 |
| :--- | :--- | :--- | :--- |
| **资产负债率** | ~60% (逐年下降) | ~72% (高位波动) | ~68% (持续扩张) |
| **净利率** | ~40% (极高) | ~20% | ~25% |
| **股息率 (预期)** | ~3.5% - 4.0% | ~1.5% - 2.0% | < 1.0% |
| **经营性现金流/净利润** | > 1.8 | ~1.3 | ~1.1 |

## 结论：谁的财务逻辑更像“年金”？

- **长江电力** 是**真正的红利资产**。其财务逻辑是“减少投入、增加分红”。
- **中国核电** 是**重工业投资品**。其财务逻辑是“持续投入、缓慢分红”。
- **三峡能源** 是**风险溢价产品**。其财务逻辑是“高频投入、博弈未来”。在目前的宏观环境下，长电的确定性对大资金具有绝对的统治力。

# 4. 极端压力测试与死法推演 (Pre-mortem)

## 假如明天是末日：谁会第一个倒下？

我们设定三个极端毁灭性场景，观察三家电力巨头的生存韧性：

### 场景一：全球气候剧变导致的长期干旱
- **背景**：长江流域出现长达五年的极枯年份，降雨量减少 50%。
- **长江电力**：**重伤**。发电量骤减将直接导致净利润腰斩，甚至无法覆盖其维持高分红承诺的现金。虽然资产还在，但其作为“红利资产”的估值逻辑将瞬间崩溃，股价面临 30%-50% 的向下重估。
- **中国核电**：**获益/稳健**。干旱对核电几乎没有影响。在水电缺位的情况下，核电的基荷价值更加突显。
- **三峡能源**：**中伤**。如果干旱伴随着无风天气，其业绩也会受损，但影响范围通常不局限于单一流域。

### 场景二：电力市场化交易导致电价崩盘（负电价时代）
- **背景**：由于新能源过剩，中午时段电价跌至负值，且政府取消了水电、核电的优先电价保护。
- **三峡能源**：**必死/深度亏损**。其成本结构决定了它必须维持一定的上网电价才能覆盖利息和折旧。一旦进入大规模负电价时代，三峡能源账面上的大量风光资产将沦为负债。
- **中国核电/长江电力**：**中伤**。它们拥有极低的边际成本，即便电价跌到一分钱，它们依然能产生正向经营性现金流。由于它们通常连接跨省特高压，具备一定的市场化议价权，但利润空间会被严重压缩。

### 场景三：核安全事故或大坝安全事故
- **背景**：全球发生重大核事故或极端地质灾害冲击。
- **中国核电**：**毁灭性打击**。核电站的运营将被无限期关停，企业将面临巨额的赔偿与清理成本，估值清零。
- **长江电力**：**毁灭性打击**。大坝安全是其绝对红线。
- **三峡能源**：**避风港**。分散式的风光电站几乎没有系统性安全事故风险。

## 死亡排序预测

**第一个倒下：三峡能源**
- **理由**：它是电力市场化内卷中的第一牺牲品。在没有行政保护的情况下，其脆弱的 Unit Economics 无法抵御电价的长期下行。

**第二个倒下：长江电力 (受气候影响)**
- **理由**：它虽然硬，但“靠天吃饭”是其最大的阿喀琉斯之踵。单一地理位置的依赖使其缺乏系统性的抗灾对冲。

**最后赢家：中国核电 (前提是安全)**
- **理由**：它是工业社会最稳定、最高密度的能量输出端。只要不发生系统性事故，它就是电网无可替代的底层逻辑。

# 0. 行业定义、TAM与产业链全景拆解

## 业务属性界定：伪装成软件的算力消耗品
全球顶级大模型基础设施（以 OpenAI、Anthropic 为代表）在表象上提供软件订阅（SaaS）和 API 服务，但在财务和物理本质上，它们是**“高度浓缩的电力与硅片折旧贩卖商”**。
它们通过千亿美元级别的极高资本开支（Capex），购买数以十万计的 GPU 集群，将电力和芯片寿命“炼制”成智能模型权重（Token 预测能力），然后再零售给 C 端和 B 端。这是一门“以算力换智能”的重资产加工业。

## 产业链产值分布与极度畸形的利润剥削
全球大模型生态链的利润分布呈现绝对的“倒金字塔”结构：

1. **底层算力霸权（吸走全行业 >70% 的真实利润）**
   - **角色**：NVIDIA（唯一的卖水人霸主）、台积电（代工垄断）。
   - **商业模式**：硬件一次性买断。NVIDIA 的数据中心业务毛利率常年维持在 75% 以上。不论 OpenAI 和 Anthropic 是亏是赚，它们融到的每一分钱最终大部分都变成了 NVIDIA 的净利润。
2. **算力租赁与云基础设施（截留 >20% 的价值）**
   - **角色**：Microsoft (Azure)、Amazon (AWS)、Google Cloud。
   - **商业模式**：由于 OpenAI 与 Anthropic 无法全量自建机房，必须与云巨头绑定（微软之于 OpenAI，亚马逊之于 Anthropic）。云巨头不仅以算力抵扣作为投资入股，还在模型推理阶段按量抽取高昂的云端税。它们既是股东，又是最大的供应商。
3. **顶尖模型基座层（惊人的 ARR 与更惊人的净亏损）**
   - **角色**：OpenAI、Anthropic。
   - **商业模式**：向 C 端收 20 美元订阅费，向 B 端卖 API。尽管到 2026 年初两家公司的 ARR（年化经常性收入）分别飙升至超 200 亿和逼近 300 亿美元，但其每年的算力开支、推理耗电与顶尖人才薪酬合计更是天文数字。目前全处于实质性的**巨额失血状态**。

## 价值分配结论
在这个万亿级别的生态中，真正的赢家是卖显卡和卖电的人。大模型基座厂商目前更像是一个“吸金黑洞”，它们负责向全球吸筹资金，然后无偿转交给底层算力垄断者。

# 5. 财务真实质量与 ROIC 杜邦归因

## 千亿美元规模的“失血巨兽”
尽管这两家巨星企业的收入增速创造了人类商业史的奇迹，但从纯粹的财务质地来看，它们是典型的“规模毁灭价值”模型。

1. **不可弥合的毛利缺口 (Margin)**
   - 极其高昂的 API 调用成本（云算力、芯片折旧、电费）构成了刚性的成本底线。由于 Meta 开源模型的疯狂倾销逼迫，它们无法将真实成本完全转嫁给下游开发者。两家企业的整体真实毛利率在扣除全口径基础设施折旧后，极度微薄甚至长期为负。

2. **极端的资产负债表与失衡的周转率 (Turns)**
   - **重资产化**：它们的资产负债表上堆满了快速贬值的 GPU 集群或长期算力租赁负债。
   - 算力资源的周转率存在天然的峰谷浪费。为了随时响应全球数亿用户的查询（ToC 业务），必须长期保持海量的冗余算力在线空转。

3. **自由现金流 (FCF) 的深渊**
   - **无情的数据**：OpenAI 2026 年预估净现金流失血 170 亿美元；Anthropic 预估失血 60 亿美元。
   - **ROIC 定性**：在自由现金流转正遥遥无期的前提下，谈论其投入资本回报率（ROIC）是毫无意义的。企业完全依靠一轮又一轮极其夸张的一级市场融资（数百亿美元级别）来维持生命体征。这不是一家正常经营的企业，更像是一项由全球科技寡头联合出资的登月工程。

## 1. 分析框架设计

### 1.1 价值投资分析框架

基于巴菲特/芒格投资哲学，构建以下分析框架：

```
┌─────────────────────────────────────────────────────────┐
│                  内在价值评估体系                          │
├─────────────────────────────────────────────────────────┤
│  一、时间维度（When & Why Now?）                          │
│    ├─ 创建时机与时代红利捕捉                              │
│    ├─ 关键战略转折点                                      │
│    └─ 当前时间窗口的边际变化                               │
├─────────────────────────────────────────────────────────┤
│  二、商业模式（Moat Analysis）                            │
│    ├─ 营收结构与现金流来源                                │
│    ├─ 护城河类型识别                                      │
│    └─ 长期竞争优势可持续性                                  │
├─────────────────────────────────────────────────────────┤
│  三、竞争格局（Competitive Position）                      │
│    ├─ 市场定位与差异化                                     │
│    ├─ 行业竞争壁垒                                         │
│    └─ 格局演变预判                                         │
├─────────────────────────────────────────────────────────┤
│  四、治理结构（Governance）                                │
│    ├─ 股权结构与利益一致性                                  │
│    ├─ 激励机制有效性                                       │

├─────────────────────────────────────────────────────────┤
│  五、财务质量（Financial Quality）                        │
│    ├─ 自由现金流 vs 净利润                                 │
│    ├─ ROE质量拆解                                         │
│    └─ 资产健康度                                           │
├─────────────────────────────────────────────────────────┤
│  六、风险评估（Pre-mortem）                                │
│    ├─ 致命缺陷识别                                        │
│    ├─ 颠覆性威胁                                          │

```

## 3. 护城河评级标准

### 护城河类型量化评分
- **无形资产**: 版权壁垒、品牌认知度（0-25分）
- **转换成本**: 用户粘性、作者绑定（0-20分）
- **网络效应**: 用户互动、社区生态（0-25分）
- **成本优势**: 规模经济、边际成本（0-15分）
- **定价权**: 提价能力、利润率（0-15分）

### 综合评级
- **AAA (80+)**: 极强的持久竞争优势
- **AA (60-79)**: 较强竞争优势
- **A (40-59)**: 中等竞争优势
- **BBB (20-39)**: 较弱竞争优势
- **BB (<20)**: 几乎无护城河

## 4. 时间窗口分析框架

### 4.1 宏观周期判断
- 2010-2015: 移动互联网爆发期
- 2015-2019: 付费阅读崛起期
- 2019-2023: 免费阅读冲击期
- 2023-至今: 短视频融合期

### 4.2 Why Now? 检验清单
- [ ] 行业是否处于拐点？
- [ ] 是否有新的监管政策？
- [ ] 技术变革是否改变竞争格局？
- [ ] 消费者行为是否发生迁移？
- [ ] 估值是否进入合理区间？

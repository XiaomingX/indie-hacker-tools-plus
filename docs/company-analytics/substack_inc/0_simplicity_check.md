# 0. 简单性与能力圈初筛 - Substack (Private)

## 业务可理解性

**核心逻辑**：Substack 是全球领先的创作者订阅平台，通过将传统的邮件通讯 (Newsletter) 与现代社交功能结合，实现了“创作者拥有自己的受众”。

- **产品逻辑**：写作工具 + 邮件分发系统 + 支付门槛 + 社交推荐网络 (Notes)。
- **盈利逻辑**：**10% 抽佣模式**。只有当创作者赚到钱时，Substack 才会赚钱。不依赖广告，不贩卖流量。
- **商业本质**：本质上是一个 **“创作者经济的基础设施服务商 (SaaS)”** 叠加了 **“社交发现网络 (Network)”**。

## 复杂性风险评估

1. **头部依赖风险 (Power Law)**：极小比例的头部作者贡献了绝大部分的 GMV。如果核心大 V 流失（如迁移至 Ghost 以节省 10% 佣金），对营收影响巨大。
2. **内部网络效应的边际递减**：虽然 Substack Notes 驱动了内部增长，但如何在大规模用户涌入后维持内容的“高质量”和“非噪音化”，是一个算法挑战。
3. **内容监管与品牌风险**：由于坚持极度的言论自由，Substack 经常面临关于极端主义或虚假信息的争议，这可能影响其与支付提供商（如 Stripe）的关系。

## 结论

**结论**：**属于“商业模式极其透明、与用户利益高度一致、且已建立强大网络效应”的优质科技资产**。

Substack 已经在 2025 年实现了现金流转正，证明了“订阅制社交”的生命力。它是理解“创作者主权”时代的必读标的。

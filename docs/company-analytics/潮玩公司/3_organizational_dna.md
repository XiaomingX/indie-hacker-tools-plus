# 3. 组织基因、团队与战略分歧

## 核心战略分歧：艺术偏执、流量收割与零售极权

### 1. 泡泡玛特：王宁的“中国版迪士尼”梦
- **组织基因**：**“尊重艺术、注重品位、坚持直营、国际化视野”**。王宁相信潮玩是纯粹的审美，不是单纯的玩具。
- **战略表现**：重金投入线下体验店和海外旗舰店，宁愿慢一点也要保住品牌的高级感。

### 2. 卡游：极致的“下沉收割机”
- **组织基因**：**“对流量的极致变现、对成瘾机制的精准计算”**。
- **战略表现**：通过海量 IP 授权横扫所有学区。

### 3. TOP TOY：叶国富的“全球潮玩工厂”
- **组织基因**：**“名创式的性价比算法、极致的渠道压榨、大规模分销”**。
- **战略表现**：利用资本力量迅速拓店。

## 战略分歧总结表

| 维度 | 泡泡玛特 | 卡游 | TOP TOY |
| :--- | :--- | :--- | :--- |
| **第一性原理** | IP 的生命力在于审美溢价 | 社交工具的生命力在于集齐的欲望 | 零售的生命力在于品类集成与便利 |
| **对待用户** | 养成感、社区感 | 诱导感、成瘾感 | 逛街感、多选感 |
| **团队底色** | 艺术家、高端买手 | 渠道老兵、心理学专家 | 供应链专家、零售操盘手 |

## 结论：谁的基因更具适应性？

- **泡泡玛特** 的基因最适合“品牌出海”和“跨周期增长”。
- **卡游** 的基因在“存量市场深度压榨”中无人能敌。

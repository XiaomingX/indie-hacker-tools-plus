## 3. 护城河评级标准

### 护城河类型量化评分
- **无形资产**: IP版权、品牌认知（0-25分）
- **转换成本**: 用户粘性、收藏体系（0-20分）
- **网络效应**: 社区、粉丝经济（0-25分）
- **规模优势**: 制造规模、渠道网络（0-15分）
- **定价权**: 限量溢价能力（0-15分）

### 综合评级
- **AAA (80+)**: 极强的持久竞争优势
- **AA (60-79)**: 较强竞争优势
- **A (40-59)**: 中等竞争优势
- **BBB (20-39)**: 较弱竞争优势

## 4. 泡泡玛特专题框架

### 核心问题
为何泡泡玛特市值最高？发展最迅速？

### 分析维度
1. **IP壁垒**：自有IP培育能力
2. **渠道网络**：零售+机器人商店
3. **品牌心智**：年轻人收藏文化
4. **设计师体系**：D2C能力
5. **全球化**：海外扩张

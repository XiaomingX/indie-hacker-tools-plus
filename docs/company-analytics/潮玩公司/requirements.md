# 潮玩公司竞品分析需求文档

## 1. 项目概述

### 分析目标
以巴菲特和芒格的价值投资视角，对全球主要潮玩公司进行深度分析，识别各公司的内在禀赋与长期投资价值，并重点解答：为何泡泡玛特发展最迅速，市值最高？

### 分析对象（10家公司）
| 公司 | 总部 | 成立时间 | 核心定位 |
|------|------|---------|---------|
| 泡泡玛特 | 北京 | 2010年 | 潮玩IP+零售 |
| 卡游游戏 | 杭州 | 2018年 | 集换式卡牌 |
| 52TOYS | 北京 | 2016年 | 潮玩IP+渠道 |
| TOP TOY | 广州 | 2020年 | 潮玩零售 |
| 若来(Rolife) | 深圳 | 2017年 | 潮玩IP |
| 万代(Bandai) | 日本 | 1950年 | 玩具制造 |
| Funko | 美国 | 1998年 | 流行文化IP |
| Kidrobot | 美国 | 2002年 | 设计师玩具 |
| Mighty Jaxx | 新加坡 | 2012年 | 艺术玩具 |
| Jellycat | 英国 | 1999年 | 毛绒玩具 |

## 2. 分析维度框架

### 2.1 发展历程与时间窗口
- 创立时机与时代红利
- 关键战略转折点
- 当前时间窗口的边际变化

### 2.2 商业模式与护城河
- 营收结构定量拆解
- IP运营能力
- 定价权来源
- 无形资产（品牌、IP版权）
- 转换成本
- 网络效应
- 规模优势

### 2.3 竞争格局与禀赋评估
- 行业竞争态势
- 各平台差异化定位
- 资源禀赋对比
- 管理层能力圈

### 2.4 治理结构
- 股权结构
- 激励机制

### 2.5 财务质量
- 自由现金流
- ROE杜邦拆解
- 资产质量

### 2.6 Pre-mortem风险评估
- 致命缺陷识别
- 颠覆性威胁
- 极端场景推演

## 4. 分析方法论

### 芒格栅格模型
- 诚实性检查
- 能力圈检查
- 冗余检查
- 失败成本检查

### 巴菲特护城河检查清单
- 是不是有定价权？
- 是否有无形资产保护？
- 转换成本是否足够高？
- 是否有网络效应？
- IP壁垒是否可持续？

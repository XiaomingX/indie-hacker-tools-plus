# 0. 商业模式与基本盘对决：蚂蚁国际 (WorldFirst) vs 空中云汇 (Airwallex) vs 连连数字

## 核心赚钱逻辑：生态流量税 vs 汇率差额税 vs 合规场景税

跨境支付行业的商业模式本质是**“对资金跨境流转效率的定价”**。然而，由于技术底层和切入场景的不同，三巨头的变现逻辑有着本质区别：

1. **蚂蚁国际 (Ant International / WorldFirst)**：**极致的“全球支付生态连接者”**。
   - **底层逻辑**：蚂蚁走的是**“网络效应”**路线。通过 Alipay+ 连接全球钱包，通过万里汇（WorldFirst）连接 B2B 卖家。它赚的是**“生态闭环内的手续费”**。其基本盘在于其**全球化的持牌体系**和**与支付宝生态的无缝协同**。它不只是支付公司，更是数字基础设施。
2. **空中云汇 (Airwallex)**：**技术驱动的“全球资金管理大师”**。
   - **底层逻辑**：空中云汇走的是**“降维打击”**路线。它自研了一套绕过 SWIFT 的全球清算网络。它赚的是**“极窄的汇率价差 (FX Spread)”**和**“嵌入式金融服务费”**。其基本盘在于其**卓越的技术产品力**和**对跨国企业（尤其是 SaaS 和平台型公司）复杂资金流转需求的极速响应**。
3. **连连数字 (LianLian Global)**：**深耕垂直场景的“合规专家”**。
   - **底层逻辑**：连连走的是**“场景深挖”**路线。它深度绑定跨境电商卖家，提供从收款、换汇到退税、供应链金融的全链路服务。它赚的是**“综合服务费”**。其基本盘在于其**数量最多的全球支付牌照**和**对中国跨境卖家业务痛点的深刻理解**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | 蚂蚁国际 | 空中云汇 | 连连数字 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 交易手续费 (高频) + 商家金融服务 | 汇率溢价 (核心) + 企业支出管理 + 嵌入式金融 | 跨境收单/收款 + 增值服务 (退税、金融) |
| **单笔交易毛利** | 中 (靠规模摊薄成本) | **高** (通过技术手段消灭了大量中间行成本) | 中 (竞争激烈，费率承压) |
| **变现核心** | **规模** (Transaction Volume) | **效率** (Velocity of Money) | **场景深度** (Service Depth) |
| **核心成本** | 银行通道费、合规风控成本 | 技术研发成本、全球流动性管理成本 | 获客成本、合规合规成本 |

## 结论：基本盘的稳固度

- **蚂蚁国际** 的基本盘最“沉”，依托蚂蚁集团的背书和全球支付网络的布局，其具备极强的行政与生态壁垒。
- **空中云汇** 的基本盘最“活”，其作为金融科技新贵，通过技术创新实现了全行业最低的转账成本，对传统银行和老牌支付公司具有颠覆性。
- **连连数字** 的基本盘最“实”，它深扎在跨境电商的毛细血管里，虽然利润受费率战影响，但其提供的增值服务极大地提升了客户粘性。

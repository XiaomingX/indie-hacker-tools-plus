# 跨境支付与汇款平台竞品分析需求文档

## 1. 项目概述

### 分析目标
以巴菲特和芒格的价值投资视角，对全球主流跨境支付与汇款平台进行深度分析，识别各平台的内在禀赋与长期投资价值。

### 分析对象（11家）
| 平台 | 总部 | 成立时间 | 核心定位 |
|------|------|---------|---------|
| XTransfer | 上海 | 2017年 | B2B跨境收款 |
| Stripe | 美国旧金山 | 2010年 | 支付基础设施 |
| Airwallex | 澳大利亚墨尔本 | 2015年 | 全球支付平台 |
| Wise | 英国伦敦 | 2011年 | 个人汇款+多币种 |
| Revolut | 英国伦敦 | 2015年 | 数字银行+汇款 |
| Remitly | 美国西雅图 | 2011年 | 汇款服务 |
| Xoom | 美国加州 | 2001年 | 跨境汇款 |
| Payoneer | 美国纽约 | 2005年 | B2B支付服务 |
| WorldRemit | 英国伦敦 | 2010年 | 国际汇款 |
| Paysend | 英国伦敦 | 2017年 | 端到端汇款 |
| OFX | 澳大利亚悉尼 | 1998年 | 机构外汇服务 |

## 2. 分析维度框架

### 2.1 发展历程与时间窗口 (When & Why Now?)
- 创建时间线与关键里程碑
- 融资历史与资本背书
- 移动互联网/金融科技红利期把握
- 监管政策边际变化

### 2.2 商业模式与护城河 (Moat)
- 营收结构定量拆解（手续费/汇率/增值）
- 定价权来源
- 无形资产（牌照、品牌）
- 转换成本（用户粘性）
- 网络效应（全球网络）
- 技术壁垒

### 2.3 竞争格局与禀赋评估 (Competitive Advantage)
- 行业集中度与竞争态势
- 各平台差异化定位
- 资源禀赋对比
- 管理层能力圈

### 2.4 治理结构与利益一致性
- 股权结构与资本背景
- 激励机制
- 合规与风控

### 2.5 财务质量与资本配置
- 自由现金流
- 增长率与规模
- 盈利能力

### 2.6 逆向风险评估 (Pre-mortem)
- 监管政策风险
- 竞争替代威胁
- 汇率波动风险
- 极端场景推演

## 4. 分析方法论

### 芒格栅格模型应用
- 诚实性检查：平台合规性与反洗钱
- 能力圈检查：管理层金融科技背景
- 冗余检查：是否存在单一市场依赖
- 失败成本检查：用户/商户转换成本

### 巴菲特护城河检查清单
- 是不是有定价权？
- 是否有无形资产（牌照/品牌）保护？
- 转换成本是否足够高？
- 是否有网络效应？
- 技术壁垒是否可持续？

## 5. 约束条件

- 时间范围：1998-2025年发展周期
- 地域范围：全球市场，侧重中国跨境场景
- 信息截止：基于公开信息分析，不涉及内幕
- 视角定位：投资研究视角，非运营诊断

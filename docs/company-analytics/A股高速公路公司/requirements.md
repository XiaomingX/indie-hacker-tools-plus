# A股高速公路公司竞品分析需求文档

## 1. 项目概述

### 分析目标

## 2. 分析维度框架

### 2.1 发展历程与时间窗口
- 路段资源禀赋与区位优势
- 路网规模与增长潜力
- 政策特许经营期限
- 当前时间窗口的边际变化

### 2.2 商业模式与护城河 (Moat)
- 营收结构定量拆解（通行费+其他）
- 定价权来源（政府定价vs市场化）
- 特许经营权壁垒
- 转换成本（用户粘性）
- 自然禀赋壁垒（优质路段）

### 2.3 竞争格局与禀赋评估
- 区域垄断属性
- 路段质量对比
- 资源禀赋对比
- 管理层能力圈

### 2.4 治理结构与利益一致性
- 股权结构（国资为主）
- 激励机制
- 分红政策

### 2.5 财务质量与资本配置
- 自由现金流
- ROE杜邦拆解
- 分红率与股息率

### 2.6 逆向风险评估 (Pre-mortem)
- 致命缺陷识别
- 新能源替代威胁
- 极端场景推演

## 4. 分析方法论

### 芒格栅格模型应用
- 诚实性检查：车流量数据真实性
- 能力圈检查：管理层高速公路运营能力
- 冗余检查：是否存在单一依赖
- 失败成本检查：用户转换成本

### 巴菲特护城河检查清单
- 是不是有自然禀赋壁垒（优质路段）？
- 是否有特许经营权保护？
- 转换成本是否足够高？
- 是否有定价权（政府vs市场化）？
- 分红政策是否稳定？

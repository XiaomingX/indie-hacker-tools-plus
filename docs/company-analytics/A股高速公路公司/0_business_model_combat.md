# 0. 商业模式与基本盘对决：招商公路 vs 宁沪高速 vs 粤高速

## 核心赚钱逻辑：资产整合者 vs 区域霸主 vs 分红提款机

高速公路行业的商业模式本质是**“特许经营权下的车流量变现”**。然而，由于经营策略与资产分布的不同，这三家龙头的基本盘呈现出显著的差异：

1. **招商公路 (CMET)**：**全国性的“公路基金”**。
   - **底层逻辑**：招商公路是 A 股唯一的全路网布局平台。它不仅自己管路，还通过参股几十家高速公路公司（如宁沪、粤高速等）获取投资收益。它赚的是“资产管理”和“全产业链（建设+智慧交通）”的钱。其基本盘在于其强大的**资源整合能力**和**资本运作效率**。
2. **宁沪高速 (Jiangsu Expressway)**：**长三角的“流量之王”**。
   - **底层逻辑**：宁沪高速的核心资产是苏沪之间的主干道，这是中国最繁忙、经济含金量最高的线路。它赚的是“极致稳健”的过路费。其基本盘在于苏沪经济圈不可动摇的**通勤与物流刚需**。它是行业内“类债属性”最强的标的。
3. **粤高速 (Guangdong Expressway)**：**大湾区的“高息收割机”**。
   - **底层逻辑**：粤高速深耕粤港澳大湾区核心路网。它追求的不是扩张，而是**极致的股东回报**。在路产剩余年限内，通过极高的分红率将利润返还给股东。其基本盘在于大湾区**高密度的私家车保有量**和**极高的通行费单价**。

## 营收与成本结构对比 (2025 预估)

| 指标 | 招商公路 | 宁沪高速 | 粤高速 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 通行费 (50%) + 投资收益 (40%) + 交通科技 | 通行费 (绝对核心) + 服务区配套 | 通行费 (绝对核心) |
| **毛利属性** | 中高 (投资收益摊薄了毛利表现) | 极高 (主干线折旧已低，流量大) | 极高 (由于资产成熟，边际成本极低) |
| **成本结构** | 管理费用、利息支出、科技研发 | 摊销、养护成本、财务费用 | 摊销、养护费用 (较纯粹) |
| **扩张潜力** | **强** (持续并购全国路产) | 中 (受限于长三角土地，主要靠改扩建) | 低 (核心路产趋于饱和，分红为主) |

## 结论：基本盘的稳固度

- **宁沪高速** 的基本盘最稳，滬寧线的车流几乎是物理上的“单行道”，替代性极低。
- **招商公路** 的基本盘最广，通过全国分散化投资对冲了单一区域的经济波动风险。
- **粤高速** 的基本盘最具“时效性”，由于路产有到期日，它是在与时间赛跑，将资产价值通过分红榨干。

## 关键发现

### 最大的"好"
- 现金流质量优秀：OCF/扣非净利>2x
- 分红稳定：分红率~50%，股息率~3%
- 国资背景：上海市国资委实控

### 最大的"坏"
- 毛利率连续4年下行：11.93%→7.83%
- 扣非利润零增长：4年增速仅0-4%
- 机构大规模撤离：基金持仓-71%

### 最大的"不确定"
- 央国企改革进度
- 资产注入预期
- 毛利率何时企稳

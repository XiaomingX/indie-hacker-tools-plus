# 外服控股（600662.SH）- 逆向思考与尾部风险

## 核心发现

1. **Pre-mortem**：假设投资亏损50%+，最可能原因——毛利率崩塌+竞争加剧+央国企改革落空
2. **做空逻辑**：高营收低利润结构脆弱，毛利率持续下行是核心风险
3. **不可证伪假设**："行业万亿市场所以公司有空间"——TAM谬误
4. **尾部风险**：大客户流失、合规风险、关联交易利益输送

---

## 1. Pre-mortem分析：假设投资亏损50%+

### 1.1 亏损情景推演

**情景一：毛利率崩塌（概率：中高）**

触发条件：
- 业务外包竞争进一步加剧，价格战升级
- 毛利率从8.56%降至5%以下
- 扣非净利润从4.76亿降至2亿以下

传导路径：
```
价格战加剧 → 外包毛利率从3%降至1-2%
→ 整体毛利率从8.5%降至5-6%
→ 扣非净利润腰斩至2-3亿
→ PE被动抬升至30x+（即使股价不变）
→ 市场给予更低估值（PE降至15x）
→ 股价从5元跌至2-3元（-40~50%）
```

**情景二：央国企改革落空（概率：中）**

触发条件：
- 市值管理政策推进缓慢
- 分红率未提升，回购未落地
- 资产注入预期落空

传导路径：
```
政策催化落空 → 市场预期下修
→ 国资改革溢价消失
→ 估值从PB 2x回落至1.2-1.5x
→ 股价从5元跌至3-4元（-20~40%）
```

**情景三：大客户流失（概率：低中）**

触发条件：
- 核心大客户（外企、国企）转向竞争对手
- 科锐国际等灵活用工龙头抢占份额
- 人事管理（高毛利）业务大幅萎缩

传导路径：
```
大客户流失 → 高毛利业务占比进一步下降
→ 利润结构恶化
→ 市场重新定价
→ 股价下跌30-40%
```

**情景四：黑天鹅事件（概率：低）**

触发条件：
- 合规问题（假外包真派遣被查处）
- 关联交易丑闻
- 管理层重大变动
- 宏观经济大幅下行

### 1.2 亏损概率评估

| 情景 | 概率 | 亏损幅度 | 期望亏损 |
|------|------|---------|---------|
| 毛利率崩塌 | 30% | -40~50% | -13.5% |
| 改革落空 | 25% | -20~40% | -7.5% |
| 大客户流失 | 15% | -30~40% | -5.3% |
| 黑天鹅 | 5% | -50%+ | -2.5% |

**综合期望亏损：约-28.8%**（概率加权）

## 2. 做空逻辑分析

### 2.1 做空论点

**核心论点：高营收低利润结构不可持续**

| 论点 | 支撑证据 | 强度 |
|------|---------|------|
| 毛利率持续下行 | 11.93%→8.56%→7.83%（4年连降） | 强 |
| 扣非利润近乎零增长 | 2022-2025年增速：-1%→0%→0.4%→4% | 强 |
| 营收含大量代收代付 | 241亿营收，实际服务费~40-50亿 | 中 |
| 竞争格局恶化 | CR5<8%，价格战持续 | 中 |
| 央国企机制僵化 | 管理层激励不足，市场化程度低 | 中 |

### 2.2 做空风险

| 风险 | 说明 |
|------|------|
| 政策催化 | 央国企市值管理可能超预期 |
| 资产注入 | 东浩兰生集团可能注入资产 |
| 行业整合 | 外服控股可能成为整合者 |
| 分红支撑 | 3%股息率提供估值底部 |

**做空逻辑评分：6/10（逻辑成立但有政策风险）**

## 3. 不可证伪假设识别

### 3.1 常见不可证伪假设

| 假设 | 为什么不可证伪 | 风险 |
|------|--------------|------|
| "万亿市场有空间" | TAM大≠公司能拿到份额 | TAM谬误 |
| "国资背景是护城河" | 但利润率持续下行 | 护城河被侵蚀 |
| "灵活用工渗透率会提升" | 提升不一定利好外服 | 受益者可能是竞争对手 |
| "央国企改革会改善治理" | 改革进度和效果不确定 | 预期落空风险 |
| "分红率会持续提升" | 取决于大股东意愿和政策 | 不确定性高 |

### 3.2 最危险的不可证伪假设

**"人力资源万亿市场，外服控股作为龙头一定受益"**

反驳：
- 行业CR5<8%，极度分散，所谓"龙头"市占率仅1.6%
- 营收241亿中大量代收代付，真实市占率<0.5%
- 市场大不等于公司能拿到份额，科锐国际增速远快于外服
- 万亿市场的受益者可能是新进入者或技术驱动型公司

**不可证伪假设评分：6/10（TAM谬误风险高）**

外服控股的风险状况：
1. **最大风险**：毛利率持续崩塌（11.93%→8.56%→7.83%），趋势未止

3. **最危险假设**："万亿市场有空间"——TAM谬误
4. **下行空间**：综合期望亏损~29%，下行风险大于上行弹性

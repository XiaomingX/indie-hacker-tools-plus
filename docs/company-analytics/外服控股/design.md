# 外服控股（600662.SH）- 分析设计

## 分析方法

**Top-Down + Bottom-Up 结合**：
- Top-Down：宏观经济→人力资源行业→公司定位
- Bottom-Up：财务质量→商业模式→估值安全边际

## 维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 行业分析 | 10% | 人力资源行业周期性强，受宏观就业影响 |
| 商业模式 | 25% | 护城河强度决定长期价值 |
| 竞争格局 | 15% | 行业集中度和竞争态势 |
| 财务质量 | 20% | FCF/ROE是价值投资核心 |
| 治理结构 | 10% | 央国企治理质量 |
| 估值 | 15% | 安全边际是买入前提 |
| 催化剂 | 5% | 边际变化驱动股价 |

## 公司画像

| 维度 | 描述 |
|------|------|
| 公司性质 | 上海地方国有企业（东浩兰生集团） |
| 上市时间 | 1993年（原名"新亚股份"，多次更名重组） |
| 核心业务 | 人力资源全链条服务 |

| 员工规模 | 管理外包员工数十万人 |
| 营收规模 | 预计百亿级（含代收代付部分） |

## 关键假设

1. 人力资源行业与宏观经济高度相关（β属性）
2. 灵活用工是行业增长核心驱动力
3. 央国企改革可能带来治理改善和资产注入
4. FESCO Adecco合资为公司提供国际化竞争力

## 风险预判

1. 经济下行导致企业缩减用工需求
2. 劳务派遣政策收紧（同工同酬、比例限制）
3. 行业竞争加剧导致费率下降
4. 央国企治理效率问题
5. 合资方关系变化

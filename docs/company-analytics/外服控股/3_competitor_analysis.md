# 外服控股（600662.SH）- 竞争格局与对手对比

## 核心发现

1. **行业格局**：极度分散（CR5<8%），"大行业小公司"，整合空间大
2. **主要对手**：中智集团（央企）、科锐国际（灵活用工龙头）、人瑞人才（港股）
3. **外服控股位置**：A股人力资源龙头，但规模优势不突出
4. **竞争边际**：价格战持续，毛利率承压，竞争未见边际改善

### 1.3 竞争强度评估

- 行业进入门槛低，中小机构数量庞大
- 服务同质化严重，价格战成为主要手段
- 毛利率持续下行（行业普遍现象）

### 2.2 科锐国际详细对比

|------|---------|---------|------|
| 营收规模 | 241亿 | ~7.77亿 | 外服规模大30倍 |

- 科锐国际更聚焦高附加值业务，外服控股更依赖低毛利外包

## 4. 竞争格局演变

### 4.1 行业整合趋势

**整合逻辑**：
- 合规成本上升，中小机构出清
- 大客户倾向选择头部服务商
- 数字化投入门槛提高
- 资本市场提供并购工具

**整合进度**：
- 目前仍处于早期阶段
- CR5<8%，整合空间巨大
- 头部企业增速快于行业平均

### 4.2 竞争是否边际改善？

**判断**：❌ 竞争未边际改善，仍在加剧

- 毛利率持续下行（11.93%→8.56%）
- 价格战未见缓解
- 新进入者不断涌现
- 客户压价普遍

**竞争边际评分：3/10（竞争仍在加剧）**

### 6.2 核心判断

外服控股面临的竞争格局：
1. **行业极度分散**：CR5<8%，整合空间大但进度慢
2. **竞争对手各有优势**：科锐增速快、中智规模大、人瑞灵活用工强
3. **外服控股优势不突出**：规模大但利润率低，增速慢
4. **竞争未见改善**：毛利率持续下行，价格战持续

**核心问题**：外服控股能否在激烈竞争中提升利润率和增速？

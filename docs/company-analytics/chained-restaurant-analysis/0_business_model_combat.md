# 0. 商业模式与基本盘对决：百胜中国 (肯德基/必胜客) vs 麦德龙 (麦当劳中国) vs 萨莉亚

## 核心赚钱逻辑：工业化效率 vs 房产逻辑 vs 极致成本管理

快餐行业的对决本质上是**“如何将廉价热量规模化分发”**。

1. **百胜中国 (Yum China)**：**极致的“中国本土化效率机器”**。
   - **底层逻辑**：百胜是中国餐饮业最大的数字化实验室。它通过极致的本土化菜单（早餐油条、老北京鸡肉卷）和强大的数字化会员体系，赚的是**“国民级心智红利”**。其基本盘是**遍布全国各级城市的数万家门店**。
2. **麦当劳中国 (McDonald's)**：**“供应链极权”下的标准化**。
   - **底层逻辑**：麦当劳在全球是房产公司，在中国则是**“供应链集成商”**。它不追求花哨的品种，追求的是极致的标准化。它赚的是**“信任溢价”**。其基本盘是**一二线城市的核心地段覆盖**。
3. **萨莉亚 (Saizeriya)**：**餐饮业的“拼多多”**。
   - **底层逻辑**：萨莉亚赚的是**“剔除一切非核心成本后的效率利润”**。它不买昂贵的洗碗机，不招专业的厨师。它赚的是**“极致性价比下的超高翻台率”**。其基本盘是**对价格极度敏感的年轻群体和白领**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | 百胜中国 | 麦当劳中国 | 萨莉亚 |
| :--- | :--- | :--- | :--- |
| **主要利润源** | 肯德基 (核心) | 汉堡/小食 | 意面/披萨 |
| **单店模型** | 投入大、回报稳 | 标准化极高 | 极简、低投入 |
| **原材料成本占比** | 中 (集中采购) | 中 (全球供应链) | **极高** (为了维持低价，几乎无溢价) |
| **人机效能** | 极高 (数字化程度最高) | 高 | 极高 (由于流程极简) |

## 结论：基本盘的稳固度

- **百胜中国** 的基本盘最广，但管理半径已接近极限；
- **麦当劳中国** 的基本盘最稳，作为“标准化”的代名词，具有抗御危机的天然能力；
- **萨莉亚** 的基本盘最奇特，它是“消费降级”背景下的终极受益者。

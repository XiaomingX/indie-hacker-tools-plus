# 4. 极端压力测试与死法推演 (Pre-mortem)

## 假如明天是末日：快餐巨头的生存排序

设定三个极端毁灭性场景：

### 场景一：原材料（鸡肉/面粉）成本暴涨 100%
- **背景**：全球供应链断裂，基础食材价格翻倍。
- **萨莉亚**：**第一个倒下/崩盘**。由于其毛利极低且主打低价，它没有任何缓冲空间，一旦涨价，其核心竞争力（性价比）将荡然无存。
- **麦当劳/百胜**：**中伤/提价转嫁**。凭借强大的议价权和品牌溢价，它们可以通过“涨个几块钱”来消化成本，用户依然会买单。

### 场景二：新锐国产快餐（如塔斯汀、塔斯汀等）彻底实现万店规模
- **背景**：国产快餐在口味和价格上全方位碾压。
- **百胜中国**：**受重伤**。它主要在三四线城市与这些对手肉搏，下沉市场的基本盘将受到剧烈侵蚀。
- **麦当劳**：**稳健**。其核心受众与国产快餐存在差异，品牌护城河较深。

### 场景三：外卖配送费涨至 20 元/单
- **背景**：劳动力短缺，外卖平台提高抽佣。
- **萨莉亚**：**稳健**。它基本不做外卖，或者外卖占比极低，主要靠堂食的高翻台率生存。
- **百胜/麦当劳**：**中伤**。外卖是它们的增长极，一旦配送费过高，单次用餐成本将不具备竞争力。

## 死亡排序预测

**第一个倒下：萨莉亚 (受制于极窄的利润安全垫)**
- **理由**：它是“走钢丝”的高手，一旦风力（原材料成本）稍微加大，就会摔落。

**第二个倒下：百胜中国 (受制于过大的管理半径和内卷的下沉市场)**
- **理由**：大象转身难，尤其是在面对无数“小象”（国产新快餐）围攻时。

**最后赢家：麦当劳中国**
- **理由**：**“品牌即信仰”**。在全球快餐史上，麦当劳已经证明了自己是抵御任何周期风险的最强机器。

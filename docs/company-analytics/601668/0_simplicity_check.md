# 0. 简单性与能力圈初筛 - 中国建筑 (601668.SH)

## 业务可理解性

**核心逻辑**：中国建筑是全球最大的建筑地产综合企业。它不仅“盖房子”（房建），还“修路桥”（基建），并“搞开发”（中海地产）。

- **产品逻辑**：全产业链建筑工程服务 + 顶级住宅地产开发。
- **盈利逻辑**：**工程利润 (薄利多销) + 地产利润 (高杠杆高周转) + 股息回报**。
- **商业本质**：本质上是**“全球基础设施建设与城市运营的超级航母”**。

## 复杂性风险评估

1. **庞大的账面资产与回款风险**：作为万亿级营收的企业，应收账款和存货余额极其巨大。地方政府和地产商的支付能力是影响其估值的核心复杂变量。
2. **地产业务的拖累与协同**：旗下中海地产虽为蓝筹，但在 2024-2026 年行业下行期，地产业务的去化压力和减值风险对母公司利润有较大扰动。
3. **“基建补缺”的有效性**：当房建业务增速放缓时，基建（尤其是能源、水务）业务能否提供足够的增量利润。

## 结论

**结论**：**属于“业务逻辑极简（盖房子修路），但财务报表极重且受宏观政策强力牵引”堆**。

中国建筑是 A 股中极少数营收超 2 万亿、PE 却不到 5 倍的巨无霸。它是典型的“低估值、高股息、强护城河”标的。

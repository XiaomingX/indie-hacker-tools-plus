## 3. 央企改革与激励机制

### 3.1 市值管理

| 举措 | 进展 |
|------|------|
| 《价值管理制度》 | 已发布 |
| 价值提升计划 | 已制定并推进 |
| 市值管理纳入考核 | 国资委要求下已响应 |
| 分红率提升 | 2025年提升至~29% |

### 3.2 股权激励

| 项目 | 状态 |
|------|------|
| 股权激励计划 | 暂无公开披露的大规模股权激励 |
| 管理层持股 | 极少，利益绑定有限 |
| 薪酬体系 | 央企标准，与业绩挂钩但弹性有限 |

**评价**：相比市场化程度更高的央企（如招商系），中国建筑的激励机制偏保守，管理层利益与股价的绑定程度有限。

## 5. 核心结论

> **治理质量定性**：典型的**央企治理模式**——股权高度集中、管理层由国资委任命、激励机制偏保守、合规性良好。

| 维度 | 评级 | 说明 |
|------|------|------|
| 股权结构 | ⭐⭐⭐ | 高度集中，制衡有限 |
| 管理层能力 | ⭐⭐⭐⭐ | 战略执行基本到位 |
| 激励机制 | ⭐⭐ | 偏保守，利益绑定不足 |
| 市值管理意愿 | ⭐⭐⭐⭐ | 响应国资委要求，分红率提升 |
| 合规性 | ⭐⭐⭐⭐⭐ | 央企标准，无重大违规 |

**投资含义**：治理结构不会成为投资的核心风险，但也不会成为超额收益的来源。关注点在于**分红率能否持续提升**以及**市值管理考核的实际效果**。

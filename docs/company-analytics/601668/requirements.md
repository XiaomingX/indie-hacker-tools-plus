# 中国建筑（601668.SH）深度研究需求文档

## 1. 研究目标

基于价值投资框架（巴菲特/芒格），对中国建筑进行全面深度剖析，产出专业级别投研报告，最终给出明确的投资评级（买入/持有/卖出）及买入时机判断。

## 3. 核心研究维度（11项）

### 3.1 宏观周期、政策导向与行业分析
- 基建投资周期判断（逆周期调节力度、专项债发行节奏）
- 房地产政策环境（三道红线、保交楼、城中村改造、收储政策）
- 新型城镇化与重大国家战略（一带一路、京津冀、长三角、粤港澳）
- 大宗商品（钢铁、水泥、沥青）价格周期对成本端的影响
- 央国企市值管理与中特估体系

### 3.2 商业模式与护城河深潜
- 营收结构拆解（房建、基建、地产开发、设计勘察、其他）
- 中国建筑独特模式：建筑+地产双轮驱动
- 护城河评估：特级资质壁垒、央企信用背书、规模成本优势、全产业链协同

### 3.3 竞争格局与对手对比
- 建筑央企横向对比（中国中铁、中国铁建、中国交建、中国电建、中国能建）
- 地产板块同业对比（保利、中海、华润等央企地产）
- 市占率变化趋势，订单获取能力

### 3.4 财务质量与资本配置
- ROE杜邦拆解（净利率、资产周转率、权益乘数）
- 经营性现金流 vs 净利润（应收款、合同资产隐雷）
- 订单储备与收入保障倍数
- 分红率与股息率

### 3.5 治理结构与管理层画像
- 股权结构（中建集团控股、国资委实控）
- 管理层稳定性与战略执行力
- 央企改革与激励机制（股权激励、分红承诺）

### 3.6 估值深潜与安全边际
- PE/PB历史分位数
- 建筑央企估值对比
- DCF/股息贴现模型测算内在价值

### 3.7 催化剂与拐点分析
- 基建投资增速拐点
- 地产板块止血/盈利修复
- 央国企价值重估（中特估）
- 订单增速与新签合同额变化

### 3.8 逆向思考与尾部风险
- 地产板块大幅减值风险
- 地方政府财政压力导致的应收款坏账
- 基建投资回报率持续走低
- 海外业务地缘政治风险

### 3.9 资金面与情绪面
- 北向资金持仓变化
- 机构持仓拥挤度
- 融资融券数据

### 3.10 终极检查清单
- 芒格栅格模型多维复核
- "Too Hard"堆判断

### 3.11 最终投资决策报告
- 综合评级与建议仓位
- 核心逻辑总结
- "为什么是现在？"

## 4. 输出要求

- 所有报告存入 `reports/601668/` 目录

- 优先使用最新财报数据（2025年报/2026Q1）

- Wind/iFinD/Choice等金融终端数据优先

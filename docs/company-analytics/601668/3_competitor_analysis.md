## 2. 中国建筑的差异化定位

## 3. 业务赛道对比

### 3.1 各央企核心赛道

```
中国建筑：房建龙头 + 地产开发 → 城中村改造是最大增量
中国中铁：铁路基建双寡头 + 矿产资源 → 利润弹性来自矿产
中国铁建：铁路基建双寡头 + 城轨 → 城轨市场份额领先
中国交建：港口/桥梁/疏浚 → 交通基建全产业链
中国电建：水利水电 + 新能源 → 水利投资+能源转型双驱动
中国能建：能源电力 + 氢能 → 新能源转型最激进
```

## 5. 核心结论

> **竞争定位**：中国建筑是建筑央企中的**绝对规模王者**，营收和利润均排名第一，ROE领先，股息率最高。但毛利率/净利率偏低，地产板块是双刃剑。

| 维度 | 中国建筑排名 | 说明 |
|------|-------------|------|
| 营收规模 | **第1** | 2.08万亿，绝对龙头 |
| 归母净利润 | **第1** | 390.7亿 |
| 新签合同额 | **第1** | 4.55万亿 |
| ROE | **第1** | 8.15% |
| 股息率 | **第1** | ~6.5% |
| PE估值 | **最低** | 4.5x，安全边际最足 |
| 毛利率 | 第4 | 9.90%，低于交建/电建/能建 |
| 净利率 | 第5 | 1.88%，仅高于电建 |

**投资含义**：在建筑央企板块中，中国建筑以**最低估值（PE 4.5x）+ 最高股息率（6.5%）+ 最大规模**构成核心吸引力。相比电建/能建有新能源转型溢价，中国建筑的价值更多体现在**高股息+低估值+城中村改造催化**。

## T01: 宏观周期、政策导向与行业分析

**目标**：定位当前建筑行业所处的宏观环境与政策周期

**执行要点**：
1. 分析当前基建投资增速与逆周期调节力度
2. 评估房地产政策环境（保交楼、城中村改造、收储、三道红线放松）
3. 判断信用周期阶段（专项债发行节奏、社融增速）
4. 大宗商品（钢铁、水泥、沥青）价格周期对建筑成本端的影响
5. 新型城镇化与区域战略（一带一路、京津冀、长三角、粤港澳）
6. 央国企估值体系（中特估）与市值管理政策

**关键数据需求**：
- 2025-2026年基建投资增速
- 房地产开发投资、销售面积、新开工面积数据
- 专项债发行规模与投向
- 钢材/水泥价格指数
- 建筑板块PE/PB历史分位数

## T02: 商业模式与护城河深潜

**目标**：拆解中国建筑"建筑+地产"双轮驱动模式，评估核心竞争壁垒

**执行要点**：
1. 营收结构定量拆解
   - 房屋建筑工程（占比、增速、毛利率）
   - 基础设施建设（占比、增速、毛利率）
   - 房地产开发与投资（中海地产+中建地产）
   - 勘察设计与其他
2. 建筑业务护城河五维评估：
   - 资质壁垒：特级资质稀缺性、总承包能力
   - 央企信用：融资成本优势、政府项目获取能力
   - 规模效应：全球最大建筑企业的采购与管理成本优势
   - 全产业链：设计-施工-运营一体化协同
   - 品牌价值：地标工程背书
3. 地产板块定位
   - 中海地产的品牌与盈利能力
   - 地产板块对集团现金流的贡献与拖累

**关键数据需求**：
- 近3年各业务板块营收/利润占比变化
- 新签合同额结构（房建/基建/地产/海外）
- 各板块毛利率趋势
- 中海地产销售数据与利润贡献

## T03: 竞争格局与对手对比

**目标**：在建筑央企中定位中国建筑的竞争力

**执行要点**：
1. 建筑央企横向对比矩阵（中国建筑、中国中铁、中国铁建、中国交建、中国电建、中国能建）
2. 关键指标对比：营收规模、净利润、新签合同额、毛利率、ROE
3. 差异化优势分析（房建龙头 vs 基建专业户 vs 多元化）
4. 地产板块横向对比（中海 vs 保利、华润、招商蛇口）
5. 市占率变化趋势

**对比维度**：
| 指标 | 说明 |
|------|------|
| 规模指标 | 营业收入、总资产、新签合同额 |
| 盈利指标 | 净利率、ROE、ROA、各业务毛利率 |
| 质量指标 | 收现比、经营现金流/净利润、应收周转 |
| 成长指标 | 营收增速、订单增速、净利润增速 |
| 估值指标 | PE、PB、股息率 |

## T04: 财务质量与资本配置

**目标**：深度审计财务数据质量，评估资本配置效率

**执行要点**：
1. ROE杜邦三因素拆解
   - 净利润率 = 归母净利润 / 营业收入
   - 资产周转率 = 营业收入 / 总资产
   - 权益乘数 = 总资产 / 归母净资产
2. 现金流质量深度分析
   - 收现比趋势（销售收到现金/营业收入）
   - 经营现金流 vs 净利润比值
   - 应收账款账龄结构与坏账计提
   - 合同资产/合同负债变动分析
3. 资产质量排查
   - 存货（地产开发成本）减值风险
   - 应收账款/营业收入占比趋势
   - 有息负债率与融资成本
   - 商誉减值测试
4. 分红政策与股东回报
   - 近5年分红率趋势
   - 股息率横向对比
   - 自由现金流覆盖率

## T06: 估值深潜与安全边际

**目标**：多维度估值测算，确定安全边际

**执行要点**：
1. PE估值法
   - 当前PE vs 建筑央企板块均值
   - 历史5年/10年PE分位数
2. PB估值法
   - 当前PB vs 历史分位数
   - 合理PB区间测算（基于可持续ROE）
3. 股息贴现模型（DDM）
   - 假设：折现率、分红增长率、永续增长率
   - 内在价值测算
4. 可比公司估值对比
   - 建筑央企PE/PB横向对比
   - 估值溢价/折价原因分析
5. 安全边际计算
   - 当前价格 vs 内在价值
   - 安全边际百分比

## T08: 逆向思考与尾部风险

**目标**：事前验尸，构建极端风险场景

**执行要点**：
1. 地产板块大幅减值
   - 中海地产+中建地产存货减值压力测试
   - 房价下跌30%情景下的资产减值测算
2. 地方政府应收款风险
   - 基建应收款中地方政府占比
   - 地方财政压力导致的坏账风险
3. 基建投资回报率恶化
   - 项目收益率持续走低
   - PPP项目退出风险
4. 海外业务地缘风险
   - 一带一路沿线国家政治风险
   - 汇率波动对海外利润的侵蚀
5. 行业系统性风险
   - 建筑行业信用风险传染
   - 上游原材料价格暴涨

## T01: 宏观周期、政策导向与行业分析

**目标**：定位当前银行业所处的宏观环境与政策周期

**执行要点**：
1. 分析当前货币政策取向（LPR走势、MLF操作、降准预期）
2. 评估银行业监管政策（资本新规、TLAC要求、金融让利实体）
3. 判断信用周期阶段（社融增速、信贷结构、票据占比）
4. 分析利率环境对银行NIM的影响
5. 银行板块估值分位与资金配置情况

**关键数据需求**：
- 2025-2026年LPR调整记录
- 最新社融/信贷数据
- 银保监会最新监管指标
- 银行板块PB历史分位数

## T02: 商业模式与护城河深潜

**目标**：拆解兴业银行营收结构，评估核心竞争壁垒

**执行要点**：
1. 营收结构定量拆解（利息净收入占比、中收占比、其他非息占比）
2. 兴业银行三大名片分析：绿色银行、财富管理、投行+FICC
3. 护城河五维评估：
   - 牌照壁垒：银行牌照稀缺性
   - 客户粘性：企业客户转换成本
   - 品牌价值：市场认知与口碑
   - 规模效应：资产规模带来的成本优势
   - 网络效应：银银平台等生态

**关键数据需求**：
- 近3年营收结构变化
- 绿色金融贷款余额及增速
- 财富管理AUM及中收贡献
- 投行业务规模与排名

## T04: 财务质量与资本配置

**目标**：深度审计财务数据质量，评估资本配置效率

**执行要点**：
1. ROE杜邦三因素拆解
   - 净利润率 = 归母净利润 / 营业收入
   - 资产周转率 = 营业收入 / 总资产
   - 权益乘数 = 总资产 / 归母净资产
2. 资产质量深度分析
   - 不良率趋势与行业对比
   - 逾期贷款剪刀差（逾期90天以上/不良）
   - 拨备覆盖率充足性
   - 关注类贷款迁徙率
3. 资本充足率与TLAC达标
   - 核心一级资本充足率
   - TLAC风险加权比率达标进度
4. 分红政策与股东回报
   - 近5年分红率趋势
   - 股息率横向对比

## T05: 治理结构与管理层画像

**目标**：评估公司治理质量与管理层能力

**执行要点**：

   - 管理层持股与利益一致性
3. 关联交易与合规
   - 关联交易规模与公允性
   - 近年监管处罚记录

## T06: 估值深潜与安全边际

**目标**：多维度估值测算，确定安全边际

**执行要点**：
1. PB-ROE估值法
   - 当前PB vs 历史5年/10年分位数
   - 合理PB区间测算（基于可持续ROE）
2. DDM估值模型
   - 假设：折现率、分红增长率、永续增长率
   - 内在价值测算
3. 可比公司估值对比
   - 股份行PB横向对比
   - 估值溢价/折价原因分析
4. 安全边际计算
   - 当前价格 vs 内在价值
   - 安全边际百分比

## T08: 逆向思考与尾部风险

**执行要点**：
1. 房地产敞口压力测试

2. 地方融资平台风险

3. 利率风险
   - 极端降息对NIM的冲击
4. 科技颠覆风险
   - 金融科技对传统银行的侵蚀
5. 系统性风险
   - 银行间市场流动性风险
   - 信用风险传染

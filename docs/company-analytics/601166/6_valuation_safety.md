### 3.2 DDM估值计算

```
阶段1（前5年）：DPS = 0.90元/年，不变
现值 = 0.90 × [(1-1.10^-5)/0.10] = 0.90 × 3.791 = 3.41元

阶段2（永续）：DPS = 0.90 × (1+1%) / (10%-1%) = 10.10元
现值 = 10.10 / 1.10^5 = 10.10 / 1.611 = 6.27元

DDM内在价值 = 3.41 + 6.27 = 9.68元/股
```

# 兴业银行（601166.SH）深度研究需求文档

## 1. 研究目标

基于价值投资框架（巴菲特/芒格），对兴业银行进行全面深度剖析，产出专业级别投研报告，最终给出明确的投资评级（买入/持有/卖出）及买入时机判断。

| 字段 | 内容 |

## 3. 核心研究维度（11项）

### 3.1 宏观周期、政策导向与行业分析
- 银行业政策环境（LPR改革、金融让利实体、系统重要性银行监管）
- 利率周期判断（当前处于加息/降息/利率走廊哪个阶段）
- 信用周期定位（宽信用/紧信用/结构性宽信用）
- 银行板块估值分位与资金配置情况

### 3.2 商业模式与护城河深潜
- 营收结构拆解（利息净收入、手续费及佣金、投资收益等）
- 兴业银行特色业务（绿色银行、财富管理、投行FICC）
- 护城河评估：牌照壁垒、客户粘性、品牌溢价

### 3.3 竞争格局与对手对比
- 股份制银行同业对比（招商、中信、浦发、民生、光大、平安、华夏）
- 市场份额变化趋势
- 差异化竞争优势

### 3.4 财务质量与资本配置
- ROE杜邦拆解（净息差NIM、资产周转率、杠杆倍数）
- 资产质量（不良率、关注类、拨备覆盖率、逾期剪刀差）
- 资本充足率与TLAC达标进度
- 分红率与股息率

### 3.5 治理结构与管理层画像
- 股权结构（福建省财政厅、中国人保等主要股东）
- 管理层稳定性与战略执行力
- 关联交易与合规风险

### 3.6 估值深潜与安全边际
- PB-ROE估值框架
- 历史5-10年PB分位数
- DDM估值模型测算内在价值

### 3.7 催化剂与拐点分析
- 未来1-2季度的业绩超预期点
- 息差拐点、资产质量改善、中收增长等边际变化

### 3.8 逆向思考与尾部风险
- 事前验尸：导致股价暴跌50%+的极端场景
- 房地产敞口、地方融资平台风险、金融科技冲击

### 3.9 资金面与情绪面
- 北向资金持仓变化
- 机构持仓拥挤度
- 融资融券数据

### 3.10 终极检查清单
- 芒格栅格模型多维复核
- "Too Hard"堆判断

### 3.11 最终投资决策报告
- 综合评级与建议仓位
- 核心逻辑总结
- "为什么是现在？"

## 4. 输出要求

- 所有报告存入 `reports/601166/` 目录
- 使用标准金融术语，禁止空谈
- 优先使用最新财报数据（2025年报/2026Q1）
- 每一步完成后暂停，等待用户确认再继续

## 5. 数据来源要求

- Wind/iFinD/Choice等金融终端数据优先
- 公司公告、年报、季报
- 央行、银保监会政策文件
- 卖方研报（需打折降权处理）

# 0. 简单性与能力圈初筛 - 兴业银行 (601166.SH)

## 业务可理解性

**核心逻辑**：兴业银行是中国领先的股份制商业银行。它通过吸纳存款并向企业（特别是工业和制造业）及个人发放贷款，赚取利息差。

- **产品逻辑**：传统的存贷汇业务 + 强大的对公业务（投行、同业、绿色金融）。
- **盈利逻辑**：**净利息收入 (NIM) + 手续费佣金 (中收) - 风险拨备 - 运营支出**。
- **商业本质**：本质上是**“全产业链金融服务的数字化集成商”**。

## 复杂性风险评估

1. **净息差 (NIM) 压力**：在 2025-2026 年低利率环境下，资产端收益下降快于负债端成本，NIM 的稳定性是最大的复杂变量。
2. **房地产敞口风险**：虽然在持续压降，但作为对公业务极其强大的银行，其底层资产的信用风险与宏观经济和地产周期深度绑定。
3. **“投行+财富”转型的成效**：非息收入的增长能否抵消利息收入的下滑，需要通过季度财报中的中收结构来验证。

## 结论

**结论**：**属于“业务逻辑经典，但财务底层受宏观变量（利率、坏账）高度扰动”堆**。

兴业银行是典型的“价值股”。在 2026 年低估值背景下，其作为“绿色金融”和“同业之王”的独特性使其具备较强的韧性。

# 2. 财务生存能力与单体模型对决

## 谁在变现“利差”？谁在变现“效率”？

化工行业的财务真相：资本支出与折旧周期的残酷算术。

### 1. 万华化学：高利润支撑下的“持续扩张”
- **单体模型**：单吨利润率显著高于同行。万华赚的不仅是加工费，更是**“技术代差红利”**。
- **财务特征**：**极高的 ROIC (投入资本回报率)**。其资产负债表极具扩张弹性。
- **生存现状**：极度安全。

### 2. 巴斯夫：受困于“老旧资产与能源重担”
- **单体模型**：欧洲基地的单体利润已严重受损。巴斯夫正在通过抛售非核心资产和投资中国（湛江一体化基地）来进行财务“换血”。
- **财务特征**：**现金流承压**。正在进行大规模的成本削减计划。
- **生存现状**：安全，但处于衰退边缘，必须通过中国市场“续命”。

### 3. 陶氏：稳健的“周期性回报”
- **单体模型**：更贴近终端消费，利润波动受大宗商品影响相对较小。
- **生存现状**：稳健。

## 财务指标对比 (2024-2025 预估)

| 指标 | 万华化学 | 巴斯夫 | 陶氏 |
| :--- | :--- | :--- | :--- |
| **净利率** | **~15%-18%** | ~2%-5% (受能源影响) | ~6%-10% |
| **ROIC** | **~20%** | ~5% | ~10% |
| **资产负债率** | 中 (扩张期) | 高 (重组期) | 中 |
| **现金流健康度** | 卓越 | 承压 | 卓越 |

## 结论：谁的财务逻辑更具韧性？

- **万华化学** 的财务逻辑最性感。它正在用 MDI 赚来的超额利润，疯狂投资精细化工，实现从“单核”到“多核”的跃迁。
- **巴斯夫** 正在经历**“存量出清”**的痛苦，其财务韧性取决于湛江基地的投产速度。

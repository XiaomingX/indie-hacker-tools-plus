## 3. 护城河评级标准

### 护城河类型量化评分
- **自然禀赋**: 岸线稀缺、水深、区位（0-30分）
- **腹地经济**: 制造业集群、经济纵深（0-20分）
- **枢纽地位**: 干线港、船舶大型化（0-20分）
- **特许经营权**: 码头专营权年限（0-15分）
- **转换成本**: 船公司粘性（0-15分）

### 综合评级
- **AAA (80+)**: 极强的持久竞争优势
- **AA (60-79)**: 较强竞争优势
- **A (40-59)**: 中等竞争优势
- **BBB (20-39)**: 较弱竞争优势
- **BB (<20)**: 几乎无护城河

## 4. 上港集团专题分析框架

### 4.1 核心问题
为何上港集团发展最迅速，市值最高？

### 4.2 分析维度
1. **自然禀赋**：长江口+深水岸线+水深优势
2. **腹地经济**：长三角最强制造业+外贸
3. **枢纽地位**：全球第一大港+干线港
4. **政策加持**：自贸区+一带一路
5. **管理层能力**：港口运营效率最高
6. **资本配置**：多元化投资（地产+金融）

### 4.3 关键数据对比
| 指标 | 上港集团 | 宁波港 | 青岛港 |
|------|----------|--------|--------|
| 集装箱吞吐量 | 全球第一 | 全球第三 | 北方第一 |
| 净利润 | 最高 | 次高 | 中等 |
| 市值 | 最高 | 次高 | 中等 |
| 毛利率 | 最高 | 中等 | 中等 |

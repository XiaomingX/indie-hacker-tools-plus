# 0. 商业模式与基本盘对决：上港集团 vs 宁波港 vs 青岛港

## 核心赚钱逻辑：门户税 vs 规模效应 vs 运营效率

剥开“海上丝绸之路”的宏大叙事，港口公司的商业模式本质是**“地理位置驱动的重资产过路费”**。其盈利逻辑取决于其腹地的经济活力与装卸效率：

1. **上港集团 (SIPG)**：**全球贸易的“黄金闸口”**。
   - **底层逻辑**：上港坐拥上海国际航运中心，核心基本盘是**极高毛利的集装箱业务**。它是典型的“门户经济”，赚的是全球价值链中最顶层、最标准化的流量税。由于上海腹地的经济不可替代性，上港是行业内溢价能力最强的玩家。
2. **宁波港 (Ningbo Port)**：**全能型的“货运航母”**。
   - **底层逻辑**：宁波港靠的是**极致的规模与货种多样性**。除了集装箱，它在铁矿石、原油等大宗商品分拨上具有绝对优势。它更像是一个“综合物流超市”，通过“全品类+超大规模”来摊薄固定资产的折旧压力。其基本盘在于长三角制造业的强劲韧性。
3. **青岛港 (Qingdao Port)**：**高净值的“外贸尖兵”**。
   - **底层逻辑**：青岛港是北方外贸的第一大口岸，其核心基本盘是**极致的运营效率与精细化管理**。它不盲目追求吞吐量第一，而是追求单位资产的最高产出率。它是北方能源、矿石和冷链物流的枢纽，赚的是“高周转”带来的效率溢价。

## 营收与成本结构对比 (2025 预估)

| 指标 | 上港集团 | 宁波港 | 青岛港 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 集装箱装卸 (绝对核心)、配套物流 | 综合货运装卸 (大宗+集装箱)、物流贸易 | 外贸集装箱、干散货、液体化工 |
| **毛利属性** | **最高** (集装箱费率高且稳定) | 中 (受大宗商品波动及贸易类业务摊薄) | 高 (外贸占比大，装卸效率行业领先) |
| **成本结构** | 固定折旧与人工 (占比 60%+) | 极高的资产折旧与能耗成本 | 相对精简的成本控制，高自动化的红利 |
| **腹地依赖** | 长三角核心区、长江流域 | 全球制造基地、全国大宗储备 | 华北及黄河流域、外贸出口导向 |

## 结论：基本盘的稳固度

- **上港集团** 的基本盘是“金融+航运”的双中心地位，只要上海还是中国与世界的连接点，其“收税权”就不可动摇。
- **青岛港** 的基本盘是“效率+外贸”，在北方港口整合中处于绝对的领导与收割地位。
- **宁波港** 的基本盘是“规模”，由于其建设投入大、战线长，在面临全球贸易萎缩时，其重资产带来的高经营杠杆风险远超前两者。

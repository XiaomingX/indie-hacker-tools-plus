# 3. 竞争格局与市场份额 - OTA 行业 (2026)

## 1. 行业格局：从“内部对决”到“外部蚕食”

2026 年，OTA 市场的竞争早已不局限于携程、美团和同程。

- **赛道 A：传统 OTA (搜索/服务导向)**
  - **领头羊**：携程 (占有率 ~45%)。
  - **挑战者**：美团 (酒旅)、同程、飞猪。
- **赛道 B：内容社交平台 (推荐/种草导向)**
  - **颠覆者**：**抖音** (GMV 爆发期)、**小红书** (精准种草)。
- **赛道 C：自营直销 (DTC 导向)**
  - **代表**：华住会、锦江、万豪。

## 3. 竞争壁垒与压力测试

### 3.1 抖音的“囤券模式”对携程的冲击
- **逻辑**：抖音通过直播间低价预售（囤券），将需求提前锁定。虽然退单率高达 50%-60%，但其产生的总 GMV 已严重影响了 OTA 的现付单量。
- **携程应对**：强化“即兴出游”的确定性和售后优势。2026 年，携程在海外市场的成功（Trip.com 贡献了约 20% 营收）为其提供了抖音无法企及的全球化避风港。

### 3.2 美团与同程的“下沉大战”
- **逻辑**：同程在下沉市场（二三线城市）的渗透已近饱和，美团通过“零售+酒旅”的组合拳在县域旅游中发起侧翼进攻。

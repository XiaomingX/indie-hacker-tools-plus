# 0. 简单性与能力圈初筛 - OTA 行业 (携程、美团、同程)

## 业务可理解性

**核心逻辑**：在线旅游 (OTA) 行业本质上是 **“旅游服务的数字中介与流量变现平台”**。其核心价值在于撮合上游碎片化的资源（航司、酒店、景区）与下游长尾的消费者需求。

- **产品逻辑**：信息聚合 + 支付担保 + 售后服务 + 会员体系。
- **盈利逻辑**：佣金 (Take Rate) + 广告位 (Marketing Services) + 增值服务 (保险、金融)。
- **行业本质**：**“资源控制力与服务履约的护城河”**。携程卖的是确定性，美团卖的是便利性，同程卖的是可触达性。

## 复杂性风险评估

1. **宏观周期敏感性**：旅游业是典型的可选消费，对 GDP 增速、地缘政治、甚至传染病极其敏感。2025-2026 年面临的是全球消费分级趋势。
2. **供应链碎片化**：上游资源（尤其是酒店）极其分散，建立起深厚且独占的供应链需要数十年的积累。
3. **竞争烈度**：抖音、小红书等内容平台正在通过“内容即交易”蚕食传统 OTA 的增量流量。

## 结论

**结论**：**属于“业务边界清晰、现金流极佳、但受宏观扰动大”的典型平台型生意**。

OTA 行业是 2026 年消费复苏中确定性最高的赛道之一。虽然面临流量变迁，但携程等巨头的供应链深度和会员心智在短期内难以被撼动。

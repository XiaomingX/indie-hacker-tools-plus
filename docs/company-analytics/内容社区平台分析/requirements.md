# 内容社区平台竞品分析需求文档

## 1. 项目概述

### 分析目标
以巴菲特和芒格的价值投资视角，对中国主流内容社区平台进行深度分析，识别各平台的内在禀赋与长期投资价值。

### 分析对象
| 平台 | 母公司 | 成立时间 | 核心定位 |
|------|--------|---------|---------|
| 米游社 | 米哈游 | 2019年 | 游戏社区+官方内容 |
| 堆糖 | 堆糖信息 | 2011年 | 图片收藏+生活美学 |
| CSDN | 创新乐知 | 2000年 | 程序员社区+技术内容 |
| 牛客 | 牛客科技 | 2014年 | 求职社区+笔面经 |
| 即刻 | 估计信息 | 2015年 | 兴趣社交+即时话题 |
| 知乎 | 智者天下 | 2010年 | 问答社区+知识付费 |
| 豆瓣 | 豆瓣网 | 2005年 | 兴趣社区+评分体系 |
| 小黑盒 | 酷郭技术 | 2017年 | 游戏社区+战绩数据 |
| 爱发电 | 爱发电网络 | 2019年 | 创作者变现平台 |

## 2. 分析维度框架

### 2.1 发展历程与时间窗口 (When & Why Now?)
- 创建时间线与关键里程碑
- 融资历史与资本背书
- 移动互联网/短视频红利期把握
- 当前时间窗口的边际变化

### 2.2 商业模式与护城河 (Moat)
- 营收结构定量拆解（广告/付费/增值）
- 定价权来源
- 无形资产（品牌、数据资产）
- 转换成本（用户粘性）
- 网络效应（社区互动）
- 内容壁垒

### 2.3 竞争格局与禀赋评估 (Competitive Advantage)
- 行业集中度与竞争态势
- 各平台差异化定位
- 资源禀赋对比
- 管理层能力圈

### 2.4 治理结构与利益一致性
- 股权结构与资本背景
- 激励机制
- 内容生态治理

### 2.5 财务质量与资本配置
- 自由现金流
- 商业化成熟度
- 资产质量

### 2.6 逆向风险评估 (Pre-mortem)
- 监管政策风险
- 竞争替代威胁
- 用户迁移风险
- 极端场景推演

### 核心问题回答
1. 每个平台的**内在禀赋**是什么？
2. 在巴菲特/芒格框架下，各平台的**长期确定性**如何？
3. **为什么是现在**这个时间点值得关注？
4. 各平台的**致命弱点**和**尾部风险**？

## 4. 分析方法论

### 芒格栅格模型应用
- 诚实性检查：平台数据真实性与动机
- 能力圈检查：管理层是否在能力圈内运营？
- 冗余检查：是否存在单一依赖风险？
- 失败成本检查：用户迁移代价

### 巴菲特护城河检查清单
- [ ] 是不是有定价权？
- [ ] 是否有无形资产保护？
- [ ] 转换成本是否足够高？
- [ ] 是否有网络效应？
- [ ] 内容壁垒是否可持续？

---

## 5. 约束条件

- **时间范围**：2000-2025年发展周期
- **地域范围**：中国大陆市场为主
- **信息截止**：基于公开信息分析，不涉及内幕
- **视角定位**：投资研究视角，非运营诊断

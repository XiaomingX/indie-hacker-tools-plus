# 0. 商业模式与基本盘对决：同花顺 vs 东方财富 vs 恒生电子

## 核心赚钱逻辑：流量税 vs 牌照税 vs 软件地租

金融科技行业的商业模式本质是**“金融服务的数字化重构”**。然而，这三家龙头的基本盘分别占据了产业链的不同关口：

1. **同花顺 (Hithink)**：**极致的“流量税收官”**。
   - **底层逻辑**：同花顺是中国最大的 C 端证券行情入口。它本质上是一个**流量中转站**。通过免费行情吸引海量散户，然后将这些流量卖给券商（开户导流）、卖给散户（炒股软件/AI投顾）。它是轻资产、高毛利的典型。其基本盘在于**C 端用户心智的垄断**和**极致的工具属性**。
2. **东方财富 (East Money)**：**互联网金融的“闭环收割者”**。
   - **底层逻辑**：东财通过财经门户（东方财富网）和天天基金网获取流量，但它不甘于只做中介，而是通过获取券商、基金、期货等**全牌照**，将流量彻底“内化”在自己的金融闭环中。它赚的是交易佣金、两融利息和基金代销费。其基本盘在于其**庞大的客户资产规模 (AUM)** 和**全生命周期的金融服务能力**。
3. **恒生电子 (Hundsun)**：**金融机构的“系统房东”**。
   - **底层逻辑**：恒生电子是 B 端机构（券商、公募、保险）的底层操作系统供应商。它不直接面对散户，而是通过售卖交易系统、资管系统实现“软件地租”。其基本盘在于**极高的替换成本**和**金融监管合规的技术垄断**。

## 营收与成本结构对比 (2025 预估)

| 指标 | 同花顺 | 东方财富 | 恒生电子 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 互联网推广服务 (导流) + 增值电信服务 | 证券业务 (佣金/利息) + 基金销售 | 软件产品销售 + 技术维护 |
| **毛利属性** | **最高** (90% 左右，典型的互联网平台) | 中高 (证券业务受资本开支影响) | 中 (受研发投入及定制化人力成本拖累) |
| **业绩锚点** | **成交额 (QPS)**：市场越热，导流越赚 | **资产规模 (AUM)**：存量资产产生的利息与佣金 | **IT预算**：金融机构的系统升级周期 |
| **AI 变现路径** | C 端 AI 投顾工具、智能选股 | 内部运营提效、智能理财推荐 | B 端核心交易系统的 AI 赋能 |

## 结论：基本盘的稳固度

- **恒生电子** 的基本盘最“硬”，金融机构更换核心系统的代价接近于“换心”，粘性极强。
- **同花顺** 的基本盘最“活”，它是散户炒股的第一触点，只要市场有波动，它就是最大的受益者。
- **东方财富** 的基本盘最“沉”，它深度参与了金融市场的风险承担（两融、质押等），其业绩受股市牛熊的直接冲击最为猛烈，是典型的“杠杆型”金融科技。

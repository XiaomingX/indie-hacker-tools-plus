# 2. 商业模式与单体经济模型拆解 (Unit Economics)

## 两种截然不同的“收税”方式
游戏分发平台的单体经济（Unit Economics）揭示了其盈利能力的上限。

### 1. Web 流量套利模型 (Poki / CrazyGames)
- **收入端 (ARPU)**：极低。由于是网页休闲游戏，单用户停留时间较短，主要靠插屏广告和 15-30 秒的激励视频获利。单用户日贡献价值（ARPDAU）通常仅在 0.05 - 0.15 美分。
- **成本端 (TAC)**：极低。核心竞争力是 **SEO (搜索引擎优化)**。只要用户在 Google 搜索 "Free Games"，Poki 出现在第一页，其获客成本（CAC）几乎为零。
- **模型特征**：**“高周转、极低毛利率、海量长尾”**。这是一门赚取流量差价的生意，其毛利取决于“广告收入 - 给开发者的分成”。

### 2. 社区广告模型 (TapTap)
- **收入端 (Ad Revenue)**：高。TapTap 并不抽取 30% 的游戏内购分成。它的收入全部来自广告位竞价。
- **核心指标：变现效率（ARPU/MAU）**：
  - 心动公司 2024-2025 年财报显示，TapTap 平台的广告收入增速远超 MAU 增速。这意味着单用户贡献的广告价值在提升。
  - **逻辑**：因为它拥有海量的独家内容和玩家评价，广告主（游戏公司）愿意为了在 TapTap 获得一个推荐位而支付远高于传统商店的 CPC。
- **成本端 (R&D & Servers)**：极高。为了维持“零分成”的高冷形象，TapTap 必须投入巨额的服务器成本来提供免费的游戏下载带宽，并投入研发资金维持社区活跃度。

## 结论
Poki 玩的是**流量规模**，TapTap 玩的是**流量浓度**。从 Unit Economics 看，Poki 的抗风险能力极强（成本极低），而 TapTap 必须持续产出高质量的自研游戏或独占游戏（如《心动小镇》）来维持其平台的“内容引力”，否则其高昂的运营开支将迅速拖垮其广告毛利。这是一门需要不断通过昂贵的“内容研发”来续命的平台生意。

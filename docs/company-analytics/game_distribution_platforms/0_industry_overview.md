# 0. 行业定义、TAM与产业链全景拆解

## 业务属性界定：无资产的注意力收割业
游戏分发平台（Poki, CrazyGames, TapTap）在本质上并不是“游戏公司”，而是**“精准流量的批发商”**。
它们的核心商业动作是：利用搜索引擎优化（SEO）、社交分享或社区内容吸引玩家，并在玩家寻找游戏的“决策窗口期”将其流量出售给开发者（通过广告或分成）。它们不拥有核心的游戏研发资产，其价值完全取决于其对“玩家注意力”的截留效率。

## 产业链产值分布与极度倾斜的利润结构
在这个产业链中，分发平台是掌握“生杀大权”的食利者：

1. **上游：游戏开发者（风险最高、存活率最低的苦力）**
   - **角色**：独立工作室、商业游戏公司。
   - **商业模式**：投入高额研发费用，赌一个爆款概率。在传统商店（Apple/Google），他们要缴纳 30% 的“过路费”。在 Poki/CrazyGames 这种 Web 平台，虽然免除了商店税，但必须将一半以上的屏幕空间让给广告，利润被广告收入分成蚕食。
2. **中游：分发平台与门户（稳赚不赔的抽水机）**
   - **角色**：Poki、TapTap。
   - **商业模式**：
     - **Web 门户（Poki/CrazyGames）**：低买高卖。通过 SEO 获得近乎免费的流量，再通过激励视频广告将流量卖给广告主。
     - **垂直社区（TapTap）**：博弈税收。通过“不分成”吸引高质量内容，从而沉淀海量核心玩家，再向开发者反向收取广告费。
3. **下游：广告主（真实的买单者）**
   - **角色**：其他急需买量的游戏、电商应用。
   - **商业模式**：通过这些平台寻找精准的“游戏化流量”，支付 CPC/CPM 成本。

## 价值分配结论
分发平台是整个游戏工业中最具抗风险能力的环节。无论游戏本身亏损还是盈利，只要玩家还在通过这些入口寻找游戏，平台就能收割稳健的流量税。其中，Web 平台的护城河在于 SEO 流量深度，而社区平台（TapTap）的护城河在于“品牌信誉与用户密度”。这是典型的利用他人研发风险实现自我利润留存的生意。

# 0. 商业模式与基本盘对决：Shein vs Temu vs AliExpress

## 核心赚钱逻辑：柔性供应链 vs 极致全托管 vs 平台流量税

出海电商的对决本质上是**“如何将中国过剩的产能，以最高效、低价的方式分发给全球穷人”**。

1. **Shein**：**极致的“实时快时尚与柔性供应链”**。
   - **底层逻辑**：Shein 不是电商，它是**“工业化的数据驱动时尚机”**。它通过抓取全网流行趋势，给几千家小工厂下 100 件起的小单（小武基模式），卖爆了再补单。它赚的是**“预测的准确性”**。其基本盘是其**深度掌控的数千家中国南方服装工厂和数亿高粘性的欧美年轻女性用户**。
2. **Temu (拼多多海外版)**：**极致的“全托管与流量价格屠夫”**。
   - **底层逻辑**：Temu 赚的是**“剥夺商家定价权后的极致利差”**。它通过“全托管”模式，让商家只负责生产，自己负责定价、物流和营销。它赚的是**“全链路极致压榨带来的低价税”**。其基本盘是拼多多母公司的**巨额流量补贴能力和对供应链成本的变态级把控**。
3. **AliExpress (速卖通)**：**“全球化的轻资产平台”**。
   - **底层逻辑**：阿里海外业务的底座。它赚的是**“平台服务费与广告税”**。它更像是一个国际版的淘宝，连接中国商家和海外买家。其基本盘是**菜鸟网络的跨境物流基建和多年的海外运营积累**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | Shein | Temu | AliExpress |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 货品差价 (自营占比高) | 佣金 + 营销服务费 | 平台扣点 + 广告费 |
| **存货风险** | 极低 (柔性供应链) | 低 (全托管模式) | 无 (轻资产) |
| **履约成本 (UE)** | 中 (空运为主) | **最高** (由于极致低价，物流成本极其敏感) | 中 (菜鸟自建网络摊薄) |
| **变现瓶颈** | 品牌向上突破难 | 地缘政治与合规成本爆炸 | 内部组织僵化，反应慢 |

## 结论：基本盘的稳固度

- **Shein** 的基本盘最“深”，它与数千家工厂的深度利益绑定是不可克隆的；
- **Temu** 的基本盘最“猛”，但在没有母公司输血的情况下，其自造血能力尚存疑；
- **AliExpress** 的基本盘最“老”，正在通过 Choice (全托管) 转型苦苦追赶。

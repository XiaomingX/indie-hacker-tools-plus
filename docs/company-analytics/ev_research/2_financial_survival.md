# 2. 财务生存能力与单体模型对决

## 谁在变现“硬件”？谁在变现“梦想”？

电动车行业的财务真相：现金流与毛利率的极限生死线。

### 1. 比亚迪：规模效应下的“赚钱机器”
- **单体模型**：依靠秦、宋等爆款车型实现极高的工厂产能利用率。由于核心零部件全部自研，比亚迪的单车净利润在国产阵营中首屈一指。
- **财务特征**：**现金流极其充沛**，已具备内生增长能力。
- **生存现状**：极度安全。

### 2. 特斯拉：高毛利后的“周期性波动”
- **单体模型**：Model 3/Y 是全球单体利润率最高的车型之一。但随着车型老化，特斯拉不得不通过频繁调价来维持销量，导致毛利承压。
- **财务特征**：**极佳的财务底子**，但由于资本支出巨大（Dojo、Optimus），其估值正经历从“成长股”向“成熟股”转型的阵痛。
- **生存现状**：极度安全。

### 3. 小米汽车：初生牛犊的“极致效率”
- **单体模型**：SU7 上市即爆火，证明了其单店订单获取效率远超传统车企。尽管目前仍处于投入期，但其毛利率转正速度惊人。
- **财务特征**：**“小米式的精益管理”**。依托手机业务产生的百亿现金，小米造车几乎没有融资压力。
- **生存现状**：极度安全。

## 财务指标对比 (2024-2025 预估)

| 指标 | 比亚迪 | 特斯拉 | 小米汽车 |
| :--- | :--- | :--- | :--- |
| **净利率** | **~6%-8%** | ~10%-12% | 暂为负 (预计 2026 转正) |
| **研发/营收比** | ~5% | ~4% | **最高** (投入期) |
| **存货周转天数** | 优 | **最优** | 优 (由于供不应求) |
| **现金储备** | 卓越 | 卓越 | 卓越 (千亿级手机现金) |

## 结论：谁的财务逻辑更具韧性？

- **比亚迪** 的韧性来自于其**“由于垂直整合而产生的利润缓冲带”**。即便下游打价格战，它可以通过上游零部件降价来对冲。
- **小米汽车** 的韧性来自于其**“由于手机基本盘而产生的容错率”**。它可以容忍汽车业务三年的亏损，只要能完成生态占位。

# 2. 财务生存能力与单体模型对决

## 谁在变现“利差”？谁在变现“效率”？

### 1. 中信证券：重资本的“平衡术”
- **单体模型**：依靠庞大的资产负债表进行运作。通过融资融券、自营投资获取收益。其财务受利率水平和市场流动性影响极大。
- **财务特征**：**营收规模巨大，但利润弹性相对较弱**。
- **生存现状**：极度安全。

### 2. 东方财富：轻资产的“杠杆奇迹”
- **单体模型**：边际成本极低。由于流量内生，其单次交易或单笔申购的净利率极高。
- **财务特征**：**净利率全行业第一**（预计 50% 左右）。在牛市中，它是弹性最大的标的。
- **生存现状**：极度安全。

### 3. 富途控股：高 ARPU 支撑下的“全球税收”
- **单体模型**：单用户贡献极高。依靠高频交易和利息收入实现盈利。
- **财务特征**：**现金流极佳**，但受限于合规成本和海外市场营销开支。
- **生存现状**：安全。

## 财务指标对比 (2024-2025 预估)

| 指标 | 中信证券 | 东方财富 | 富途控股 |
| :--- | :--- | :--- | :--- |
| **净利率** | ~20%-25% | **~50%** | ~40% |
| **ROE** | 中 (~8%-10%) | **最高 (~15%-20%)** | 高 |
| **获客成本比例** | 较高 (人力) | **最低** | 中 |
| **资产周转率** | 低 (重资产) | **高** | 高 |

## 结论：谁的财务逻辑更具韧性？

- **东方财富** 的财务逻辑最性感。它已经完成了从“流量”到“变现”的闭环，且没有沉重的线下网点负担。
- **中信证券** 的财务逻辑最稳健，它拥有对抗长期熊市的资产厚度。

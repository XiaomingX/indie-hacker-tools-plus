# 0. 商业模式与基本盘对决：中信证券 vs 东方财富 vs 富途控股

## 核心赚钱逻辑：资本投行霸权 vs 流量代销帝国 vs 极致体验的数字经纪

证券行业的对决本质上是**“对资产端和流量端的绝对占有权之争”**。

1. **中信证券 (Citic)**：**极致的“牌照极权与头部溢价”**。
   - **底层逻辑**：中信是“券商中的老大哥”。它通过深厚的政企关系和资产端的绝对优势，赚的是**“制度性溢价”**。其基本盘是**国内几乎所有重大 IPO、并购和机构交易的参与权**。它赚的是“富人的钱”和“大机构的钱”。
2. **东方财富 (East Money)**：**极致的“基金代销与长尾流量变现”**。
   - **底层逻辑**：东方财富不是券商，它是**“披着券商皮的互联网流量平台”**。通过天天基金网积累的亿级理财流量，以极低成本转化为证券交易。它赚的是**“流量分发的边际利润”**。其基本盘是**海量的国内基民和股民群体**。
3. **富途控股 (Futu)**：**极致的“产品体验与全球化经纪”**。
   - **底层逻辑**：富途赚的是**“跨市场交易的效率税”**。通过极致的 App 体验和社区属性，吸引全球（尤其是离岸中国背景）的高净值零售投资者。它赚的是**“极致用户体验带来的高频交易佣金”**。其基本盘是**高 ARPU 值的全球华人及国际化零售投资者**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | 中信证券 | 东方财富 | 富途控股 |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 投行 (20%) + 自营 (40%) + 经纪 | 基金代销 (核心) + 经纪 | 证券经纪 (核心) + 利息收入 |
| **毛利率** | 中 (受人力和重资产拖累) | **最高** (纯互联网模式) | 高 (轻资产) |
| **获客成本 (CAC)** | 高 (依赖线下与关系) | **最低** (内源流量) | 中 (主要靠口碑与买量) |
| **变现瓶颈** | 强监管导致的业务受限 | 基金市场低迷带来的代销下滑 | 地缘政治与跨境监管风险 |

## 结论：基本盘的稳固度

- **中信证券** 的基本盘最稳，它是金融系统的基石；
- **东方财富** 的基本盘最广，它占据了大众理财的入口；
- **富途控股** 的基本盘最具弹性，受全球市场波动和监管政策影响最剧烈。

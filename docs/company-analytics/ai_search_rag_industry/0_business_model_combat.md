# 0. 商业模式与基本盘对决：Perplexity vs Tavily vs Exa (fka Metaphor)

## 核心赚钱逻辑：面向人的答案 vs 面向 AI 的索引 vs 语义化搜索引擎

AI 搜索赛道已经分裂为“面向终端用户”和“面向开发者（AI Agent）”两个完全不同的战场。

1. **Perplexity**：**极致的“人类知识摘要机”**。
   - **底层逻辑**：它是 B2C 模式。它赚的是**“人类搜索效率税”**。它通过整合多个 LLM 和自研 RAG 引擎，直接输出答案。其基本盘是**对传统搜索习惯的革命性替代**。
2. **Tavily**：**Agent 的“专业外挂数据库”**。
   - **底层逻辑**：它是 B2B 模式。它赚的是**“Agent 的算力溢价”**。它为开发者提供专门优化的搜索 API，确保 AI 拿到的数据是干净、结构化且高相关的。其基本盘是**海量的 Agent 开发者生态**。
3. **Exa (Exa.ai)**：**基于“语义嵌入”的下一代索引**。
   - **底层逻辑**：它赚的是**“精准链接税”**。它不按关键词搜，按语义搜。它更像是一个面向机器的“大蓝页”。其基本盘是其**独特的语义检索算法**。

## 营收与成本结构对比 (2025 预估)

| 指标 | Perplexity | Tavily | Exa |
| :--- | :--- | :--- | :--- |
| **主要收入来源** | 订阅费 (Pro) + 广告 (即将上线) | API 调用费 (按量计费) | API 调用费 + 企业定制索引 |
| **毛利属性** | 中 (受大模型 API 调用成本拖累) | **高** (专注数据清洗与索引，算力密度低) | 高 (自研索引架构) |
| **单用户/单调用价值** | 中 (20美元/月) | 低 (但频率极高) | 中 |
| **变现瓶颈** | Google SGE 的正面清剿 | OpenAI/Anthropic 自研搜索工具 | 被传统搜索巨头像素级模仿 |

## 结论：基本盘的稳固度

- **Perplexity** 的基本盘最危险，因为它直接对抗 Google；
- **Tavily** 的基本盘最稳，因为它是 Agent 的“基础设施”，只要 Agent 还要上网，就需要它；
- **Exa** 的基本盘最具爆发力，因为它在重新定义“什么是互联网索引”。

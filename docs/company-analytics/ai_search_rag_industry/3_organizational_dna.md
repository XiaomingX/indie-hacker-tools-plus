# 3. 组织基因、团队与战略分歧

## 核心战略分歧：做 App、做工具还是做协议？

### 1. Perplexity：Aravind 的“反 Google 联盟”
- **组织基因**：**“对传统搜索的复仇”**。团队核心多来自 OpenAI 和 Meta 的研究员。
- **战略表现**：激进。他们想做一个全新的门户，甚至想做一个 AI 时代的浏览器。
- **死穴**：**“路径太长”**。从搜索切到生态，中间隔着 Google 十年的护城河。

### 2. Tavily：开发者背景的“极客思维”
- **组织基因**：**“务实、高效、透明”**。
- **战略表现**：极其克制。不碰 C 端，只做开发者最趁手的搜索工具。
- **优势**：由于专注，所以专业。它几乎是所有著名开源 Agent 框架（如 LangChain）的默认搜索工具。

### 3. Exa：对“链接”的重新定义
- **组织基因**：**“对信息本质的思考”**。
- **战略表现**：他们认为链接不只是文本，而是一种语义节点的连接。

## 战略分歧总结表

| 维度 | Perplexity | Tavily | Exa |
| :--- | :--- | :--- | :--- |
| **战略重心** | 用户增长、品牌占领 | 开发者集成、数据质量 | 语义索引、企业级服务 |
| **对待巨头** | 正面硬刚 | 侧翼补充 | 技术创新 |
| **团队底色** | 明星科学家、明星 PM | 资深后端、爬虫专家 | AI 算法极客 |

## 结论：谁的基因更具适应性？

- **Tavily** 的基因目前最契合 Agent 爆发的趋势。
- **Perplexity** 正在经历从“小而美”到“大而全”的痛苦转型。

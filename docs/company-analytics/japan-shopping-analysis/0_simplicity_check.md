# 0. 简单性与能力圈初筛 - 日本零售/消费行业 (2026)

## 业务可理解性

**核心逻辑**：日本零售业的本质是 **“在极致缩量的老龄化社会中，通过效率优化、审美溢价与全球化扩张，榨取剩余利润”**。2025-2026 年，日本零售业受益于“低汇率带来的入境消费”与“结构性通胀下的消费分级”。

- **产品逻辑**：寻宝式折扣店 (唐吉诃德) vs 极致供应链百元店 (大创/Seria) vs 全球化服饰平台 (优衣库)。
- **盈利逻辑**：高毛利非标品 (唐吉诃德) vs 规模经济 (大创) vs 品牌溢价与全球市场 (迅销)。
- **行业本质**：**“经营效率与用户心智的阵地战”**。

## 复杂性风险评估

1. **汇率波动的双刃剑**：日元持续低位虽然刺激了入境游消费（利好唐吉诃德、优衣库），但显著推高了原材料进口成本（压减百元店利润）。
2. **劳动力成本的刚性上升**：日本劳动力短缺导致薪资水平在 2025-2026 年显著上调，这是所有重资产、人力的零售企业共同的毛利侵蚀风险。
3. **消费分级的非线性特征**：中产阶级缩减开支转向折扣店，而高净值人群依然追逐品牌。

## 结论

**结论**：**属于“现金流极其充沛、管理层极度自律、且具备全球复制能力”的稳健赛道**。

唐吉诃德是捕捉 **“消费娱乐化”** 的首选；迅销 (优衣库) 是捕捉 **“全球化扩张”** 的基石资产；Seria 是 **“审美溢价”** 驱动的效率标杆。

# Task 6: 0_simplicity_check - 能力圈过滤

## 一句话核心逻辑

**星巴克的赚钱逻辑**: "通过'第三空间'体验和全球品牌溢价，在门店销售咖啡及周边产品，赚取售价与运营成本的差价"

---

## 业务可理解性验证

### 核心商业模式

```
收入 = 门店数量 × 单店营收
     = 直营门店(80%) + 特许经营(10%) + 其他(10%)

成本 = 咖啡豆 + 人工 + 租金 + 运营费用

净利润 = 收入 - 成本
```

### 业务结构（2024年）

| 业务 | 占比 | 毛利率 | 说明 |
|------|-----|-------|-----|
| 直营门店 | ~80% | ~70% | 全球35000+门店 |
| 特许经营 | ~10% | ~90% | 品牌授权费 |
| 其他 | ~10% | ~50% | 食品/商品/糖浆 |

## 能力圈判定

### 问题1: 能否用一句话解释星巴克的赚钱逻辑？

**答案**: 可以。星巴克通过"第三空间"体验和品牌溢价，在门店向消费者销售咖啡及食品，赚取售价与运营成本的差价。

### 问题2: 业务逻辑是否过于复杂？

**答案**: 否。星巴克的业务模式非常清晰：
- 门店销售咖啡（占收入80%）
- 每家门店的营收模型可以拆解为：客单价 × 客流量
- 成本结构简单：咖啡豆、人工、租金

### 问题3: 是否有无法理解的技术变革风险？

**答案**: 有一定风险。电动汽车对咖啡店的影响需要关注：
- 充电时间约30分钟，与咖啡消费场景匹配
- 但充电时消费者更可能留在车内而非进店

## 竞争力可理解性

### 星巴克的竞争优势

1. **"第三空间"概念**
   - 消费者在星巴克不只是喝咖啡
   - 是工作、社交、休息的场所
   - 竞争对手难以复制

2. **全球品牌价值**
   - Interbrand全球品牌50强
   - 品牌溢价约30-50%
   - 50年积累的品牌认知

3. **规模经济**
   - 全球35000+门店
   - 统一供应链管理
   - 降低采购成本

4. **数字化粘性**
   - 9000万星享会员
   - 移动支付+积分体系
   - 高频用户数据

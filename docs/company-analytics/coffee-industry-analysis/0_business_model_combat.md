# 0. 商业模式与基本盘对决：星巴克 vs 瑞幸 vs 库迪

## 核心赚钱逻辑：第三空间溢价 vs 极致效率快取 vs 价格战绞肉机

咖啡行业的对决本质上是**“对成瘾性液体的分发成本之争”**。

1. **星巴克 (Starbucks)**：**极致的“第三空间与品牌溢价”**。
   - **底层逻辑**：星巴克卖的不是咖啡，是**“体面地消磨时间”**。它通过高昂的门店装潢和品牌光环，赚的是**“房产中介+社交溢价”**。其基本盘是**中高端白领的商务与休闲需求**。
2. **瑞幸 (Luckin)**：**极致的“数字化快取与爆品引擎”**。
   - **底层逻辑**：瑞幸赚的是**“算法驱动的效率钱”**。通过极致的小店模型（Pick-up）、全自动化订货系统和“生椰拿铁”式的营销节奏，实现了对传统咖啡馆的降维打击。它赚的是**“高周转下的成瘾性消费”**。其基本盘是**千万级别的私域用户和遍布全国的万店网络**。
3. **库迪 (Cotti)**：**“极致价格战”下的加盟博弈**。
   - **底层逻辑**：库迪赚的是**“联营合作伙伴的风险溢价”**。通过 8.8/9.9 元的超低价强行入场，旨在通过规模效应拖死对手。其基本盘是**那些被瑞幸教育过、但依然对价格极度敏感的下沉市场用户**。

## 营收与成本结构对比 (2024-2025 预估)

| 指标 | 星巴克 | 瑞幸 | 库迪 |
| :--- | :--- | :--- | :--- |
| **单杯成本 (含人工/租金)** | **最高** (约 15-20元) | 低 (约 8-10元) | 极低 (但由于低价，毛利微薄) |
| **主要利润源** | 高毛利单品 (如拿铁/季节限定) | 爆款饮品 + 极高的运营效率 | 联营服务费 (当前主要靠烧钱换份额) |
| **货币化率/客单价** | **最高** (约 35-40元) | 中 (约 15-18元) | 极低 (约 9-12元) |
| **变现瓶颈** | 品牌老龄化，性价比竞争 | 价格战导致的单杯毛利压缩 | 现金流枯竭及加盟商流失 |

## 结论：基本盘的稳固度

- **星巴克** 的基本盘最“稳”，但正在流失非核心客群；
- **瑞幸** 的基本盘最“香”，它已经完成了从“便宜”到“习惯”的跨越；
- **库迪** 的基本盘最“悬”，它是一场豪赌，赌的是对手先倒下。

# 8. 逆向思考与尾部风险 - 赣粤高速 (600269.SH)

## 1. 极端情景模拟：收租生意走向衰落的可能路径

为了践行芒格的逆向思维，我们必须构思导致赣粤高速核心价值受损的极端场景。

### 场景 A：货运模式的革命性替代 (概率：极低，影响：致命)
- **过程**：如果低空经济（货运无人机）或极低成本的铁路运输彻底取代了高速公路的中短途货运。
- **结果**：高速公路沦为单纯的客运通道，流量单价暴跌。

### 场景 B：金融黑天鹅的“反噬” (概率：中低，影响：中高)
- **过程**：公司持股的证券公司或银行出现系统性财务危机，不仅公允价值清零，甚至涉及连带债务。
- **结果**：公司多年积累的路费现金被用于填补投资黑洞，分红中断。

### 场景 C：改扩建的“债务陷阱” (概率：中，影响：中)
- **过程**：多个路段同时开工，由于原材料价格暴涨，实际投资远超预算，且银行贷款利率上调。
- **结果**：公司进入漫长的债权偿还期，自由现金流被利息吞噬。

## 2. 核心假设的反向证伪

- **假设 1：赣粤两地间始终有巨大的物流需求**。
  - **证伪**：如果由于产业链转移，导致原本南北运输的需求消失或被海运替代。
- **假设 2：收费年限可以无限延长**。
  - **证伪**：如果国家政策突变，强制取消所有高速公路收费。

---

## 3. 结论与防御建议

赣粤高速的风险不在于“路”，而在于“钱”的分配。

**防御建议**：
1. **剥离非核心金融资产**：督促管理层聚焦主业，减少二级市场波动对净利润的干扰。
2. **动态债务管理**：利用低利率环境置换长债，锁定财务成本。
3. **分红下限承诺**：在公司章程中明确最低分红率，保护股东的收益预期。

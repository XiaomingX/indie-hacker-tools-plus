## 二、宏观周期分析

### 2.1 基建投资周期定位

当前处于"十四五"规划收官阶段（2021-2025），交通基础设施投资仍是稳增长的重要抓手。

- **政策基调**：中央经济工作会议持续强调"适度超前开展基础设施投资"
- **江西省定位**：作为中部崛起战略的重要省份，交通基础设施建设持续受到政策支持
- **周期阶段**：基建投资增速从高速转向中速，但结构性机会仍在

### 2.2 公路货运量与GDP弹性

高速公路车流量与GDP增长高度相关，弹性系数约为0.8-1.2：

| 指标 | 关系 |
|------|------|
| 客车流量 | 与居民消费、汽车保有量正相关 |
| 货车流量 | 与工业生产、物流需求正相关，弹性更高 |
| GDP弹性 | 货运量弹性约1.0-1.2，客运弹性约0.8-1.0 |

**关键驱动因子**：
- 江西省2024年GDP增速约5%，高于全国平均水平
- 汽车保有量持续增长，2024年全国突破3.4亿辆
- 电商物流需求持续扩张

### 2.3 大宗商品影响

| 商品 | 影响路径 | 当前状态 |
|------|---------|---------|
| 油价 | 影响运输成本和车流量 | 油价中枢下移有利于公路运输 |
| 建材 | 影响养护成本和新建项目 | 价格相对稳定 |
| 粮价 | 影响农产品运输需求 | 平稳 |

## 三、行业政策分析

### 3.1 收费期限政策（核心风险点）

**现行法规**：《收费公路管理条例》（2004年）
- 政府还贷公路：最长20年（中西部25年）
- 经营性公路：最长25年（中西部30年）

**政策动态**：
- 二十届三中全会明确提出"推动收费公路政策优化"
- 条例修订已推进多年，业界预期适当延长收费期限
- 大量早期建设的高速公路正面临收费到期问题

**对赣粤高速的影响**：
- 需关注各路产的具体收费到期时间
- 政策修订方向将直接影响公司长期价值

### 3.2 费率调整政策

- 收费标准由国家和省级部门制定，受管制程度高
- 2024年修订的《江西省公路条例》自2025年3月1日起施行
- 探索差异化收费："不同路段、不同时段、不同车型"
- 差异化收费有助于提升路网运营效率

### 3.3 节假日免费政策

- 春节、清明、五一、国庆四个节假日7座以下客车免费
- 绿色通道政策：鲜活农产品运输车辆免费
- **风险提示**：免费政策存在扩大化可能，将直接影响通行费收入

### 3.4 高速公路REITs

- 2021年6月首批基础设施REITs上市，高速公路是重要底层资产
- 代表项目：浙商沪杭甬REIT、华夏越秀高速REIT
- 政策持续推动REITs扩容
- **机会**：REITs为高速公路企业提供新的融资渠道和退出机制
- **挑战**：收费期限剩余年限直接影响REITs估值

## 四、江西省经济与交通分析

### 4.1 经济发展概况

| 指标 | 数据 |
|------|------|
| 2024年GDP | 约3.4万亿元 |
| GDP增速 | 约5%（高于全国） |
| 全国排名 | 约第15位 |
| 经济特征 | 承接产业转移，工业化进程加速 |

### 4.2 交通区位优势

公司路产位于"国家公路网中承上启下的交通要道"：
- 位于京港澳和沪昆两大干线之间
- 拥有多个连接省会和外省的重要通道
- 承接东部产业转移的物流通道

### 4.3 "十四五"交通规划

江西省"十四五"综合交通运输体系规划：
- 构建"六纵六横"综合运输通道
- 推动赣州国际陆港等物流枢纽建设
- 高速公路网络持续完善

## 五、行业Beta判断

### 5.1 正面因素

| 因素 | 影响 |
|------|------|
| 稳增长政策 | 基建投资仍是逆周期调节工具 |
| 产业转移 | 中部地区承接东部产业，物流需求增长 |
| 汽车保有量 | 持续增长，客车流量有支撑 |
| REITs政策 | 提供新的融资和价值发现渠道 |

### 5.2 风险因素

| 因素 | 影响 |
|------|------|
| 收费期限 | 政策不确定性，影响长期价值 |
| 免费政策 | 可能扩大化，减少收入 |
| 高铁竞争 | 对长途客运形成替代 |
| 经济下行 | 货运需求可能下滑 |

### 5.3 综合判断

**行业Beta为中性偏正面**：
- 短期：车流量随经济复苏稳步增长
- 中期：差异化收费政策有望提升运营效率
- 长期：收费期限政策是核心变量，需密切关注修订方向

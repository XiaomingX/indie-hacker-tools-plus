# 3. 竞争格局与市场份额 - 赣粤高速 (600269.SH)

## 1. 行业格局：地域分割的稳定平衡

高速公路行业具有天然的地域属性，省际竞争主要体现在**路网连通性**和**分红回报率**上。

- **阵营 A：全国性龙头**：**招商公路**。优势在于全产业链布局和跨省优质资产。
- **阵营 B：区域性王者**：**赣粤高速**（江西）、**宁沪高速**（江苏）、**深高速**（广东）。各省内核心路段多为独家运营。
- **阵营 C：专业性公司**：重点布局智慧交通或物流配套的标的。

## 3. 竞争壁垒与压力测试

### 3.1 赣粤的“估值洼地”优势
- **性价比**：相比宁沪高速等一线城市标的，赣粤的 PE 更低，股息率极具竞争力。在 2026 年资金寻找“确定性”时，赣粤具备更高的向上弹性。
- **智慧交通输出**：子公司方兴科技不仅服务于母公司，也在向外省输出智慧路网方案，这使其具备了超越“地理限制”的竞争潜能。

### 3.2 潜在挑战
- **跨省竞争**：如果周边省份（如湖北、广东）修建了更具成本优势的平行高速路，可能导致部分长途货车绕行。但赣粤所持路段多为国家主干网（G45等），分流风险可控。

---

## 4. 结论

**竞争地位：A 股公路板块中的“高分红潜力股”**。

赣粤高速不仅在省内拥有无可撼动的垄断力，其较低的估值分位使其在板块内具备更强的吸引力。

**警示**：需关注各公司在 **公募 REITs** 领域的进展。如果招商公路等龙头通过 REITs 实现资产出表加速扩张，赣粤若跟进速度较慢，可能在资产规模上掉队。

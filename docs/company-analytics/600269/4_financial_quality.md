# 4. 财务真实质量与单位经济模型 - 赣粤高速 (600269.SH)

## 1. 模拟股东盈余 (Owner Earnings) 穿透

赣粤高速的财务质量呈现出“核心业务稳健、投资业务波动”的双重特性。

| 项目 (2025 全年实际) | 金额 (亿人民币) | 说明 |
|--------------------|---------------|------|
| **营业收入 (Revenue)** | 62.83 | 同比增长 4.98% |
| **归母净利润** | 12.80 | 基本持平，反映了核心公路业务的稳定性 |
| **减：金融资产公允价值变动** | ~1.5 - 2.5 | 波动核心，2026 Q1 导致大幅利润计提 |
| **加：折旧与摊销 (D&A)** | ~25.0 - 30.0 | 公路股核心现金回流项 |
| **减：资本开支 (改扩建)** | ~15.0 - 20.0 | 维持性开支及部分扩建尾款 |
| **模拟股东盈余 (Owner Earnings)** | **~25.0 - 35.0** | **现金流能力远超净利润，这是高分红的底层逻辑** |

**财务评定**：典型的“重折旧、轻资本”中后期标的。
- **派息能力**：2025 年分红 ~4 亿元（10 派 1.7 元），分红率约 31%。由于 FCF 远高于净利润，公司具备将分红率提升至 50% 以上的财务潜力。
- **单位经济模型 (Unit Economics)**：
  - **单公里营收**：随着昌九高速扩容完成，单公里的车流量天花板被拉高。

## 2. 核心风险指标：金融投资的敞口

### 2.1 金融资产风险 (The "Noise")
- **现状**：公司持有大量券商和银行股权。这些资产在股市波动期会产生剧烈的账面盈亏（公允价值变动），但不影响路费现金收缴。
- **深度解析**：2026 Q1 净利润下滑 35.77% 几乎完全由国盛证券等股价波动引起。投资者应学会“剥离噪音看路费”。

### 2.2 杠杆率与利息压力
- **指标**：资产负债率处于行业平均水平。
- **逻辑**：在 2026 年低利率背景下，公司通过借低利贷置换高利贷，财务费用有望进一步下降，释放利润空间。

---

## 3. 资产收益率与分红确定性

- **经营性现金流净额**：连续多年保持在 30 亿-40 亿规模。
- **安全边际**：路费收入是预收现金性质，没有坏账风险。

---

## 4. 结论与总结

**财务评定：一个被“金融波动”掩盖的优质收租资产**。

赣粤高速的底色是极好的，但金融投资增加了预测难度。

**关键监控指标**：
1. **剔除公允价值变动后的“扣非净利润”表现**。
2. **货币资金充足度**：确保在改扩建高峰期不影响派息。
3. **昌九高速扩建后的费率确认与摊销期限调整**。

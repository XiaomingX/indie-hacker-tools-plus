# 3. 竞争格局与市场份额 - 中国创新药行业 (2026)

## 1. 行业格局：双头并进与赛道突围

2026 年中国创新药市场呈现“综合性巨头”与“专精特新”共存的格局：

- **全球商业化巨头：百济神州 (The Global Giant)**
  - **地位**：在 BTK 抑制剂赛道全球领先，泽布替尼已成为“十亿美金分子”。
  - **2025 动态**：营收规模已接近 400 亿人民币，是真正的跨国药企 (MNC) 预备役。
- **全产业链航母：恒瑞医药 (The Full-service Powerhouse)**
  - **地位**：在 PD-1、ADC、代谢等多个赛道全面布局。
  - **优势**：极强的国内商业化网络，创新药营收占比已突破 58%。
- **赛道定义者：康方生物 / 荣昌生物 / 康诺亚**
  - **地位**：在双抗、ADC、自免等细分赛道拥有全球首创 (First-in-class) 潜力。

## 3. 竞争壁垒与压力测试

### 3.1 百济：防御全球 MNC 的联手围剿
- **逻辑**：泽布替尼正在蚕食阿斯利康、艾伯维等巨头的市场份额。
- **应对**：通过海量头对头 (Head-to-head) 临床数据建立“同类最佳”的学术门槛。

### 3.2 恒瑞：防御“Biotech 借船出海”的侧翼进攻
- **风险**：初创公司通过 BD 方式将海外权益卖给 MNC，可能在海外市场快速超越恒瑞的自营进度。

---

## 4. 结论

**竞争地位：百济是“外战英雄”，恒瑞是“内战霸主”，康方是“技术变量”**。

**关键监控**：
- **国产依沃西与 K 药全球头对头数据的后续长期生存率对比**。
- **信达生物玛仕度肽在 2026 年的销售曲线斜率**。

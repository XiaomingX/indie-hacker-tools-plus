# 创新药公司价值投资分析 - 设计文档

## 分析框架

### 一、创新药行业特性

**创新药 vs 仿制药的本质差异**：

| 特性 | 创新药 | 仿制药 |
|------|--------|--------|
| 研发周期 | 10-15年 | 3-5年 |
| 研发成本 | $2-30亿 | $200-500万 |
| 成功率 | ~10% | ~90% |
| 定价 | 高（专利保护） | 低（市场竞争） |
| 护城河 | 强（专利+独占期） | 弱（集采压力） |
| 代表 | 百济、信达、荣昌 | 恒瑞部分业务 |

### 三、巴菲特护城河五维评估模型（创新药定制版）

```
护城河评估 = f(无形资产, 转换成本, 网络效应, 成本优势, 定价权)
```

**1. 无形资产（Intangible Assets）**
- 研发能力（FIC/BIC管线）
- 专利组合（化合物、制剂、适应症）
- 技术平台（ADC、双抗、CAR-T）
- 人才团队（科学家、临床团队）

**2. 转换成本（Switching Costs）**
- 医生/患者用药习惯
- 医院准入壁垒
- 医保目录粘性
- 临床数据积累

**3. 网络效应（Network Effects）**
- 临床试验网络（患者招募）
- 真实世界数据（RWD）
- KOL关系网络
- 国际临床合作

**4. 成本优势（Cost Advantages）**
- 规模化研发（管线数量）
- 生产成本（生物药 vs 小分子）
- 临床开发效率

**5. 定价权（Pricing Power）**
- 创新药定价能力
- 医保谈判地位
- 差异化竞争优势
- 出海定价能力

### 四、芒格式逆向分析框架

**创新药行业的特殊风险**：

| 风险类型 | 描述 | 影响 |
|---------|------|------|
| 研发风险 | 临床失败 | 管线归零 |
| 审批风险 | FDA/NMPA不批准 | 多年投入损失 |
| 商业化风险 | 销售不达预期 | 峰值销售下调 |
| 竞争风险 | 同靶点扎堆 | 价格战 |
| 政策风险 | 集采/医保谈判 | 利润压缩 |

### 五、估值框架

**创新药公司估值方法**：
```
DCF管线估值 = Σ [峰值销售 × 概率 × 折现因子]

关键参数：
├── Peak Sales（峰值销售）
├── Success Probability（成功概率）
├── Discount Rate（折现率10-12%）
├── Patent Life（专利期）
└── Time to Peak（达峰时间）
```

**成功率参考**：
- 临床I期：60-70%
- 临床II期：30-40%
- 临床III期：50-60%
- NDA/BLA：80-90%

## 分析方法

### 定量分析
- 管线数量与阶段分布
- 研发投入（费用化vs资本化）
- 商业化收入与增速
- 现金流与现金消耗

### 定性分析
- 技术平台竞争力（ADC、双抗）
- 管理层背景与稳定性
- 国际临床试验布局
- 商业化执行力

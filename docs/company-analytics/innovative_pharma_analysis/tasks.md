# 创新药公司价值投资分析 - 任务分解

## 阶段一：基础信息收集与整理

### Task 1.1: 平台发展时间线梳理
**任务**：收集7家公司创立时间、关键里程碑、IPO时间

- [ ] **恒瑞医药 (1970-2024)**
  - 1970年创立，连云港制药厂
  - 2000年上市，专注仿制药
  - 2010年后转型创新药
  - 2021年集采压力，业绩下滑

- [ ] **中国生物制药 (1997-2024)**
  - 1997年创立
  - 2013年收购泰德制药
  - 2018年收购Billrunext
  - 多元化：制药+投资

- [ ] **百济神州 (2010-2024)**
  - 2010年创立，创始团队王晓东、欧雷强
  - 2016年NASDAQ上市
  - 2019年首款自研药泽布替尼获批
  - 2021年科创板上市
  - 全球化最彻底的中国创新药

- [ ] **再鼎医药 (2014-2024)**
  - 2014年创立，杜莹创立
  - 2017年NASDAQ上市
  - 2020年首款产品则民上市
  - License-in模式代表

- [ ] **信达生物 (2011-2024)**
  - 2011年创立，俞德超创立
  - 2018年香港上市
  - 2018年信迪利单抗获批（PD-1）
  - 2023年PD-1出海受阻

- [ ] **荣昌生物 (2008-2024)**
  - 2008年创立，王威东创立
  - 2022年科创板上市
  - 2021年维迪西妥单抗获批（ADC）
  - ADC技术平台领先

- [ ] **百利天恒 (1996-2024)**
  - 1996年创立
  - 2023年科创板上市
  - 2023年BL-B01D1（ADC）授权给BMS
  - 双抗/ADC平台

# 创新药公司价值投资分析 - 需求文档

**注**：用户问题末尾提及"携程"应为复制粘贴错误，本报告基于创新药公司（百济神州、恒瑞医药、中国生物制药、再鼎医药、信达生物、荣昌生物、百利天恒）进行分析。

## 分析目标
以巴菲特和芒格的价值投资框架，对比分析以下7家中国创新药公司的发展历程、禀赋差异：
- 百济神州（BeiGene）
- 恒瑞医药（Hengrui Pharma）
- 中国生物制药（Sino Biopharm）
- 再鼎医药（Zai Lab）
- 信达生物（Innovent Biologics）
- 荣昌生物（RemeGen）
- 百利天恒（Sichuan Baili Pharmaceutical）

## 核心研究问题

### 1. 发展历程对比
- 创立时间与成立背景
- 关键发展里程碑（研发突破、产品上市、IPO）
- 重大危机与应对（集采压力、FDA审查）
- 国际化路径

### 2. 核心产品与业务分析
- 核心产品管线（PD-1、ADC等）
- 创新药研发阶段（临床I/II/III期）
- 商业化能力（销售团队、准入能力）
- 专利布局与技术平台

### 3. 禀赋评估（巴菲特护城河框架）
- **无形资产**：研发能力、专利、技术平台
- **转换成本**：医生/患者粘性、医院准入
- **网络效应**：患者数据、临床试验网络
- **成本优势**：规模化研发、生产成本
- **定价权**：创新药定价、医保谈判

### 4. 芒格式逆向思维
- 各公司"失败场景"推演
- 核心脆弱点识别（集采/医保/竞争）
- 管理层可信度评估

### 5. 核心问题：哪家创新药公司发展最迅速/禀赋最高？
- 研发效率对比（FIC/BIC能力）
- 商业化成功对比
- 国际化能力对比
- 资本配置效率对比

## 分析维度
1. 发展时间线对比
2. 核心产品管线对比
3. 商业模式与商业化能力
4. 创新药行业特性（高风险/高投入/长周期）
5. 国际化 vs 中国市场

## 输出要求
- 生成完整的对比分析报告
- 以投资机构视角输出专业研报格式
- 明确回答"哪家创新药公司发展最迅速/禀赋最高"

## 使用框架
- 巴菲特护城河理论
- 芒格栅格模型
- DCF估值（管线估值）
- 创新药评估（临床成功率、峰值销售）

# 0. 简单性与能力圈初筛 - 创新药行业 (2026)

## 业务可理解性

**核心逻辑**：创新药行业本质上是 **“高风险、长周期、高收益的知识产权生意”**。2025-2026 年，中国创新药企业正式进入了“全球化变现”与“平台化盈利”的交汇期。

- **产品逻辑**：自研管线 (Pipeline) + 全球临床 (Global Trials) + 商业化落地 (Commercialization)。
- **盈利逻辑**：产品销售 (Product Sales) + 对外授权 (Licensing Out/BD)。
- **行业本质**：**“用极高的研发确定性去对抗生理机制的不确定性”**。

## 复杂性风险评估

1. **研发失败的二元性**：临床试验结果（如 3 期数据）是典型的“0 或 1”事件。康方生物依沃西在 2026 年 5 月的数据波动证明了这一风险的剧烈性。
2. **全球化合规与地缘博弈**：对于百济神州等全球化程度极高的公司，FDA 的审批标准和美国《生物安全法案》的潜在扰动是不可控变量。
3. **医保谈判定价压力**：国内市场的高壁垒与低定价权之间的矛盾，迫使企业必须走“出海”路线。

## 结论

**结论**：**属于“需要深厚医学背景、极强风险承受能力、且处于全球价值链重塑”的高难度赛道**。

百济神州是捕捉 **“全球重磅炸弹”** 溢价的首选；恒瑞医药是 **“传统巨头转型创新”** 的压舱石；信达生物是 **“首次实现盈利”** 的经营标杆。

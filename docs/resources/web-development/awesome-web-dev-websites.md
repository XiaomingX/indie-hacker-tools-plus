# Web 开发与技术探索精选网站 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，获取信息的渠道已经从“通用搜索”转向“垂直社区”。
> - **效率优先**：利用 **daily.dev** 聚合你的技术流，每天只需 5 分钟即可同步全球技术趋势。
> - **基建升级**：对于 MVP 项目，优先考虑 **Fly.io** 或 **Vercel**，将部署成本降至最低。

---

## 🚀 部署、后端与云平台 (Cloud & Backend)

- [ ] [**Fly.io**](https://fly.io/) - **[首选]** 真正的全球分布式部署平台，支持 Docker 容器秒级启动，特别适合需要低延迟响应的应用。
- [ ] [**Appwrite**](https://appwrite.io/) - 开源的 Firebase 替代品，提供全套后端服务（Auth, DB, Storage, Functions），支持自托管。
- [ ] [**Vercel**](https://vercel.com/) - Next.js 的母公司，前端部署与 Edge Functions 的金标准。
- [ ] [**Supabase**](https://supabase.com/) - 基于 PostgreSQL 的全栈后端，提供实时订阅、存储和向量搜索能力。

---

## 🎨 前端、框架与静态生成 (Frontend & SSG)

- [ ] [**Astro.build**](https://astro.build/) - **[2026 推荐]** 构建以内容为中心的极速网站的首选框架，支持“群岛架构”，零 JS 启动。
- [ ] [**Shadcn UI**](https://ui.shadcn.com/) - 彻底改变了 UI 组件库的游戏规则，不再是库，而是你可以完全控制的代码。
- [ ] [**Tailwind CSS**](https://tailwindcss.com/) - 原子化 CSS 的统治者，v4 版本带来了极致的编译速度。

---

## 📰 资讯、社区与学习 (News & Learning)

- [ ] [**Daily.dev**](https://daily.dev/) - 开发者专属的浏览器首页，自动聚合来自 GitHub, StackOverflow, Medium 的高质量技术文章。
- [ ] [**This Week in Bevy**](https://thisweekinbevy.com/) - 关注 Rust 领域最火的 Bevy 游戏引擎动态。
- [ ] [**Mastodon**](https://mastodon.social/) - 去中心化的社交网络，是技术宅和独立开发者交流的高品质社区。
- [ ] [**Indie Hackers**](https://www.indiehackers.com/) - 学习如何从 0 到 1 构建赚钱的副作用项目，充满实战经验。

---

## 🛠️ 实用工具与生产力 (Utility Tools)

- [ ] [**Excalidraw**](https://excalidraw.com/) - 最好的在线白板工具，手绘风架构图的首选。
- [ ] [**Raycast**](https://www.raycast.com/) - macOS 上的万能启动器，丰富的插件生态可替代大部分小工具。
- [ ] [**Dictionary.com**](https://www.dictionary.com/) - 英语词汇学习与精准表达的必备工具。

---

## 💡 选型建议
1. **构建极速博客或文档**：选 **Astro** + **Vercel**。
2. **需要快速搭建全栈应用**：选 **Next.js** + **Supabase**。
3. **追求极致的服务器响应**：选 **Rust/Go** + **Fly.io**。
4. **获取每日技术灵感**：安装 **daily.dev** 插件。

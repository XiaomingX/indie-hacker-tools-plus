# 2026 加密货币基础与主流币种指南 (Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，加密货币已不仅仅是投机资产，而是独立开发者构建“无国界支付”和“去中心化存储”的底层基座。
> - **PoS 主导**：除了比特币，几乎所有主流公链都已转向 Proof of Stake (PoS)，传统显卡挖矿已成为历史。
> - **L2 扩容**：关注 Arbitrum, Optimism 和 Base，它们提供了极低成本且安全可靠的交易环境。
> - **稳定币标配**：对于跨境结算，USDT (TRON/Ethereum) 和 USDC (Solana/Base) 是最稳健的选择。

## 🚀 主流公链与代币 (Mainstream Assets)

- [ ] **比特币 (BTC)** - 算法：SHA-256。数字黄金，价值存储的核心。
- [ ] **以太坊 (ETH)** - 算法：PoS。智能合约的鼻祖，庞大的 L2 生态支撑。
- [ ] **Solana (SOL)** - 算法：PoH + PoS。极致性能，2026 年最活跃的零售与 MEME 生态。
- [ ] **Stablecoins (USDT/USDC)** - 跨境支付与避险的刚需资产。

## 💡 选型建议
1. **构建全新的 Web3 应用**：优先考虑在 **Solana** 或 **Base (L2)** 上开发，以获取最低手续费。
2. **大额资产存储**：务必使用 **硬件钱包** 并物理备份助记词，拒绝任何托管钱包。
3. **日常支付结算**：首选 **USDT (TRC20)** 或 **USDC (Base)**，确认时间快且手续费低。
4. **获取行业前沿洞察**：每天刷一遍 **DexScreener** 的 Trending 榜单。

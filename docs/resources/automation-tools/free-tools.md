# 独立开发者必备：免费/高性价比工具集 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，独立开发者的座右铭应该是 **"吝啬"**。
> - **画图**：不再纠结软件，**Excalidraw** 是目前最流行的手绘风格，适合分享到推特。
> - **视频**：**剪映 (CapCut)** 的 AI 自动剪辑能力已经完全覆盖了大部分短视频营销需求。
> - **IDE**：**Cursor** 虽然收费，但其带来的 10 倍开发效率提升，是比任何“免费”IDE 更大的节约。

---

## 🎨 绘图、脑图与设计 (Design & Diagramming)

- [ ] [**Excalidraw**](https://excalidraw.com/) - **[首选]** 极简手绘风格，支持端到端加密协作，开发者分享思路的最佳选择。
- [ ] [**Draw.io (diagrams.net)**](https://app.diagrams.net/) - 最全能的免费流程图工具，支持直接将图片保存到 GitHub 仓库。
- [ ] [**ProcessOn**](https://www.processon.com/) - 国内协同绘图的首选，模板库非常丰富。
- [ ] [**Mermaid.js**](https://mermaid.js.org/) - 程序员专属，通过 Markdown 语法直接生成流程图和甘特图。
- [ ] [**Figma**](https://www.figma.com/) - 免费档位已足够支持 3 个项目，是现代 UI 设计的事实标准。

---

## 🎬 视频编辑与多媒体 (Video & Media)

- [ ] [**DaVinci Resolve**](https://www.blackmagicdesign.com/products/davinciresolve/) - **[专业级]** 免费版功能强大到离谱，涵盖剪辑、调色、特效。
- [ ] [**剪映 (CapCut)**](https://www.capcut.com/) - AI 驱动，支持自动字幕、音色克隆和一键成片，适合出海短视频获客。
- [ ] [**OBS Studio**](https://obsproject.com/) - 录屏与直播的开源标准，功能极其稳定。
- [ ] [**HandBrake**](https://handbrake.fr/) - 免费开源的视频压制神器，解决视频体积过大的问题。

---

## 💻 在线协作与代码环境 (Code & Collab)

- [ ] [**StackBlitz**](https://stackblitz.com/) - 基于 WebContainers，能在浏览器里跑完整的 Node.js 环境，秒开 Demo。
- [ ] [**CodeSandbox**](https://codesandbox.io/) - 适合前端团队协作，支持与 GitHub 实时同步。
- [ ] [**Code-server**](https://github.com/coder/code-server) - 在你自己的服务器上跑 VS Code，实现 iPad 随时随地办公。
- [ ] [**Exposed (Ngrok 替代品)**](https://github.com/beyondcode/expose) - 将本地服务映射到公网，方便进行 Webhook 调试。

---

## 💡 选型建议
1. **写技术博客配图**：选 **Excalidraw**。
2. **做复杂架构设计**：选 **Draw.io** 或 **Visual Paradigm Online**。
3. **制作 App 宣传片**：选 **剪映** (快) 或 **DaVinci** (稳)。
4. **快速复现前端 Bug**：选 **StackBlitz**。

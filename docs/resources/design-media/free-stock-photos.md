# 免费图库与多媒体素材 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，图库图片容易给人“同质化”的感觉。
> - **个性化微调**：使用图库图片后，通过 AI 工具（如 Canva 或 Midjourney）进行二次处理，增加品牌专属色彩。
> - **SVG 优先**：在网页端，能用图标和插画（SVG）的地方尽量不用大图（JPG），提升加载速度。
> - **CDN 分发**：生产环境务必使用 Vercel Edge Network 或 Cloudflare 镜像这些图片，严禁直接链向图库源站。

---

## 📸 综合摄影图库 (Stock Photos)

- [ ] [**Unsplash**](https://unsplash.com/) - **[行业标准]** 全球质量最高的免费商业摄影图库。
- [ ] [**Pexels**](https://www.pexels.com/) - 同时提供海量高质量图片与免费 4K 视频素材。
- [ ] [**Pixabay**](https://pixabay.com/) - 素材种类最全，包含照片、插画、矢量图、音乐和音效。
- [ ] [**Gratisography**](https://gratisography.com/) - 风格独特、怪诞、充满创意的图库，适合拒绝平庸的品牌。
- [ ] [**Picjumbo**](https://picjumbo.com/) - 个人摄影师建立的高质量图库。
- [ ] [**Reshot**](https://www.reshot.com/) - 独家、非同寻常的免版税插画、图标和照片。
- [ ] [**Rawpixel**](https://www.rawpixel.com/) - 包含大量复古、古典艺术作品的数字化版本。
- [ ] [**ISO Republic**](https://isorepublic.com/) - 优秀的独立摄影图库，适合商业博客。

## 🎨 插画与设计素材 (Illustrations & Design)

- [ ] [**Canva Free Photos**](https://www.canva.com/photos/free/) - 方便直接在 Canva 设计中使用。
- [ ] [**Undraw**](https://undraw.co/illustrations) - 开发者最爱的可定制色调的开源 SVG 插画库。
- [ ] [**DrawKit**](https://www.drawkit.com/) - 精美的免费与付费插画包。
- [ ] [**Humaans**](https://www.humaans.com/) - 自由组合的人物插画系统。

## 💡 选型建议
1. **构建专业 B 端官网**：首选 **Unsplash** 高冷风格照片。
2. **需要展示人物互动的 Web 界面**：首选 **Undraw** 插画。
3. **内容运营与短视频**：选 **Pexels** 的 4K 视频素材。
4. **追求复古或艺术格调**：选 **Rawpixel**。

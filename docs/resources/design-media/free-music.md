# 免费音乐与音效资源 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，音轨质量直接决定了产品的“廉价感”还是“高级感”。
> - **版权意识**：优先选择 **CC0 (Public Domain)** 或 **CC BY (需署名)** 授权，避免因音乐下架导致产品违规。
> - **AI 辅助**：如果找不到合适的曲风，考虑使用 [Suno](https://suno.com/) 或 [Udio](https://www.udio.com/) 生成独家配乐。
> - **格式选择**：Web 端推荐使用 **WebP (Audio)** 或 **Ogg**，兼顾高保真与小体积。

---

## 🎵 综合音乐库 (Music Libraries)

- [ ] [**CCMixter**](http://dig.ccmixter.org/) - 大量社区驱动的免费音乐，适合作为背景音。
- [ ] [**Bensound**](https://www.bensound.com/) - 质量极高的免版税音乐，广泛用于视频和展示。
- [ ] [**Free Music Archive (FMA)**](http://freemusicarchive.org/) - 具有极高艺术性的开源音乐库。
- [ ] [**Jamendo Music**](https://www.jamendo.com/) - 支持按心情和风格搜索，包含大量独立艺人作品。
- [ ] [**Incompetech**](https://incompetech.com/music/) - 经典配乐大师 Kevin MacLeod 的作品集，适合影视和游戏。
- [ ] [**WOWA**](https://www.wowa.me/) - 极简主义的高质量免费音乐。
- [ ] [**Sound Image**](http://soundimage.org/) - Eric Matyas 创作的数千首免费配乐。

---

## 🔊 音效与循环 (SFX & Loops)

- [ ] [**Freesound**](https://freesound.org/) - 全球最大的 CC 授权音效库，适合游戏开发与 UI 反馈音。
- [ ] [**Sound Bible**](http://soundbible.com/) - 简单好用的免费音效资源站。
- [ ] [**Free Loops**](http://free-loops.com/) - 适合音乐制作的循环采样。
- [ ] [**OpenGameArt (Audio)**](https://opengameart.org/art-search-advanced?field_art_type_tid%5B%5D=12) - 专为游戏设计的免费音效与背景音乐。

---

## 🏛️ 公有领域与存档 (Public Domain & Archive)

- [ ] [**Internet Archive (Netlabels)**](https://archive.org/details/netlabels) - 存档了数百万首公有领域音乐。
- [ ] [**Public Domain Info**](https://www.pdinfo.com/royalty-free-music.php) - 帮助确认作品是否已进入公有领域。
- [ ] [**Musopen**](https://musopen.org/music/) - 免费的古典音乐资源，非常适合高级感 PPT 或展示。

---

## 💡 选型建议
1. **构建网页 UI 反馈音**：首选 **Freesound (CC0)**。
2. **需要高级感的宣传片**：选 **Bensound** 或 **Udio (AI 生成)**。
3. **长期维护的项目**：务必在项目文档中记录下每首音乐的**具体授权协议 (License)**。
4. **游戏背景音乐**：选 **Incompetech** 或 **OpenGameArt**。

# 高质量图标库与资源精选 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，图标不仅是装饰，更是 **"微交互"** 的核心。
> - **格式选择**：**SVG** 是唯一标准。利用 **Lucide** 或 **Heroicons** 的 React/Vue 组件，可以轻松通过代码控制图标的颜色、粗细和动画。
> - **按需加载**：不再建议引入全量字体包。使用 **Iconify** 可以实现数百万图标的按需加载，极大减少页面首屏体积。

---

## 🏗️ 开发者首选库 (Developer's Choice)

- [ ] [**Lucide**](https://lucide.dev/) - **[2026 推荐]** 基于 Feather 进化而来，图标数量翻倍（1400+），完美适配所有主流前端框架，支持高度自定义。
- [ ] [**Heroicons**](https://heroicons.com/) - Tailwind CSS 团队出品，提供 Outline 和 Solid 两种风格，与现代 UI 界面极致贴合。
- [ ] [**Iconify**](https://iconify.design/) - **[万能框架]** 统一了 100 多个图标库（含 FontAwesome, Material 等）的访问接口，支持按需加载。
- [ ] [**Font Awesome 6**](https://fontawesome.com/) - 行业常青树，生态最全，支持 Webfont 和 SVG，适合需要极大图标覆盖面的项目。

---

## 🎨 设计与创意资源 (Design & Creative)

- [ ] [**The Noun Project**](https://thenounproject.com/) - 全球设计师贡献，拥有数百万个极具设计感的图标，适合寻找独特视觉风格。
- [ ] [**Flaticon**](https://www.flaticon.com/) - Freepik 旗下，提供大量彩色、扁平化图标包，适合制作幻灯片、海报和小程序。
- [ ] [**Phosphor Icons**](https://phosphoricons.com/) - 提供 6 种粗细风格，非常适合需要精细调节视觉重量的专业级 UI。
- [ ] [**Remix Icon**](https://remixicon.com/) - 每一个图标都经过精心打磨，风格中性且统一，开源免费商用。

---

## 🌐 品牌、社交与特定场景 (Brands & Specials)

- [ ] [**Simple Icons**](https://simpleicons.org/) - **[必备]** 收集了全球 3000+ 知名品牌的官方图标（含颜色规范），展示社交链接的首选。
- [ ] [**Devicon**](https://devicon.dev/) - 专门收集编程语言、开发工具和框架的图标。
- [ ] [**Flagpack**](https://flagpack.xyz/) - 260+ 国家/地区旗帜图标，支持多种风格（圆形、方形、扁平），适配国际化项目。

---

## 🛠️ 图标效率工具 (Efficiency Tools)

- [ ] [**Iconmonstr**](https://iconmonstr.com/) - 极简工具，支持在下载前在线调整图标的填充、线条和背景。
- [ ] [**SVG Repo**](https://www.svgrepo.com/) - 强大的 SVG 搜索引擎，支持按授权协议（如 CC0）进行精准筛选。
- [ ] [**IcoMoon**](https://icomoon.io/) - 将你选中的图标打包成专属的字体或 SVG Sprite，极致压缩体积。

---

## 💡 选型建议
1. **构建现代 Web App**：选 **Lucide** 或 **Heroicons**。
2. **需要大量品牌图标**：选 **Simple Icons**。
3. **追求极致加载速度**：使用 **Iconify** 实现按需按量加载。
4. **制作精美 PPT 或海报**：到 **Flaticon** 寻找成套的彩色图标包。

# 游戏设计与数值策划实战资源 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，独立游戏成功的关键在于**“心流设计”**与**“数值平衡”**。
> - **数值驱动**：不要拍脑袋定数值，利用 Excel 或 Python 模拟上万次战斗，确保收益曲线平滑。
> - **玩法创新**：在经典玩法（如消除、MOBA、RPG）上增加一个“AI 变量”或“独特机制”往往能产生奇效。
> - **社交传播**：参考《羊了个羊》，极端的难度设计有时是最好的营销工具。

---

## 🎮 经典案例深度解析 (Case Studies)

- [ ] [**《传奇》：数值与即时战斗的基石**](#) - 学习如何通过极简属性设定（战法道）构建深度的装备追求与 PK 生态。
- [ ] [**《仙剑奇侠传》：剧情驱动与五灵数值**](#) - 探究剧情节奏与回合制数值平衡（如速度、抗性）的结合。
- [ ] [**《羊了个羊》：极端数值与病毒式营销**](#) - 分析“0.1% 通关率”如何利用幸存者偏差制造社交话题。
- [ ] [**《王者荣耀》：MOBA 英雄属性与平衡算法**](#) - 掌握英雄定位（战坦刺）与动态数值调整策略。
- [ ] [**《文明 6》：复杂资源产出与时代推进机制**](#) - 学习 4X 游戏中的资源转换（粮食/生产力/科技）与外交博弈。
- [ ] [**《信长之野望》：宏观策略与内政数值设计**](#) - 探索大名经营、武将成长曲线与战场士气变化。
- [ ] [**《狼人杀》：社交博弈与阵营胜率平衡**](#) - 学习非对称竞技中的发言规则与逻辑漏洞设计。
- [ ] [**《原神》：元素反应系统与经济循环**](#) - 分析开放世界探索奖励与多角色元素交互的数值架构。
- [ ] [**《大富翁》：随机性与策略的数值博弈**](#) - 学习如何平衡运气（掷骰子）与策略（地产投资/卡牌）。

## 🛠️ 策划必备工具 (Essential Tools)

- [ ] [**Machinations.io**](https://machinations.io/) - **[必选]** 可视化游戏经济与数值平衡模拟工具。
- [ ] [**GameDesign.pro**](https://gamedesign.pro/) - AI 辅助生成关卡概念与剧情大纲。
- [ ] [**Unity Game Simulation**](https://unity.com/products/game-simulation) - 利用云端模拟上百万次游戏运行，自动发现平衡性漏洞。

## 💡 选型建议
1. **独立开发者起步**：优先模仿《羊了个羊》的简单机制，辅以极高的社交互动性。
2. **构建复杂策略游戏**：必须使用 **Excel/Python** 进行全量数值跑测。
3. **注重剧情表现**：参考《原神》，将资源获取（奖励）深度嵌套在探索与剧情任务中。
4. **强社交竞技**：核心关注“胜率平衡”与“防作弊数值验证”。

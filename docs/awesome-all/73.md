# awesome-solana（Solana 资源精选）

## 用户核心资源

### 钱包
- **[Phantom Wallet](https://phantom.app/)**：主流非托管加密钱包，支持多链操作，浏览器扩展和移动端均可用，界面友好，支持 NFT 管理及代币交换。

### 跨链桥
- **[Allbridge](https://allbridge.io/)**：支持多链资产跨链转移，操作简便，费用合理。
- **[Wormhole](https://wormholebridge.com/#/transfer)**：高安全性的跨链通信协议，Solana 与多主流链无缝互通。

### 质押与流动性质押
- **[Marinade Finance](https://marinade.finance/)**：主流流动性质押协议，质押 SOL 获得 mSOL 代币，可用于 DeFi 进一步利用，APY 波动但一般在 5%-7% 之间。
- **[Lido](https://solana.lido.fi/)**：流动性质押平台，质押 SOL 获取 stSOL，同样支持 DeFi 生态使用，APY 稳定。

### 去中心化交易所与收益聚合
- **[Orca](https://www.orca.so/)**：操作简洁的AMM去中心化交易所，支持流动池及收益农场。
- **[Saber](https://saber.so/)**：跨链稳定币兑换平台，提供高效低滑点的稳定资产交换。
- **[Tulip Protocol](https://tulip.garden/leverage)**：集中管理多种收益农场，支持杠杆和收益优化。

### NFT生态
- **[Metaplex](https://www.metaplex.com/)**：官方 NFT 铸造和销售工具，支持定制化和多样化功能。
- **[Solanart](https://solanart.io/)**：早期知名 Solana NFT 市场，大量优质作品集中交易。

### 区块链浏览器及数据工具
- **[SolScan](https://solscan.io/)**：功能丰富的区块链浏览器，支持交易、代币、NFT 数据查看。
- **[Solana Explorer](https://explorer.solana.com/)**：官方链上浏览器，实时信息准确。

## 开发者资源

### 官方文档与开发框架
- **[Solana 官方文档](https://docs.solana.com/)**：最权威的 Solana 技术文档，覆盖从入门到进阶内容。
- **[Anchor 框架](https://project-serum.github.io/anchor/getting-started/introduction.html)**：Solana 智能合约 Rust 开发的标准框架，大幅简化合约编写和与前端交互。
- **[Solana Cookbook](https://solanacookbook.com/)**：包含大量实用示例和最佳实践，适合快速上手。

### 开发教程
- **[Solana 开发实战课程](https://soldev.app/course)**：全面的 Web3 开发课程，适合零基础开发者。
- **[Hello World 示例](https://github.com/solana-labs/example-helloworld)**：官方简单智能合约示例，快速体验 Solana 合约开发。
- **[全栈 Solana 开发指南](https://dev.to/dabit3/the-complete-guide-to-full-stack-solana-development-with-react-anchor-rust-and-phantom-3291)**：综合 React、Anchor 和 Phantom 钱包的完整教程。

### 常用开发工具
- **[SPL Token UI](https://spl-token-ui.com/)**：管理 SPL 代币的便捷前端界面。
- **[Solana Playground](https://beta.solpg.io/)**：在线编写、调试智能合约的沙箱环境，方便快速迭代开发。

### 视频教程
- **[Rust 与 Solana 合约开发入门](https://www.youtube.com/watch?v=gA7hFdq2h9Q)**：详细讲解合约基本结构和开发流程。
- **[Anchor 框架实操教程](https://www.youtube.com/watch?v=FmdPAwsqJC4)**：深度介绍 Anchor 和高效智能合约开发。

## 网络监控与安全

### 状态监控
- **[Solana Status](https://status.solana.com/)**：官方网络和服务状态实时监控。
- **[Solana Beach](https://solanabeach.io/)**：链上数据可视化和验证节点状态分析。

### 安全资源
- Solana 智能合约可能存在的常见漏洞和防范指南（建议关注社区安全白皮书与审计报告）。
- 推荐使用 **[Soteria](https://soteria.dev/)** 等自动漏洞检测工具，及时发现合约风险。

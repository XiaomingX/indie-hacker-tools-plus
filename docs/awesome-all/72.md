# awesome-flutter：2024 最新 Flutter 学习与开发资源大全


## 1. 入门必看（零基础快速上手）
入门阶段优先选择**官方权威资源**和**体系化教程**，避免走弯路。

- [Flutter 官方入门 Codelab](https://docs.flutter.dev/get-started/codelab)  
  核心价值：手把手教你搭建第一个 Flutter 应用，覆盖 Widget 基础、布局逻辑、页面导航，适配 Flutter 3.x 最新版本，绝对是新手起点。
- [为何选择 Flutter 2024](https://flutter.dev/showcase)  
  核心价值：官方展示字节跳动、阿里巴巴等企业的生产级案例，清晰说明 Flutter 跨移动/桌面/Web/嵌入式的优势，帮你明确技术选型理由。
- [Flutter Web 快速入门](https://docs.flutter.dev/development/platform-integration/web)  
  核心价值：官方教程，从 Web 环境配置到性能优化（如图片懒加载），解决“Flutter 写网页卡不卡”“如何适配浏览器”等实际问题。


## 2. 优质学习平台（系统提升）
这些博客/网站持续输出高质量内容，覆盖从基础到进阶的全场景。

| 平台名称                | 链接                          | 核心价值                                                                 |
|-------------------------|-------------------------------|--------------------------------------------------------------------------|
| Flutter 官方博客        | https://medium.com/flutter     | 第一时间获取 Flutter 新特性（如 Impeller 引擎）、团队最佳实践，权威无水分。 |
| Didier Boelens' Blog    | https://didierboelens.com     | 聚焦实际开发问题（如手势冲突、权限适配），提供可直接复用的代码方案，支持英法双语。 |
| Reso Coder              | https://resocoder.com/category/flutter/ | 体系化教程，侧重“Clean Architecture + 状态管理”，适合想写“工程化代码”的开发者。 |
| FilledStacks            | https://www.filledstacks.com/ | 从 UI 布局到后端集成（如 Firebase）的实战指南，附带完整项目源码。          |


## 3. 手把手实用教程（边练边学）
通过具体项目掌握技术点，比纯理论更高效。

- [Flutter 状态管理入门](https://codelabs.developers.google.com/codelabs/flutter-state-management#0)  
  核心价值：对比 `setState`、`Provider`、`Riverpod` 的用法与场景，新手能快速理解“为什么需要状态管理”。
- [Firebase 完整集成实战](https://firebase.google.com/codelabs/flutter-firebase-analytics#0)  
  核心价值：覆盖“手机号登录→云数据库存数据→远程配置改样式”全流程，适配最新 FlutterFire SDK。
- [10 个 Flutter 实战小项目](https://github.com/ResoCoder/flutter-tutorials)  
  核心价值：从“待办清单”到“天气 App”，每个项目对应一个技术点（如动画、网络请求、本地存储），代码可直接运行。


## 4. 入门核心资源（工具与指南）
整理了新手高频用到的“字典级”资源，随用随查。

- [Flutter 官方文档（中文）](https://docs.flutter.dev/guides)  
  核心价值：最权威的 Widget 说明、API 文档、平台适配指南，新手遇到问题先查这里。
- [2024 Flutter 学习路线图](https://github.com/fluttertools/flutter_roadmap_2024)  
  核心价值：按“基础→进阶→专项”划分阶段，标注每个阶段的重点（如基础学 Widget，进阶学 Riverpod）。
- [Flutter 布局小抄（2024 版）](https://medium.com/flutter-community/flutter-layout-cheat-sheet-2024-5363348d037e)  
  核心价值：用“代码+效果图”详解 `Row`/`Column`/`Stack` 等 10+ 布局组件，解决“怎么排 UI 才不歪”的问题。
- [Flutter Widget 速查工具](https://flutter-widget.live/)  
  核心价值：可视化展示所有官方 Widget，点击可看源码和效果，找组件时不用翻文档。


## 5. 进阶技术指南（工程化与架构）
脱离“写 Demo”阶段，掌握企业级开发能力。

- [Flutter 状态管理全家桶](https://suragch.medium.com/flutter-state-management-2024-4c71a2f2f0c1)  
  核心价值：对比 `Riverpod`（推荐）、`BLoC`、`GetX` 的优缺点，附中型项目实战代码。
- [Flutter Clean Architecture 实战](https://resocoder.com/2020/09/09/flutter-clean-architecture-course-1-domain-layer/)  
  核心价值：教你如何拆分“数据层→领域层→表现层”，让代码可测试、易维护。
- [Flutter 性能优化指南](https://docs.flutter.dev/perf/rendering-performance)  
  核心价值：官方讲解“如何解决 UI 卡顿”，覆盖 `const` 优化、列表懒加载、图片缓存等实战技巧。


## 6. 高阶深度解析（底层与引擎）
适合想深入 Flutter 原理的进阶开发者。

- [Flutter 渲染管线解析](https://www.youtube.com/watch?v=996ZgFRENMs)（2023 年更新）  
  核心价值：讲解“Widget→Element→RenderObject”三棵树的关系，理解 Flutter 为什么“UI 渲染快”。
- [Impeller 渲染引擎详解](https://flutter.dev/docs/development/tools/impeller)  
  核心价值：Flutter 3.x 主推的新引擎，解决老引擎（Skia）的卡顿问题，本文解析其原理与启用方式。
- [Flutter 自定义 RenderObject 实战](https://medium.com/flutter-community/flutter-custom-renderobject-101-7055051ad99a)  
  核心价值：教你手写自定义组件（如复杂进度条），深入理解 Flutter UI 渲染本质。

## 10. 多媒体与设备交互（功能扩展）
处理声音、摄像头、位置等设备能力。

### 声音相关
- [speech_to_text](https://pub.dev/packages/speech_to_text)  
  核心价值：语音转文字，支持 iOS/Android/Web，可设置语言（中文、英文）、识别精度。
- [just_audio](https://pub.dev/packages/just_audio)  
  核心价值：功能全面的音频播放器，支持后台播放、进度条控制、音质调节。

### 设备交互
- [camera](https://pub.dev/packages/camera)  
  核心价值：调用手机摄像头，支持拍照、录像、切换前后置镜头，适配最新系统版本。
- [geolocator](https://pub.dev/packages/geolocator)  
  核心价值：获取设备位置（经纬度）、监听位置变化，支持 iOS/Android/Web，解决权限适配问题。


## 11. 数据存储方案（本地与云端）
根据项目需求选合适的存储方式。

| 存储类型       | 工具/库                          | 核心价值                                                                 |
|----------------|----------------------------------|--------------------------------------------------------------------------|
| 云端存储       | Firebase Storage                 | 背靠 Google 云，适合存图片、视频等大文件，集成简单，支持权限控制。       |
| 本地安全存储   | flutter_secure_storage           | 存敏感数据（如 Token、密码），iOS 用 Keychain，Android 用 Keystore，安全可靠。 |
| 本地数据库     | Hive                             | 轻量级 NoSQL 数据库，无需原生代码，读写速度快，适合存列表数据（如历史记录）。 |
| 复杂本地查询   | Drift（原 Moor）                 | SQLite 的 Dart 封装，支持复杂 SQL 查询、联表操作，适合数据量大的 App。    |

## 16. 状态管理框架对比（选对工具效率翻倍）
不同框架适合不同场景，避免盲目跟风。

| 框架       | 核心优势                                  | 适用场景                                  | 学习难度 |
|------------|-------------------------------------------|-------------------------------------------|----------|
| Riverpod   | 无上下文依赖、支持监听、测试友好          | 中大型项目、团队协作                      | 中等     |
| BLoC       | 逻辑与 UI 分离、状态变化可追溯            | 复杂业务逻辑（如电商下单流程）            | 中等     |
| Provider   | 轻量、易用、官方推荐入门                  | 小型项目、简单状态管理（如主题切换）      | 简单     |
| GetX       | 集成路由、状态管理、依赖注入，功能全面    | 快速开发原型、中小型项目                  | 简单     |

## 19. 开发效率工具（提效必备）
这些工具能解决“打包慢、调试烦”等问题。

- [Flutter DevTools](https://docs.flutter.dev/tools/devtools)  
  核心价值：官方调试工具，可查看 Widget 树、监控内存占用、调试网络请求，必装。
- [flutter_native_splash](https://pub.dev/packages/flutter_native_splash)  
  核心价值：生成启动页（Splash Screen），支持渐变背景、品牌 Logo，适配 iOS/Android。
- [dart_code_metrics](https://pub.dev/packages/dart_code_metrics)  
  核心价值：检查代码规范（如未用的变量）、分析性能问题，帮你写出高质量 Dart 代码。

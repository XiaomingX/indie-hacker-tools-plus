# learning-seo-starter
## 常用于SEO的材料
### awesome类（内容特色汇总）
 - xxx领域的X个最常用的YYY.
 - 图文、视频加分.

### 竞品比较类（A vs B对比）
 - A的好处，B的好处，两者的劣势，适用场景.

### 定义：什么是X，常用场景，优势与劣势（类wiki）
 - 新概念定义，新趋势定义.

### 羊毛党：如何获得便宜、免费的YYY（包装概念）
 - 方法、步骤、价值.

### 标题党
 - 带感叹号，震惊体.

# Awesome AI Music Generation (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年的音乐生成已从“实验性”转向“生产级”。
> - **商业配乐**：使用 **Suno** 或 **Udio** 的 API 为你的产品视频生成高质感配乐，成本几乎为零。
> - **个性化体验**：利用 **Stable Audio 2.0** 生成特定风格的白噪音或冥想音乐。

---

## 🎵 核心创作工具 (Top Products)

- [ ] [**Suno**](https://suno.com/) - 2026 年音乐生成领域的现象级产品，支持长达 4 分钟的高保真全曲生成。
- [ ] [**Udio**](https://www.udio.com/) - 独立开发者心目中的音质之王，更具艺术感的生成效果。
- [ ] [**Stable Audio 2.0**](https://stableaudio.com/) - 能够根据文本生成连贯且高质感的音频片段，适合短视频配乐。
- [ ] [**Soundraw**](https://soundraw.io/) - 专为内容创作者设计的免版权音乐平台，可自定义结构。
- [ ] [**AIVA**](https://www.aiva.ai/) - 老牌 AI 作曲家，支持导出 MIDI，适合进阶二次创作。
- [ ] [**ElevenLabs Audio**](https://elevenlabs.io/) - 2026 年新推出的音效生成功能，是独立游戏开发的音效神器。

---

## 🛠️ 开源项目与库 (Open Source)

- [ ] [**Audiocraft (Meta)**](https://github.com/facebookresearch/audiocraft) - 包含 MusicGen, AudioGen 等顶级开源模型。
- [ ] [**Magenta (Google)**](https://github.com/magenta/magenta) - 探索机器学习在艺术创作中可能性的老牌研究项目。
- [ ] [**Muzic (Microsoft)**](https://github.com/microsoft/muzic) - 包含多种音乐理解与生成的深度学习模型。
- [ ] [**Riffusion**](https://github.com/riffusion/riffusion) - 将音频表示为图像（声谱图）并利用 Stable Diffusion 生成。
- [ ] [**VampNET**](https://github.com/hugofloresgarcia/vampnet) - 利用双向 Transformer 进行高保真音乐合成。

## 💡 选型建议
1. **追求极致画质与流行感**：首选 **Suno**。
2. **需要 API 自动化集成**：关注 **Suno API** 或 **Stable Audio API**。
3. **独立游戏开发者**：建议常备 **ElevenLabs Audio** 生成实时音效。

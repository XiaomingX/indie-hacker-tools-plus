# 聊天机器人与智能体工具包 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年的“聊天机器人”已进化为“智能体 (Agent)”。
> - **低代码平台**：使用 **Dify** 或 **Coze** 快速搭建具备 RAG 和工作流能力的 Agent。
> - **实时交互**：利用 **OpenAI Realtime API** 或 **Hume AI** 实现极低延迟的语音交互。

---

## 🏗️ 智能体开发与编排 (Agent Orchestration)

- [ ] [**Dify**](https://dify.ai/) - 独立开发者构建 AI 应用的首选，支持可视化编排 RAG、工作流和多模型切换。
- [ ] [**Coze (扣子)**](https://www.coze.cn/) - 字节跳动出品，拥有极强的插件生态和一键发布到多平台的能力。
- [ ] [**Auto-GPT**](https://github.com/Significant-Gravitas/AutoGPT) - 自动化 Agent 的先驱，2026 版本在任务规划与自我修正方面表现稳健。
- [ ] [**AgentGPT**](https://agentgpt.reworkd.ai/) - 浏览器原生的 Agent 构建平台，支持 MCP 协议连接企业工具。
- [ ] [**LangFlow**](https://www.langflow.org/) - 基于节点的 LangChain 可视化界面，适合复杂逻辑的设计。

---

## 💻 数据库与数据分析 (Data & DB)

- [ ] [**Chat2DB**](https://chat2db.ai/) - 智能 SQL 客户端，支持自然语言转 SQL 及数据库联邦查询。
- [ ] [**PandasAI**](https://github.com/gventuri/pandas-ai) - 赋予 Pandas 对话能力，一键生成报表与可视化图表。
- [ ] [**AskYourDatabase**](https://www.askyourdatabase.com/) - 直接与数据库聊天，支持复杂的多表关联分析。

---

## 🎥 音视频与多模态 (Media & Multimodal)

- [ ] [**BibiGPT**](https://bibigpt.co/) - 视频内容分析领军者，支持秒级生成 B 站、YouTube 摘要。
- [ ] [**AudioGPT**](https://github.com/AudioGPT/AudioGPT) - 集成多种语音处理模型，支持实时语音合成与翻译。
- [ ] [**Ask-Anything (Video)**](https://github.com/Ask-Anything/Ask-Anything) - 强大的视频理解框架，支持实时画面分析与问答。

---

## 🛡️ 安全、测试与文档 (Security & Docs)

- [ ] [**PentestGPT**](https://github.com/GreyDaze/pentestgpt) - 基于 LLM 的自动化渗透测试辅助工具。
- [ ] [**Vault-AI**](https://github.com/vault-ai/vault-ai) - 专注于隐私保护的本地知识库方案，适配金融/医疗场景。
- [ ] [**BibiGPT-Desktop**](https://github.com/BibiGPT/BibiGPT) - 桌面端音视频摘要工具，支持本地文件处理。

---

## 💡 选型建议
1. **快速原型开发**：首选 **Coze** 或 **Dify**。
2. **需要深度自定义逻辑**：推荐 **LangFlow** 或 **Auto-GPT**。
3. **针对特定数据（如 SQL/Excel）**：使用 **Chat2DB** 或 **PandasAI**。
4. **视频内容站长**：**BibiGPT** 是必备的引流与内容自动化工具。

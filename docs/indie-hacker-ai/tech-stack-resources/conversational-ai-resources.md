# 对话式 AI (Conversational AI) 核心资源 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年的对话式 AI 已从“文字聊天”全面转向“原生语音交互”。
> - **实时语音**：利用 **Vapi** 或 **Retell AI** 可以在几分钟内构建一个具备人类级语气的语音 Agent。
> - **情感分析**：通过 **Hume AI** 识别用户的细微情绪波动，提供更具同理心的反馈。

---

## 🎙️ 语音智能体平台 (Voice Agent Platforms)

- [ ] [**Vapi**](https://vapi.ai/) - 2026 年最热门的语音 AI 平台，超低延迟，支持一键对接电话、网页和 App。
- [ ] [**Retell AI**](https://www.retellai.com/) - 专注于高保真语音交互，支持复杂的对话逻辑编排。
- [ ] [**Hume AI**](https://www.hume.ai/) - 情感计算（EVI）领军者，赋予对话机器人真实的情感理解能力。
- [ ] [**Bland AI**](https://www.bland.ai/) - 强大的电话营销与客服 Agent 平台，支持大规模并发拨打。
- [ ] [**ElevenLabs Conversational AI**](https://elevenlabs.io/) - 结合了顶级 TTS 与实时交互能力的完整方案。

---

## 🛠️ 核心开发框架与 SDK (Dev Tools)

- [ ] [**LangChain**](https://www.langchain.com/) - 依然是构建复杂对话逻辑（Memory, Tools, RAG）的基础框架。
- [ ] [**Livekit**](https://livekit.io/) - 提供实时音视频流基础设施，其 **Agents SDK** 是构建原生语音应用的核心。
- [ ] [**Pipecat**](https://github.com/pipecat-ai/pipecat) - 开源的实时 AI 对话框架，支持多模型、多 TTS 灵活组合。
- [ ] [**Daily.co**](https://www.daily.co/) - 全球领先的 WebRTC 平台，针对 AI 对话进行了深度优化。

---

## 📦 部署渠道与平台 (Deployment)

- [ ] [**WhatsApp Business API**](https://developers.facebook.com/docs/whatsapp/) - 全球私域流量变现的首选渠道。
- [ ] [**Telegram Bot API**](https://core.telegram.org/bots) - 开发成本最低、功能最灵活的机器人部署渠道。
- [ ] [**微信开放平台**](https://open.weixin.qq.com/) - 国内场景必备，支持公众号、小程序和企业微信。
- [ ] [**Discord**](https://discord.com/developers/docs/intro) - 游戏与技术社区互动的核心阵地。

---

## 📚 学习与设计指南 (Design & Learning)

- [x] **Designing Bots**: 对话设计入门必读，聚焦交互逻辑而非技术实现。
- [x] **Conversation Design Institute**: 获取全球认可的对话设计师认证。
- [x] **Google Dialogflow CX Guide**: 官方免费的高级对话设计指南。

---

## 💡 选型建议
1. **构建电话客服机器人**：首选 **Vapi** 或 **Bland AI**。
2. **需要极致的情感化沟通**：推荐集成 **Hume AI**。
3. **开发原生移动端语音 App**：使用 **Livekit Agents** + **Gemini 1.5 Flash**。
4. **低预算个人项目**：**Telegram Bot** + **Dify** 是最快路径。

## 📦 2026 主流模型选型 (Models)

- [ ] [**GPT-5 (OpenAI)**](https://openai.com/gpt-5) - 推理能力的巅峰，支持全模态交互。
- [ ] [**Claude 4 (Anthropic)**](https://claude.ai/) - 依然是编程与长逻辑处理的最佳选择。
- [ ] [**Llama 4 (Meta)**](https://ai.meta.com/llama/) - 开源界无可争议的领导者，提供极强的推理性能。
- [ ] [**DeepSeek-V4**](https://www.deepseek.com/) - 全球性价比最高的 MoE 模型，代码能力极佳。
- [ ] [**Qwen3 (Alibaba)**](https://modelscope.cn/) - 中文语境下理解力与指令遵循最强的模型。
- [ ] [**Gemma 3 (Google)**](https://blog.google/technology/developers/google-gemma-3/) - Google 出品的极致轻量级开源模型，适合端侧部署。

## 💡 选型建议
1. **构建代码助手**：首选 **Claude 4** 或 **Cursor** 原生集成。
2. **需要极致低成本推理**：使用 **Groq** 托管的 **Llama 4**。
3. **中文客服/知识库**：推荐 **Qwen3** 或 **DeepSeek-V4**。
4. **本地隐私离线场景**：使用 **Ollama** 运行 **Gemma 3** 或 **Phi-4**。

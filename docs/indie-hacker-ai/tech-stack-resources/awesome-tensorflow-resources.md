# TensorFlow 核心资源 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年的 TensorFlow 生态核心在于 **Keras 3.0**。
> - **跨框架能力**：利用 Keras 3.0 编写的代码可以无缝在 TensorFlow、PyTorch 和 JAX 之间切换，极大提升了灵活性。
> - **生产级部署**：在需要极致稳定性和大规模分布式训练的工业级场景中，TensorFlow 依然是首选。

---

## 🏗️ 入门与核心教程 (Learning)

- [ ] [**TensorFlow 官方教程**](https://www.tensorflow.org/tutorials) - 权威的起步指南，Colab 代码一键运行。
- [ ] [**Coursera: TensorFlow 专项课程**](https://www.coursera.org/specializations/tensorflow-in-practice) - 由 Google 工程师亲授，系统性学习 TF 实战。
- [ ] [**Effective TensorFlow 2**](https://github.com/vahidk/EffectiveTensorflow2) - 聚焦高效建模与性能优化的最佳实践。
- [ ] [**TF 2.x 项目模板**](https://github.com/Mrgemy95/Tensorflow-Project-Template) - 规范化的代码脚手架，适合快速启动新项目。

---

## 🛠️ 高效开发工具与库 (Ecosystem)

- [ ] [**Keras 3.0**](https://keras.io/) - 2026 年的核心推荐，支持多后端、极简 API。
- [ ] [**TensorFlow Datasets (TFDS)**](https://www.tensorflow.org/datasets) - 一键获取并预处理 500+ 标准数据集。
- [ ] [**TensorBoard**](https://www.tensorflow.org/tensorboard) - 模型训练可视化的行业标准工具。
- [ ] [**TensorFlow Probability (TFP)**](https://www.tensorflow.org/probability) - 专注于贝叶斯推理与概率建模的强悍库。

---

## 🚀 模型部署与加速 (Production)

- [ ] [**TensorFlow Serving**](https://www.tensorflow.org/tfx/guide/serving) - 专为生产环境设计的高性能模型托管系统。
- [ ] [**TensorFlow Lite (TFLite)**](https://www.tensorflow.org/lite) - 移动端与边缘计算（IoT）部署的黄金标准。
- [ ] [**TensorFlow.js**](https://www.tensorflow.org/js) - 在浏览器或 Node.js 中直接运行与训练模型。
- [ ] [**TFX (TensorFlow Extended)**](https://www.tensorflow.org/tfx) - 端到端的机器学习流水线（MLOps）平台。

## 💡 选型建议
1. **构建跨平台 Web AI 应用**：选 **TensorFlow.js**。
2. **需要极致稳定的工业级部署**：选 **TensorFlow Serving**。
3. **编写框架无关的深度学习代码**：使用 **Keras 3.0**。
4. **开发移动端实时视觉应用**：选 **TensorFlow Lite**。

# 阿里云 (Alibaba Cloud) 创业者计划 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，阿里云是**亚太出海与大模型落地**的首选。
> - **通义千问 (Qwen)**：Qwen 2.5/3 系列模型在中文语境与代码能力上极具优势，通过阿里云可以获得海量免费调用额度。
> - **跨境加速**：利用阿里云的全球加速 (GA) 解决跨境访问延迟问题。

## 💰 财务支持与代金券 (Financials)

- [ ] [**阿里云创业者计划**](https://www.aliyun.com/startup) - 申请 **$3,000 至 $120,000** 的云服务代金券。
- [ ] [**新用户试用中心**](https://free.aliyun.com/) - 包含 100+ 款云产品免费试用，涵盖核心计算、存储、数据库。
- [ ] [**算力券/模型券**](https://www.aliyun.com/solution/ai) - 针对 AI 初创公司，申请用于 API 调用与 GPU 租赁的专项补贴。

## 🏗️ AI 与大模型专项 (AI & LLM)

- [ ] [**ModelScope (魔搭社区)**](https://modelscope.cn/) - 国内最大的开源模型社区，提供免费的 GPU 推理算力。
- [ ] [**PAI (人工智能平台)**](https://www.aliyun.com/product/bigdata/learn) - 全托管的 AI 开发平台，支持一键部署 Qwen 系列模型。
- [ ] [**百炼 (Model Studio)**](https://www.aliyun.com/product/bailian) - 针对企业级大模型应用的一站式开发平台。

## 🌐 出海与合规支持 (Global & Compliance)

- [ ] [**全球加速 (GA)**](https://www.aliyun.com/product/ga) - 优化全球用户访问延迟，出海必备。

## 💡 申报建议
1. **优先通过孵化器申请**：通过合作伙伴渠道申请往往能获得更高的初始额度。
2. **聚焦 AI 应用层**：阿里云目前对基于 Qwen 模型开发的垂直领域 Agent 有极高的补贴意愿。
3. **关注“模型周”活动**：阿里云经常举办模型体验活动，赠送大量 API 额度。

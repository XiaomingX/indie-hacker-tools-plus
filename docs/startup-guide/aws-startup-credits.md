# AWS Activate 初创扶持计划 (2026 Checklist)

> [!TIP]
> **Indie Hacker Insight**: 2026 年，AWS 是**全球化与企业级 AI** 的金标准。
> - **Amazon Bedrock**：通过统一 API 访问 Claude 3.5/4、Llama 4、Mistral 等顶级模型。
> - **额度极高**：通过合作伙伴渠道（如 Segment, Stripe），初创公司极易获得 **$25,000** 甚至更高的额度。

---

## 💰 财务支持与额度 (AWS Activate)

- [ ] [**AWS Activate Founders**](https://aws.amazon.com/activate/) - 针对未融资项目，提供 **$1,000** 额度、技术支持与培训。
- [ ] [**AWS Activate Portfolio**](https://aws.amazon.com/activate/) - 针对已融资项目，提供最高 **$100,000**（常见为 $25,000）的云服务抵扣券。
- [ ] [**AWS Free Tier**](https://aws.amazon.com/free/) - 包含 EC2, S3, RDS 等产品的 12 个月免费额度，以及 Lambda 等永久免费项。

---

## 🧠 AI 与大模型专项 (AI & Generative AI)

- [ ] [**Amazon Bedrock**](https://aws.amazon.com/bedrock/) - 无服务器方式接入顶级大模型，支持 RAG 工作流与 Agent 编排。
- [ ] [**SageMaker 专项补贴**](https://aws.amazon.com/sagemaker/) - 针对机器学习开发，可申请专项算力补贴进行模型训练与微调。
- [ ] [**Trainium & Inferentia**](https://aws.amazon.com/machine-learning/trainium/) - AWS 自研 AI 芯片，提供比标准 GPU 更高的性价比。

---

## 🌐 业务加速与市场 (Market & Growth)

- [ ] [**AWS Marketplace**](https://aws.amazon.com/marketplace/) - 将你的 SaaS 产品直接上架，触达全球数百万 AWS 企业客户。
- [ ] [**Global Startup Program**](https://aws.amazon.com/partners/global-startup/) - 针对中后期项目，匹配专属伙伴经理，助力全球联合销售。
- [ ] [**垂直行业加速器**](https://aws.amazon.com/startups/accelerators/) - 针对医疗、教育、金融等赛道的专项孵化活动。

---

## 💡 选型建议
1. **追求极致的全球可用性**：AWS 的节点覆盖是全球最完善的。
2. **需要多模型灵活切换**：利用 **Amazon Bedrock** 实现一个 API 调用多个顶级闭源模型。
3. **寻找企业级客户**：通过 **AWS Marketplace** 建立可信的采购渠道。
4. **撸额度秘籍**：先申请 Founders 拿到 $1,000，融资后再申请 Portfolio 拿到 $25,000。
